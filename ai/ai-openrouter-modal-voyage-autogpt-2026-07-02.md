## 跨产品核心洞见

1. **openrouter alternative（KD=8！）是本报告集里 KD 第三低的词**：170/mo 搜索量，几乎没有竞争内容——Ollama on Olares 是完整的本地 OpenRouter 替代，一篇内容可以直接排名。

2. **OpenRouter 靠 janitor ai（150万/mo）和 chub ai（9万/mo）借力获取大量流量**：这两个 AI 角色扮演平台使用 OpenRouter 作为 API 后端，带来了大量跨界流量——Olares 也可以通过支持这类应用的本地部署来覆盖相同用户群。

3. **Modal 的定位是"无服务器 GPU"**：modal pricing（KD=16，CPC=$7.64）说明按小时租用 GPU 的用户在研究成本——这批用户正好可以对比"购买 RTX 5070/Olares One 一次性成本"，是 Olares One 硬件产品的目标用户。

4. **Voyage AI 靠"voyage"这个一般词（旅行）获取大量意外流量**（18,100/mo）：说明品牌命名和 SEO 竞争是相互影响的，Olares 的品牌词搜索量会随时间增长，现在布局品牌词内容很重要。

5. **AutoGPT 已经式微**（SEO rank 126万）：曾经 GitHub 最热的项目现在网站流量极低，说明早期 AI Agent 热潮已经过去，市场已经转向 LangGraph、n8n 等更成熟的工具——Olares 应该布局这些"继任者"的相关词。
