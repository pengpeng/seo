## 核心洞见

- **docs 占 78% 流量**，vLLM 靠 recipes（食谱）教程页排名模型词：deepseek ocr（8,100月搜）、qwen3-vl-embedding（2,900）、qwen3-next（1,900）均通过文档中的具体教程获得排名。Olares 应该复制这个模式——每个新热门模型发布就出"在 Olares 上运行 XXX"的教程。
- **技术特性词 KD 异常低**：vllm quantization（KD=13）、vllm lora（KD=15）、vllm speculative decoding（KD=17）——这些工程师级别词 KD 低、转化率高，Olares 有资格写，因为 Olares One 是运行这些特性的实际硬件。
- **vllm 5090（90月搜，KD=19）**：RTX 5090 + vLLM 是高端用户的搜索词，Olares One 搭载高性能 GPU，这是精准的硬件+软件内容组合。
- **意外发现**：vllm.entrypoints.openai.api_server（390月搜，KD=6）——这是 vLLM 作为 OpenAI 兼容 API 服务器的技术词，KD=6 几乎零竞争，Olares 上搭建兼容 OpenAI API 的本地推理服务是常见需求。
