## 项目理解（前置）

RTX 4090 是 NVIDIA Ada Lovelace 架构的上一代旗舰消费显卡（2022-10 发布，2024 年底随 50 系上市逐步停产）。**24GB GDDR6X** 让它成为消费级本地 AI 的分水岭：单卡 Q4 量化跑 70B 级 LLM、出图（Stable Diffusion / Flux）、小规模微调都够用，是 Reddit r/LocalLLaMA、r/StableDiffusion 里默认的"性价比天花板"。停产后二手/挂牌词（used 4090、4090 ebay、4090 for sale）依然活跃，说明存量用户与接盘买家两头都在搜。

> 性能对照统一引 [olares-one-benchmarks.md](/Users/pengpeng/seo/directions/hardware/research/olares-one-benchmarks.md)（Olares 第一方实测；Olares One = RTX 5090 Mobile **24GB**，与 4090 同属 24GB VRAM 段位，可作同级参照）。

| 型号 | 显存 | 参考价（USD） | 本地 LLM 能力 |
|------|------|------------|-------------|
| **RTX 4090** | 24GB GDDR6X | $1,600+（停产仍抢手） | ⭐⭐⭐⭐⭐ 70B Q4 可跑，出图/微调主力 |
| RTX 4080 Super（语境） | 16GB GDDR6X | ~$900 | ⭐⭐⭐⭐ 30B 流畅 |
| RTX 4070 Ti Super（语境） | 16GB GDDR6X | ~$700 | ⭐⭐⭐⭐ 30B 流畅 |

**24GB 段位实测参照（Olares One，同 24GB VRAM）**：Qwen3-30B-A3B MoE 157→81 tok/s（vLLM，并发 1→8 不崩）；Gemma3-27B dense 38→29 tok/s；Flux.1 dev 出图冷 15.51s / 热 8.32s。**诚实边界**：24GB 装不下 GPT-OSS-120B 这类超大 MoE，需 offload（Olares One 上 36→4 tok/s），Mac Studio 96GB 统一内存在此项反超——写"能不能跑 120B"类对比时须主动承认。

## 核心洞见

1. **品牌护城河**：`rtx 4090`（49,500）/`4090`（33,100）主型号词 KD 41-46，被 NVIDIA 品牌权威天然占位，正面刚性价比低。**应绕开纯型号词，主攻"选卡决策词 + 场景词"**——4090 无独立官网，型号词流量分散在 nvidia.com/geforce 下，内容空档大。

2. **可复制的打法**：走**决策漏斗内容**而非品牌词。上游"选卡对比"（4090 vs 5090、best gpu for llm）+ 下游"部署教程"（self hosted llm、ollama gpu）成对布局，一篇对比文 + 一篇部署文互相导流；4090 无程序化落地页护城河，纯内容即可切入。

3. **对 Olares 最关键的 3 个词**：① `4090 vs 5090`（4,400/KD27，A 主战场：换代对比加整机选项）；② `used 4090`（1,300/KD21，B 主战场：存量大卡盘活）；③ `self hosted llm`（320/KD22/CPC$3.12，B+隐私叙事，商业价值高）。A（买整机）与 B（盘活已有卡）在 dGPU 组正好各有大词承接。

4. **最大攻击面**：**"买卡≠能用"的软件栈痛点**——搜 best gpu for llm / self hosted llm 的人买了卡还得自己配 CUDA/驱动/推理引擎/远程访问。Olares One 打"整机开箱免攒机免配栈"（轴 1）+ $3,999 整机 TCO（轴 2）；对已有卡的人打"一键装 AI 应用 + 安全远程"（B）。**诚实边界**：24GB 装不下 GPT-OSS-120B 需 offload，对比 Mac Studio 96GB 时须主动承认。

5. **隐藏低 KD 金矿**：`best gpu for llm`（KD=8！）、`best gpu for stable diffusion`（KD14）、`best gpu for ai video generation`（KD14）、`what is the amd equivalent to rtx 4090`（KD14）、`dual 4090`（KD18）、`24gb gpu`（KD19）、`used 4090`（KD21）——一批 KD<25 的选卡/二手/显存词，量精准、商业意图强、竞争极低，是内容优先级最高的一批。

6. **GEO 前瞻词**：`rtx 4090 vs mac studio`（10）、`4090 vs 5090 llm`（0）、`4090 vs a6000`（20）、`4090 vs 3090 ai`（20）——近零量但语义高度契合"本地 AI 选卡"，现在发深度对比（含 Olares One + 诚实的 24GB vs 统一内存边界），抢 AI Overview / Perplexity 引用位，产品词一上量即承接。

7. **与既有分析的关联**：nvidia-dgx.md 已记 `4090` 33,100/KD46（此处复核为主词 `rtx 4090` 49,500/KD46 + `4090` 33,100/KD41），并印证"5090/5080 型号词 KD 37-46、内容层竞争不高"的判断在 4090 同样成立。本报告补齐了 nvidia-dgx.md 缺失的**部署侧信号词层**（self hosted llm / private llm / ollama gpu / best gpu for llm），是对既有 GPU 型号词表在"选卡→部署"漏斗下游的补充。
