## 项目理解（前置）

RTX 3090（2020 Ampere 旗舰）与 3090 Ti（2022 小改款）都带 **24GB GDDR6X**，是消费级里唯二在二手市场跌到千元级的 24GB 大显存卡。虽然算力已被 40/50 系超越，但在本地 AI 社区它靠"每 GB 显存最便宜"活成了长青款：单卡跑 70B Q4、双卡 NVLink 凑 48GB 跑 100B+/微调，是预算敏感 prosumer 入本地 LLM 的经典起手式。这批 DIY 攒机、追求"用最少钱把大模型跑在自己机器上"的人，正是 Olares 自托管叙事的核心人群。

## 核心洞见

1. **品牌护城河（能否正面刚）**：3090 无独立官网，型号主词（`rtx 3090` 18.1K/KD56、`3090` 14.8K/KD45）由 nvidia.com 与大媒体/评测站霸榜，KD 高、纯信息意图，正刚不划算。但**对比词与品类导购词是开放战场**：`3090 vs 4090`（KD23）、`3090 ti vs 4090`（KD17）、`best gpu for llm`（KD8）几乎无强内容护城河，可正面拿下。

2. **可复制的打法**：型号词单打不过品牌站，但**"型号 × 用途 × 对比"的程序化长尾**是空档——`3090 vs 4090`、`3090 vs 5090`、`3090 vs 3090 ti`、`rtx 3090 vs 4080` 一整簇对比词 KD 全在 10–23，可用同一模板批量产出"X vs Y for local AI"内容，每篇都收口到 Olares One / Olares 部署。高 CPC 品类词（`self hosted ai` $3.90、`how to run llm locally` $4.61、`local ai server` $4.26）说明商业价值高、值得重点投内容。

3. **对 Olares 最关键的 2-3 个词**：① `3090 vs 4090`（880/KD23，信息 A 最高量对比战场）；② `best gpu for llm` / `best gpu for local llm`（KD 8/15，选卡导购金矿）；③ `dual 3090` + `used rtx 3090`（信息 B——把二手/双卡闲置算力用 Olares 盘活成私有 AI 云）。

4. **最大攻击面（痛点词）**：3090 的痛点不在价格（它极便宜），而在**"卡便宜但把它变成能用的 AI 系统很麻烦"**——`how to run llm locally`（KD38、CPC$4.61）、`local llm hardware`、`local ai server` 揭示大量用户卡在"买完卡之后怎么搭栈"。这正是 Olares 的攻击面：单买 3090 只解决算力，还得自装驱动/CUDA/推理引擎/前端/远程访问；Olares One 整机 + Olares OS 把这一切开箱化。诚实口径：不打"比 3090 便宜"，打"整机 TCO + 私有云 OS 全栈 + 每美元可用 AI"。

5. **隐藏低 KD 金矿**：`best gpu for llm`（KD8）、`best gpu for machine learning`（KD10）、`3090ti vs 4090`（KD10）、`3090 vs 3090 ti`（KD12）、`gpu for local llm`（KD13）、`dual 3090`（KD14）、`best gpu for stable diffusion`（KD14）、`best gpu for local llm`（KD15）——一批 KD<15 的选卡/对比词，量虽中小但意图极硬、CPC 有价，是投产比最高的一档。

6. **GEO 前瞻布局**：`rtx 3090 llm` / `3090 llm` / `3090 vs 4090 llm` / `3090 tokens per second` / `local llm gpu requirements` / `homelab gpu` 全部 KD≈0、量 10–30，`3090 ollama` / `run llm on 3090` / `dual 3090 llm` 甚至 US 库暂无量——本地 AI 需求正快速增长，提前发"在 3090 上跑本地 LLM / 双卡实测 tok/s / 部署到 Olares"内容，可抢占 AI Overview / Perplexity 引用位，产品/需求放量即承接。

7. **与既有分析的关联**：3090 补齐了 dGPU 组"最便宜 24GB 入口"这一端——与同组 `rtx-4090`（性能对标件）、`rtx-5090`（新架构）形成"3090 便宜 / 4090 性能 / 5090 新架构 + Olares One 开箱"的完整选卡矩阵。对比词（`3090 vs 4090`、`3090 vs 5090`）应与 4090/5090 报告交叉内链；`best gpu for llm`、`self hosted ai`、`how to run llm locally` 与 olares-500-keywords 的自托管/本地 AI 词族高度重合，可并入统一内容簇。
