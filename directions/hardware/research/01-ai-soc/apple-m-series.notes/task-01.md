## Findings
- 2025 Mac Studio 只有两档 SKU：入门 M4 Max（起 $1,999，36GB，14C CPU/32C GPU，410 GB/s）与 M3 Ultra（起 $3,999，96GB，28C CPU/60C GPU，819 GB/s）；无 M4 Ultra。[1][9][10]
- M4 Max 升级款（16C CPU/40C GPU）带宽升到 546 GB/s，可配 48/64/128GB；128GB 版约 $3,999。[1][13][19]
- M3 Ultra 顶配 32C CPU/80C GPU（+$1,500），内存可配 96/256/512GB，256GB 约 $5,999、满配可到 $14,099。[9][10][22]
- 官方规格页仍列 M3 Ultra 可配到 512GB；但 Macworld 与多篇评测称 2026-03 因 DRAM 缺货，Apple 一度只剩 96GB 可下单、砍掉 256/512GB——两说冲突，属供应波动。[1][10][11][18]
- 各代内存带宽（官方）：M4=120、M4 Pro=273、M4 Max=410/546、M1/M2 Max=400、M1/M2 Ultra=800、M3 Ultra=819 GB/s。[1][2][5][13][14]
- 2024 Mac mini：M4 起 $599（16/24/32GB，120 GB/s）、M4 Pro 起 $1,399（24/48/64GB，273 GB/s）；内存购买后不可升级。[2][13][16]
- 历史 Mac mini：M1 2020 起 $699（8/16GB，~68 GB/s）、M2 2023 起 $599（8/16/24GB，100 GB/s）、M2 Pro 2023 起 $1,299（16/32GB，200 GB/s）。[6][15][17]
- 历史 Mac Studio：M1 Max/M2 Max 起 $1,999（400 GB/s）、M1 Ultra/M2 Ultra 起 $3,999（800 GB/s，M2 Ultra 可到 192GB）。[3][5][14]
- Apple 从未出过 M3 Mac Studio / M3 Mac mini，也无独立 M3/M4 Ultra——Ultra 一线在 2025 由 M3 Ultra 承担。[9][12][14]
- 统一内存是 Mac 本地 LLM 的核心卖点：GPU 可寻址全部内存、无 VRAM 墙、模型不"溢出"到慢速 RAM；512GB M3 Ultra 是唯一能单机跑 DeepSeek-V3/R1 671B 的消费级机器。[18][22][23]
- Apple Silicon 不支持 CUDA，本地 LLM 走 MLX / llama.cpp(Metal) / Ollama(0.19 起用 MLX 后端) / LM Studio；生态 2025 起已是一线目标。[20][21][23]
- Mac Studio vs DGX Spark：DGX Spark 固定 128GB、273 GB/s，但有满 CUDA 栈与 1 PFLOPS FP4；Mac 在推理带宽/容量胜、DGX 在训练/prefill 胜；Olares One（$3,999/x86-Linux）可作第三选项。[18][19]

## Deep Read Notes
### Source [10]: AppleInsider — Mac Studio 2025 review（全文精读）
Key data: 入门 M4 Max $1,999 只有 36GB 一档、不可单独加内存；升 16C/40C 仅 $300 且送 48GB；64GB(+$200)、128GB(+$800)。M3 Ultra 起 $3,999/96GB/1TB，256GB(+$1,600)，顶芯 +$1,500，512GB(+$2,400)，满配 $14,099。M4 Max 546 GB/s vs M2 Max 400；M3 Ultra 819 vs M2 Ultra 800。
Key insight: M4 Max 单核快于 M3 Ultra，M3 Ultra 只在大规模并行/大内存（AI、数据科学、仿真）场景才值溢价——"most powerful Mac ever" 面向的是内存吃紧型负载。
Useful for: 第二章 SKU 表价格/带宽、第一章定位、第五章 M4 Max vs M3 Ultra 选择争议。

### Source [18]: SpecPicks — DGX Spark vs Mac Studio M3 Ultra（全文精读）
Key data: DGX Spark 固定 128GB LPDDR5x/~273 GB/s/1 PFLOPS FP4/$4,699（2026-02 涨价）；Mac Studio M3 Ultra 96/128/256GB（512GB 于 2026-03-06 因 DRAM 缺货下架）/819 GB/s/$3,999 起。Llama2-70B Q4 推理 M3 Ultra ~14 tok/s > DGX 11-13。软件栈 Mac=MLX/llama.cpp/PyTorch-MPS，DGX=CUDA/TensorRT-LLM。
Key insight: 两机同价（~$5-10K）解决相反问题——Mac 赢"容量×带宽"下的推理，DGX 赢训练/长上下文 prefill 与 CUDA 生态；均无法单机装下 671B Q4(~448GB)（除 512GB Mac）。
Useful for: 第四章 Message A 三方对标（+Olares One）、第五章争议。

### Source [21]: AI/TLDR — Run LLMs on a Mac（精读）
Key data: MLX 为 Apple 2023 末开源、2025 成熟；数组驻留共享内存、无 CPU↔GPU 拷贝。Ollama `brew install ollama`，0.19 起默认用 MLX 后端，快于直接 llama.cpp Metal。llama.cpp 走 Metal+GGUF 兼容最广。
Key insight: "无需装独显——GPU 已在芯片里、共享统一内存"是 Apple Silicon 对本地 AI 新手的最大心智门槛降低点，也是关键词入口（"run llm on mac"类）。
Useful for: 第一章生态、第三章引擎组合关键词（ollama/mlx）。
