## Findings (≤12)

1. Strix Halo = AMD Ryzen AI Max 300 系列单芯片 APU:16 Zen5 核 + 40-CU Radeon 8060S(RDNA 3.5,gfx1151)+ XDNA2 NPU,256-bit LPDDR5X-8000/8533,~256 GB/s 带宽,最高 128GB 统一内存 [15][16][17][27]。
2. 型号梯队(消费):Max+ 395(16C/32T,8060S 40CU,126 TOPS)/ Max 390(12C/24T,8050S 32CU)/ Max 385(8C/16T,8050S 32CU,最高仅 64GB) [17][22][3]。
3. PRO 梯队(HP 等商用):Max+ PRO 395 / PRO 390 / PRO 385 / PRO 380(6C/12T),TDP 默认 55W,ISV 认证 [8][9]。
4. 128GB 统一内存中 Windows 下最多 96GB 划给 iGPU,Linux 上可更高(GTT 可达 ~120-128GB),可本地跑 70B/120B 级模型 [1][27][28][29]。
5. GMKtec EVO-X2 是最便宜的 128GB Strix Halo 盒子(64GB ~$1,499 / 128GB ~$1,999),2.5GbE,风噪偏大,已量产 [3][6][7][20]。
6. Framework Desktop(4.5L Mini-ITX)是 Linux/开放性标杆,官方支持 Fedora 43 / Ubuntu 25.10,$1,099(385/32GB)-$1,999(395/128GB) [25][26][27]。
7. Beelink GTR9 Pro 主打网络(双 10GbE)与静音(均热板),MSRP ~$1,985,Intel E610 网卡在 Linux 需驱动 workaround [2][20]。
8. Minisforum MS-S1 Max 是唯一带真实 PCIe x16 插槽 + 10GbE SFP+ 的 Strix Halo SFF,~$2,399,USB4 v2 [7][24][5]。
9. HP 双旗舰:ZBook Ultra G1a 14"(笔记本工作站,PRO 系列,Ubuntu 24.04 官方选项,$2,747+~$4,049)与 Z2 Mini G1a(紧凑桌面工作站,395/390/385/380,$1,399-$2,649) [8][9][10][20]。
10. Asus ROG Flow Z13(2025)GZ302 是唯一平板形态(395 或 390,最高 128GB,$2,199 起;另有 Kojima 联名版) [12][13][14]。
11. 反例:Minisforum AI X1 Pro 用的是 Ryzen AI 9 HX 370 / HX 470(Strix Point,Radeon 890M,SO-DIMM 内存),**不是 Strix Halo**,不应误列 [23][3]。
12. Olares 裸机 ISO 要求 x86-64("ARM not supported" 仅指 ISO)、Ubuntu 22.04-25.04 / Debian 12-13、≥4 核 /8GB/150GB SSD;Olares 经 script/Docker 也支持 ARM Linux/macOS/Windows。所有 Strix Halo 设备均为 x86-64 → 走裸机路径原生高适配 [30][31]。GPU 加速：NVIDIA 最成熟；AMD Ryzen AI Max APU（Radeon 核显）经 ROCm 已被 Olares 加速（1.12.4 起逐步落地、1.12.6/1.12.7 加调度器标签，成熟度随版本）。

### [28][29] Strix Halo 上 Ubuntu 实测(社区)
- GMKtec EVO-X2 装 Ubuntu 24.04 / 24.04.3 + ROCm 跑 llama.cpp/ollama:BIOS 把 UMA/VRAM carve-out 设最小(0.5-1GB),用 GRUB `ttm.pages_limit` 把 ~120-128GB GTT 让给 GPU;需 `HSA_OVERRIDE_GFX_VERSION=11.5.1`。→ 证明 x86-64 Linux 路径成熟,Olares OS 可直接落地。

### [30][31] Olares Docs(合规唯一来源)
- 硬件:≥4 核,≥8GB RAM,≥150GB SSD(机械盘不支持)。裸机 ISO 要求 x86-64(ARM not supported 仅指 ISO);但 Olares 经 script/Docker 也支持 ARM Linux/macOS/Windows。Strix Halo 是 x86-64,直接走裸机路径。
- 系统:Ubuntu 22.04-25.04 LTS / Debian 12 或 13;脚本 `curl -fsSL https://olares.sh | bash -` 或 ISO 安装。
- GPU 可选,仅支持 NVIDIA(Turing+,≥8GB VRAM)。**重要:AMD iGPU(8060S)不在 Olares 官方 GPU 加速支持内 → Olares 上的 GPU 型 AI 应用当前面向 NVIDIA;Strix Halo 的 AMD iGPU 能否被 Olares 应用调用为 [unverified]。**
