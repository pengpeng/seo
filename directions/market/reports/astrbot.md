## 项目理解（前置）

AstrBot 是一个开源的 All-in-One Agentic 聊天机器人平台与开发框架，允许用户将 LLM（OpenAI、Anthropic、Gemini、DeepSeek、Ollama、LM Studio 等）接入 QQ、微信、WeChat公众号、企微、飞书、钉钉、Telegram、Discord、Slack、KOOK 等十余个 IM 平台，并通过 Docker 或一键启动器自托管部署。项目由 Soulter 主导，于 2022 年底发布，在中国开发者社区积累了深厚用户基础，同时英文文档与国际化正在快速推进。核心差异化在于：**原生 MCP 支持 + 1,000+ 插件生态 + 知识库 RAG + 多平台统一接入**，是个人/团队把 LLM 跑进日常 IM 工作流的最低摩擦路径。

## 核心洞见

1. **品牌护城河**：`astrbot`（3,600/月，KD 46）是中文开发者社区驱动的品牌词，英文 SEO 几乎空白。正面刚品牌词成本较高，但**切入点在非品牌功能词**——AstrBot 官方网站本身几乎不做 SEO，这意味着第三方内容（如 Olares blog）在教程/替代词上**几乎没有抵抗**。

2. **可复制的打法**：AstrBot 官网唯一的 SEO 成功案例是 docs 站靠技术配置长尾词（"lm studio docker base url" KD 11，排名第 1）拿自然流量。**打法可复制**：围绕各 LLM 提供商的配置词（"ollama telegram", "deepseek discord bot", "lm studio docker compose" 等）写教程，每个词 KD 0–15，几乎无竞争。

3. **对 Olares 最关键的 3 个词**：
   - **`self hosted ai chatbot`**（KD = 4）：最纯净的 Olares + AstrBot 场景词，即写即排
   - **`ai chatbot discord`**（KD 24，170/月）：高频实际需求词，AstrBot on Olares = 私有 Discord AI 机器人
   - **`chatgpt discord bot`**（KD 35，590/月）：替代 ChatGPT 订阅的价格攻击面，用 Olares + AstrBot + Ollama 来回应

4. **最大攻击面**：**闭源 chatbot 平台的价格与数据隐私**——Botpress、Poe Bot 等商业平台有月费、数据上传云端的痛点。`botpress alternative`（KD 15）、`botpress self hosted`（KD 18）都在 40 月量以下，但 CPC $3–5 意味着商业意图明确，内容层很快可转换。

5. **隐藏低 KD 金矿**：
   - `mcp chatbot`（KD 0）、`chatbot mcp`（KD 0）：MCP 是 AstrBot 的原生支持特性，当前这两个词几乎没有内容竞争——快速写 GEO 内容抢 AI Overview 引用位
   - `local llm discord` / `deepseek discord bot`（KD 0）：热点叠加词，DeepSeek 爆红使"DeepSeek + 免费 Discord 机器人"成为高频需求，目前内容空白

6. **GEO 前瞻布局**：`mcp chatbot`、`ai agent discord`、`local llm discord`——这三个词当前搜索量 10–20，但语义上完全对应 AstrBot + Olares 的核心叙事（Agent-Native OS + MCP 生态）。在 Perplexity / AI Overview 里抢"什么是带 MCP 的 chatbot？"的引用位，边际成本极低。

7. **与既有词表的关联**：本报告的"ai chatbot discord"（KD 24）、"chatgpt discord bot"（KD 35）与 olares-500-keywords 里可能的 LLM/Agent 场景词有高度互补——AstrBot 是把这些词"落地"到具体 IM 场景的最佳桥梁，Olares 提供硬件/OS 层，AstrBot 提供 IM 接入层，合力拿"怎么在自己家里建 AI 聊天机器人"这个搜索漏斗。
