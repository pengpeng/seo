## 核心洞见

- **几乎纯中文产品**，英文市场几乎无 SEO。MiGPT 是把小米小爱音箱接入 ChatGPT/Ollama 的工具，用户群体是中国小米设备用户。
- **豆包api（3,600月搜，KD=37）**靠 MiGPT 配置文档排名——说明中国 AI API 服务的教程词有流量，Olares 的中文 AI 配置文档可以覆盖这类词。
- **硅基流动 api（KD=1）**：KD 几乎为零，说明完全无竞争的中文 API 教程词，Olares 中文内容机会巨大。
- **Olares 角度**：MiGPT on Olares = 小米智能家居 + 本地 Ollama + Olares 统一管理，这是针对中国用户的独特场景内容。
