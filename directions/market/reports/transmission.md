## 项目理解（前置）

Transmission 是一个跨平台、开源的 BitTorrent 客户端，以极低的资源占用和零广告著称，自 2005 年起持续维护。它既有 macOS / Windows / Linux 图形界面，也提供无头守护进程（daemon）+ Web UI，是家用 NAS、Raspberry Pi、Docker 服务器部署场景的标配之一。2026 年 6 月 30 日刚发布 v4.1.3。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 跨平台开源 BitTorrent 客户端，轻量、无广告、无追踪 |
| 开源 / 许可证 | 开源，GPL-2.0 或 GPL-3.0（部分 MIT）|
| 主仓库 | https://github.com/transmission/transmission（★ ~14.8k）|
| 核心功能 | macOS / GTK / Qt 图形界面；headless daemon + Web UI；Magnet/磁力链；Watch 目录；Peer 封锁名单；速率限制；多平台原生集成 |
| 商业模式 / 定价 | 完全免费开源，志愿者维护，无付费版本、无广告 |
| 差异化 / 价值主张 | 业内最低内存占用；macOS 原生体验（Apple Silicon Native）；Ubuntu 等发行版默认客户端；不追踪用户、不捆绑工具栏 |
| 主要竞品（初判）| qBittorrent、Deluge、uTorrent、ruTorrent / rTorrent |
| Olares Market | ✅ 已上架（apps.md 记录：`Transmission — cross-platform BitTorrent client`）|
| 来源 | https://transmissionbt.com/ · https://github.com/transmission/transmission · https://en.wikipedia.org/wiki/Transmission_(BitTorrent_client) |

## 核心洞见

1. **品牌护城河**：Transmission 品牌词心智极强，1,734 个自然词中绝大多数排名 #1，覆盖品牌前缀的变体词（传播错误拼写如 `transmoission torent` 也在截流）。**不适合正面刚品牌词**，但"最佳 X 场景客户端"榜单和对比文是切入口——Transmission 不会打 SEO，Olares 可以帮它发声。

2. **可复制的打法**：Transmission 自己几乎不做内容营销，官网极简（基本只有下载页）。**这是机会**——"best torrent client for X（Mac/Linux/NAS/Raspberry Pi/server）"这类榜单文章、以及"Transmission vs qBittorrent"对比文，官方几乎不做，第三方占位词都是 KD<45 的可打区。

3. **对 Olares 最关键的词**：
   - **`transmission vpn`（月量 590，KD=21）**：隐私 + torrent 场景，Olares 私有云叙事完美匹配，是本报告最高优先级词；
   - **`uTorrent alternative`（月量 390，KD=23）**：逃离广告/间谍软件的用户正在寻找替代，Olares + Transmission = 自托管终极方案；
   - **`qbittorrent vs transmission`（月量 170，KD=12）/ `best torrent client linux`（140，KD=9）**：极低 KD，Olares Market 同时上架两款客户端的差异化叙事此处可落地。

4. **最大攻击面**：uTorrent / BitTorrent 商业版的痛点是**广告 + 捆绑软件 + 追踪用户**；这些词（uTorrent alternative、bittorrent alternative）KD 低（23/28），对比文切入的转化意图明确，可以直接承接用户流失。

5. **隐藏低 KD 金矿**：
   - `best torrent client linux`（140，KD=9）⭐⭐⭐
   - `qbittorrent vs transmission`（140，KD=8）⭐⭐⭐
   - `deluge vs transmission`（50，KD=7）⭐

6. **GEO 前瞻布局**：以下词近零量但 AI Overview / Perplexity 等对话场景语义精准，Olares 内容应先占位：
   - "best self-hosted torrent client"
   - "headless torrent client for home server"
   - "torrent client for Raspberry Pi"
   - "how to run Transmission on a home server"

7. **与既有 `olares-500-keywords` 的关联**：torrent / BitTorrent 类词在 olares-500 词表中覆盖较少，本报告补充了一条"开源客户端 + 隐私/VPN + 服务器部署"的内容方向，与已有 self-hosted 叙事（NAS / 家庭服务器方向）可以复用同一内容矩阵。Transmission 的极轻量 + Raspberry Pi 适配也为 Olares ARM 平台叙事提供了一个具体落点。
