## 项目理解（前置）

Halo 是由 Fit2Cloud（飞致云，同款公司旗下还有 1Panel）开发的开源建站 CMS，基于 Java/Spring Boot，以 Docker 方式部署。它的差异化在于：插件化架构（功能模块可热插拔）+ 主题市场 + RESTful API，支持博客、知识库、企业官网、在线商城四大场景，并已引入 AI 辅助创作与知识库问答。项目社区活跃（39k+ Star，3.7M+ Docker Pulls，300+ 版本），文档与社区主要面向中文用户，英文市场几乎空白——这既是弱点，也是 Olares 的正面机会。

## 核心洞见

1. **品牌护城河**：halo.run 在英文 SEO 市场几乎没有护城河——140 个排名词全为中文，英文词排名为零。"halo cms"、"halo docker"、"halo open source"的 KD 均为 0，是可以用单篇英文文章轻松占位的 SERP 空白地带。正面刚中文市场无意义，英文出圈才是 Olares 的打法。

2. **可复制的打法**：Halo 自身没有英文内容打法可复制，但同类项目（Ghost、WordPress）的对比词（ghost vs wordpress 170/月 KD 20，wordpress alternative 390/月 KD 23）已有成熟流量。借竞品对比词流量、在文章中将 Halo+Olares 作为第三选项推荐，是最快起量的路径；同时可做"Halo on Olares"的部署教程内容抢 halo+docker/self-hosted 类词。

3. **对 Olares 最关键的词**：
   - `strapi alternative`（70/月，KD=8）——几乎无竞争的高价值对比词，Olares Market 同时支持 Strapi 和 Halo，可写双产品对比文
   - `contentful alternative`（140/月，KD=11，CPC $19.34）——商业意图极强，Halo 的内容 API 可正面替代 Contentful，Olares 作部署底座
   - `wordpress alternative`（390/月，KD=23）——最大流量入口，Halo 是 WordPress 的现代化自托管替代选项

4. **最大攻击面**：Ghost 的痛点是 Pro 版贵（$9-25/月），Contentful/Sanity/Strapi 云版本也有 API 限制/计费墙。Halo 社区版完全免费自托管，Olares 一键部署消除运维门槛——"stop paying for Ghost Pro / Contentful, self-host Halo on Olares for free"是最锐利的攻击叙事。

5. **隐藏低 KD 金矿**：
   - `directus alternative`（30/月，KD=5）——KD 极低，几乎空白
   - `ghost cms alternative`（20/月，KD=0）——零竞争
   - `sanity alternative`（20/月，KD=0，CPC $8.92）——零竞争但高 CPC，商业意图强
   - `halo cms`（10/月，KD=0，CPC $6.39）——率先布局可长期持有

6. **GEO 前瞻布局**：
   - `halo cms self-hosted`（量极低，语义极精准）→ 进 FAQ 段抢 AI Overview 引用
   - `open source blog with ai assistant`（近零量）→ Halo 内置 AI 问答知识库，是 GEO 差异化锚点
   - `self-hosted cms with plugin marketplace`（近零量）→ 针对 Halo 插件市场做概念布局
   - `halo cms vs ghost`（零量）→ AI 搜索已有人在问，提前布局

7. **与既有 olares-500-keywords 的关联**：Halo 场景补充了 Olares 的"内容建站/个人出版"维度，与 olares-500-keywords 中"self-hosted"、"open source"、"docker"系列词形成协同——Halo 相关对比文可复用这些词的 SEO 积累，为 Olares Market 的内容建站品类开辟新的入口。
