## 核心洞见

- **tumblr（1,830,000月搜）带来 4,758 月流量**：Ghost 制作了 ghost.org/vs/tumblr/ 对比页排名竞品词——这是博客工具 SEO 最有效的策略。Olares 应该为所有 Market 应用制作"Olares vs [竞品]"对比页，可以借助竞品的巨大搜索量带来流量。
- **ghost template（1,300月搜，KD=26，CPC=$18.70）**：主题/模板词 CPC 极高，说明有强烈的商业意图，Ghost 用户在为主题付费，Olares 上 Ghost 的定制化内容是高价值方向。
- **ghost editor（KD=9）**：全场最低 KD 之一，Ghost 编辑器教程几乎无竞争。
- **broadway news 和 scouted today 通过 explore.ghost.org 带来超高量流量**——Ghost 托管了实际媒体出版商，通过出版商内容积累 SEO。Olares 的 Ghost 实例同样可以做这类内容托管策略。
- **意外发现**：ghost api（CPC=$10.49）——Ghost API 词 CPC 高，说明有开发者在为 Ghost API 集成方案付费，Olares 提供 Ghost Headless CMS + API 是高商业价值内容。
