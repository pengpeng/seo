## 项目理解（前置）

Refly 是一个开源 AI 工作区平台，已从早期的"Agentic Workspace for Human-AI Collaboration"（自由画布 + 多线程 AI 对话 + RAG 知识管理）演进为"The First Open-Source Agent Skills Builder"——用 Vibe Workflow（自然语言→确定性 Agent 技能）将业务逻辑固化为可版本化、可跨平台部署的 Agent 技能。用户在画布 IDE 中描述工作流，Refly 将其编译为可导出为 MCP Server、REST API、Webhook 的可重用技能，并可直接集成进 Claude Code、Cursor、Lovable、Slack、Lark 等平台。核心受众从个人知识工作者扩展至希望在生产环境部署可靠 Agent 的企业团队。

## 核心洞见

1. **品牌护城河**
   refly.ai 当前 SEMrush Rank 超 260 万，US 月流量仅 205，几乎全部来自品牌词。它几乎没有非品牌 SEO 积累，说明：a）品牌本身尚未在搜索引擎建立护城河；b）正面竞争任意品类词都有胜算，无需硬刚头部玩家。

2. **可复制的打法**
   Refly 当前完全没有 SEO 落地页布局。可借鉴的对手打法是 AFFiNE、AppFlowy 等开源工具用"self-hosted X"细分词撑起大量长尾页——Refly（及 Olares）复制这套「一词一页 + 内部链接」程序化落地页策略，成本极低且 KD 普遍在 15-20。

3. **对 Olares 最关键的词**
   - **`obsidian alternative`**（590/mo, KD 16）：单词最优机会，商业意图强，可写"Refly vs Obsidian：Open-Source Agentic Canvas on Olares"直接带 Olares 链接
   - **`self hosted notion alternative`**（110/mo, KD 20）：漏斗底部精准词，已决意自托管的用户，Olares 一键部署是完美 CTA
   - **`ai note taking app`**（2,900/mo, KD 29）：量最大、KD 边界友好，信息型内容+Olares 路径可获取大量顶漏斗流量

4. **最大攻击面**
   Notion 最大弱点是**没有自托管选项 + 积分式定价**（用 AI 功能消耗积分），用户痛点明确。`personal knowledge management software`（KD 12, CPC $53）说明愿意为 PKM 付高价的用户群真实存在，Refly+Olares 可以"一次买断，自托管，永无订阅"作为攻击角度。Obsidian 的弱点是没有原生云协作和 AI 功能，Refly 画布+RAG 可直接填补。

5. **隐藏低 KD 金矿**
   - `logseq alternative`：KD 仅 **6**，月量 40，Logseq 用户极度在意开源和本地优先，与 Refly+Olares 画像完美重叠
   - `affine self hosted`：KD **15**，月量 170，AFFiNE 是 Refly 最近的竞品，自托管对比文绝对低竞争可排
   - `personal knowledge management software`：KD **12**，CPC $53，高意图低竞争的宝藏词

6. **GEO 前瞻布局**
   重点前置布局以下 AI Overview / Perplexity 引用词（回答"什么是…"的 FAQ 格式）：
   - "What is an agentic workspace?"（Refly 自创品类，抢先占位）
   - "What is a vibe workflow?"（新术语，GEO 定义抢占）
   - "best self-hosted Notion alternative with AI"（精准问题词，触发 AI Overview）
   - "open source agent skills builder"（开发者社区搜索词，与 Claude Code/Cursor 生态相关）

7. **与既有分析的关联**
   `agentic ai tools`（880/mo, KD 47）与 Olares 500 词分析中 AI agent 相关词重叠，但 KD 过高不宜正面竞争。更好的打法是用`self hosted notion alternative`、`obsidian alternative`等低 KD 词先积累权威，再向`agentic ai tools`等头部词延伸。Olares 整体叙事"Agent-Native OS"与 Refly 的 Agent Skills Builder 定位高度协同，内容可互相强化。
