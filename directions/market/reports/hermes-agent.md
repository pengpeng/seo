## 核心洞见

- **188K GitHub Stars，官网月流量 1,205**：几乎全靠拼写错误词（hermes agnet：541流量）。任何认真的 Hermes Agent 部署教程都可以轻松超过官网流量。
- **hermes ai（KD=15，CPC=$8.10）**：KD 极低，CPC 高，Olares Market 已上架，内容发布几乎无阻力。
- **多语言用户基础强**：中文（爱马仕 ai 3,600月搜）、韩文（헤르메스 170月搜）、日文页面均在排名，亚洲用户是核心群体，但中文 SEO 内容几乎空白。
- **CPC 普遍高**（$6-12）：hermes ai sdr（$12.16）、hermes agent github（$12.21）——企业开发者在评估工具，付费意愿强。
- **意外发现**：hermes 微信（1,600月搜，KD=18）——Hermes Agent 支持微信接入，这个词量大KD不高，说明中国企业用户对微信+AI Agent 有真实需求，是 Olares 中文内容的精准切入点。
