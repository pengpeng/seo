## 项目理解（前置）

ShowDoc 是一款面向 IT 团队的在线文档协作工具，核心场景覆盖 API 文档、数据字典、技术说明文档与团队知识库。它基于 Markdown 编辑器，支持接口模板插入、自动从代码注释生成文档（搭配 RunApi 客户端）、多人协作与权限管理，可免费云托管或自托管到私有服务器（支持 Docker 部署）。项目 2015 年创立，主仓库已获 12,820 星，最新版 v3.8.4（2026-05-25）；使用方包括腾讯、百度、华为、字节跳动等超过 10 万个互联网团队。英文站点 showdoc.com.cn/help-en 有完整英文文档，证明有出海意图，但 US SEO 几乎空白。

## 核心洞见

1. **品牌护城河**：ShowDoc 品牌词"showdoc"在 US 只有 590 月量、KD 14——护城河极浅。它是中国市场之王（10 万+ 团队），但在英文市场是陌生人。这意味着"ShowDoc alternative"内容的竞争几乎为零，可以直接正面打，不用担心被强大的品牌防守压制。

2. **可复制的打法**：ShowDoc 本身在 US 没有任何内容打法可以参考。应对标竞品 GitBook（6,600 月量，KD 59）、BookStack（6,600 月量，KD 56）、Wiki.js（3,600 月量，KD 50）的内容布局——这些竞品 KD 高但带动了大量"alternative"低 KD 长尾词。**最可复制的打法是"替代文 + 自托管教程"组合**：一篇"GitBook alternative self-hosted"，一篇"confluence alternative open source"，在文中自然引入 ShowDoc + Olares 部署方案。

3. **对 Olares 最关键的词**：
   - **`self hosted wiki`**（260 月量，KD 14）：量价比最高的机会词，ShowDoc 完美覆盖，Olares 一键部署消除最后摩擦。
   - **`gitbook alternative`**（140 月量，KD 13）：KD 最低的主词候选，ShowDoc 与 GitBook 功能高度重叠，且免费、自托管，竞争者不多。
   - **`technical documentation software`**（590 月量，KD 23，CPC $16）：量最大的可攻主词，CPC 高说明商业价值，ShowDoc 切入"免费开源自托管"角度差异化明显。

4. **最大攻击面**：主要竞品的定价痛点显著——GitBook 云端版 $8–$32/月/用户，Confluence 云版更贵；文档数据由第三方托管有隐私顾虑；大团队成本随用户数线性上升。ShowDoc 完全免费+开源，加上 Olares 的一键部署，**"零成本私有部署 vs 按用户付费云端"**是最大的攻击面，也是 CPC 高（$16–$25）证明的高意向词场景。

5. **隐藏低 KD 金矿**：`open source developer portal`（170 月量，KD 15，CPC $4）是被低估的词——它比"api documentation tool"KD 低了 11 个点，而量相当，ShowDoc 完全覆盖"开发者门户"场景。同理，`knowledge base open source`（70 月量，KD 17，CPC $9）和`open source knowledge base`（140 月量，KD 16）成本极低的机会，可并入同一篇文章。

6. **GEO 前瞻布局**：以下近零量词是 AI Overview / Perplexity 引用位的目标：
   - `"self-hosted GitBook alternative"` → ShowDoc + Olares 一键部署教程
   - `"open source Confluence alternative self-hosted"` → FAQ 段落
   - `"self-hosted API documentation tool free"` → Olares Market ShowDoc 页面
   - `"ShowDoc vs GitBook"` → 对比表格段落（ShowDoc 免费/私有数据 vs GitBook 付费/托管）
   这些词语义精准，即便量近零，AI 搜索引用频次会超过传统 CTR 估算。

7. **与既有分析的关联**：当前 olares-500-keywords 词表中，"self-hosted"方向的文档/知识库类词覆盖较少，ShowDoc 填补了这一空白。`technical documentation software`、`api documentation tool`、`gitbook alternative`、`self hosted wiki` 这四个词可以作为新簇加入下游 seo-content 文章选题。ShowDoc 也是少数能同时覆盖"自托管 wiki + API 文档 + 技术文档"三合一场景的工具，适合作为 Olares Market 应用生态文章的锚点应用。
