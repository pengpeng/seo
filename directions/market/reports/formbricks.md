## 项目理解（前置）

Formbricks 是一个开源的体验管理（XM）平台，定位为 Qualtrics/Typeform/SurveyMonkey 的自托管替代方案。它提供网站内嵌问卷、in-app 产品调研、链接式调研三合一，核心差异化在于：GDPR 原生合规、数据完全归用户（可 Docker 一键自部署），以及高度灵活的触发条件（按用户属性/事件精细分组）。目标用户是注重数据隐私的中大型 SaaS 团队和开发者，既提供免费云端（Hobby）也支持完全离线自托管。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 自托管、隐私优先的开源体验管理平台（XM） |
| 开源 / 许可证 | 是，AGPLv3（企业功能部分 EE License） |
| 主仓库 | [github.com/formbricks/formbricks](https://github.com/formbricks/formbricks)（★ ~12,500） |
| 核心功能 | In-product 调研、链接调研、邮件内嵌调研、NPS 自动化、预建模板库、AI 分析、多语言翻译 |
| 商业模式 / 定价 | 云端：Hobby 免费（250 次/月）/ Pro $74/月 / Scale $325/月；自托管完全免费（付费可解锁企业功能） |
| 差异化 / 价值主张 | 开源可审计 + 自托管数据主权 + GDPR/CCPA/HIPAA 合规 + 精细事件触发器 + 无限席位 |
| 主要竞品（初判）| Typeform、Qualtrics、SurveyMonkey、JotForm、Survicate |
| Olares Market | 已上架（⬜ 待完成报告） |
| 来源 | [formbricks.com](https://formbricks.com)、[formbricks.com/pricing](https://formbricks.com/pricing)、[github.com/formbricks/formbricks](https://github.com/formbricks/formbricks) |

## 核心洞见

1. **品牌护城河**：Formbricks 品牌词本身搜索量仅 590/月（排 P2），心智极弱——Typeform/Qualtrics/SurveyMonkey 是绝对主导。**不要正面刚品牌词，而是从"替代词 + 自托管/开源 + 隐私合规"三个低竞争维度迂回切入。**

2. **可复制的打法**：Formbricks 自身靠"海量调研博客"带流量（TOP 流量词全部是信息长尾，如"open ended survey questions"月量 2.2 万流量 1,443）。但这些词与产品卖点脱节——流量进来的是"学生找问题例子"而非"企业买调研工具"的用户。**Olares 的机会在商业词一侧：qualtrics alternative（KD=9, CPC=$54）、product feedback tool（KD=17）、customer feedback software（KD=29）这三个词商业价值最高、Formbricks 自身并未充分覆盖。**

3. **对 Olares 最关键的词**：
   - **`qualtrics alternative`**（KD=9, CPC=$54）：几乎无竞争但极高购买意图，是"开源 Qualtrics on Olares"最精准的入口；
   - **`typeform alternative`**（KD=20, 量 320）：最大替代词流量池，"self-hosted Typeform"叙事落 Olares；
   - **`product feedback tool`**（KD=17, CPC=$22）：与 Olares Personal Agent 生态结合，讲产品 → 用户反馈 → AI 分析的完整闭环。

4. **最大攻击面**：Typeform/SurveyMonkey/JotForm 的**定价不透明 + 数据存第三方 + 响应次数上限**是最大痛点词。用户搜 "typeform alternative free"（KD=11）、"self hosted typeform"（KD=0）时正在逃离——Formbricks on Olares 提供"无限次数 + 数据自持 + 免费自托管"的三重答案。

5. **隐藏低 KD 金矿**：
   - `qualtrics alternative`：KD=9，$54 CPC，量 70 但极精准；
   - `formbricks docker`：KD=0，量 40，技术用户自部署教程；
   - `self hosted typeform`：KD=0，量 20，精准需求；
   - `open source nps`：KD=0，量 20，NPS 自托管需求；
   - `on premise survey software`：KD=0，量 20，企业级部署场景。

6. **GEO 前瞻布局**：AI Overview / Perplexity 会被问"最好的开源调研工具是什么"——在以下词的 FAQ 段落植入引用锚：`open source nps`、`self hosted typeform`、`gdpr survey tool`、`on premise survey software`、`open source feedback`。这些词 KD=0，现在写零成本抢先占位。

7. **与既有分析的关联**：olares-500-keywords 中的"self-hosted"叙事与本报告高度一致。Formbricks 补充了"survey / feedback / NPS"垂直场景的新词集群，特别是 `customer feedback software`（$25 CPC）和 `nps survey tool`（$57 CPC）两个极高价值词此前尚未覆盖，可作为 Olares 内容矩阵的新补充方向。
