## 项目理解（前置）

LangBot 是一个开源的生产级平台，用于构建可接入 Discord、Slack、Telegram、WeChat、QQ、LINE、DingTalk、Lark 等 10+ IM 平台的 AI 机器人。它的核心定位是**"AI 应用交付基础设施"**——将用户已有的 Dify/Coze/n8n/Langflow 工作流或直接 LLM（OpenAI、Claude、DeepSeek、Ollama 等）桥接到所有主流聊天平台，并内置 RAG 知识库、插件系统（100+ 扩展）、速率限制、访问控制、MCP 协议支持等生产就绪能力。超过 35% 的用户以 Dify 为后端。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 把 AI 应用部署到任意 IM 平台的开源生产级机器人平台 |
| 开源 / 许可证 | 是，Apache 2.0 |
| 主仓库 | [langbot-app/LangBot](https://github.com/langbot-app/LangBot)（★ 16,600+，1,474 forks，90 贡献者） |
| 核心功能 | 多平台 IM 接入（10+）、LLM/Agent 编排、内置 RAG 知识库、插件生态（100+）、Web 管理面板、MCP 协议、沙箱 Skills |
| 商业模式 / 定价 | 开源自托管免费（Apache 2.0）；LangBot Cloud 托管版按月付费 |
| 差异化 / 价值主张 | 一次部署覆盖全平台；深度集成 Dify/Coze/n8n 等 LLMOps；内置企业级管控（速率、访问控制、监控）；插件生态活跃 |
| 主要竞品（初判） | Botpress、Flowise（Chatflow）、Rasa、Coze（闭源）、n8n（工作流侧）、Dify（LLMOps 侧） |
| Olares Market | 未上架 |
| 来源 | [langbot.app](https://langbot.app)、[GitHub](https://github.com/langbot-app/LangBot)、[docs.langbot.app](https://docs.langbot.app) |

## 核心洞见

1. **品牌护城河**：LangBot 品牌词本身 KD 仅 27，量 210，是少有的"品牌词=机会词"的情况——海外英文社区认知度尚低，Olares 在其 Market 上架后可第一批吃到品牌词流量，正面刚性低。主要竞品 Botpress/Flowise 品牌词 KD 44/43，已形成护城河，不宜正面攻品牌词，转攻对比/替代词。

2. **可复制的打法**：SERP 中 `dify vs n8n`（KD 13）排名的基本是 Medium/博客小站——说明这类"两个 open source 工具对比"词，专注用户视角的长篇博文就能排进前 10。可复制到：`botpress vs flowise`、`dify vs flowise`、`rasa vs botpress`（全部 KD≈0）。同时，"chatbot for discord"词 top 10 中出现了 docs.inkeep.com、chat-data.com 这类 niche 小站，说明 KD 27 可以被中型域名撬动。

3. **对 Olares 最关键的 3 个词**：
   - `LLM chatbot`（KD 22，Vol 390）——最大的"量×低KD"机会，Olares 本地 LLM 叙事的完美载体；
   - `knowledge base chatbot`（KD 28，Vol 210，CPC $7.87）——RAG 直接词，Olares 私有文档场景；
   - `dify vs n8n`（KD 13，Vol 140，CPC $7.04）——极低 KD 且已有 SERP 可模仿模式，LangBot 是两者的 IM 接入层，Olares 同时托管三者。

4. **最大攻击面**：Coze 是最大痛点竞品——闭源、按量付费、国内厂商（出海有隐忧），`coze alternative`（KD 10）和 LangBot 的自托管开源定位形成完美对比。其次 Botpress 近期从开源转商业化（`botpress alternative` KD 15），正值用户流失期。

5. **隐藏低 KD 金矿**：`rasa vs botpress`（KD 0）、`botpress vs flowise`（KD 0）、`dify vs flowise`（KD 0）——三个对比词全部零竞争，均有 20-30 月量；单篇综合对比文（"5 Best Open Source Chatbot Platforms 2026"）可一次覆盖全部，成本极低。另：`wechat bot`（KD 27，Vol 140）是海外几乎无人覆盖的蓝海词，LangBot 是少数支持的开源工具。

6. **GEO 前瞻布局**：
   - `MCP chatbot`（Vol 20，KD 0）：2026 MCP 协议席卷 AI 工具链，LangBot 已内置 MCP 支持，应抢占"MCP protocol chatbot"、"chatbot with MCP tools"等零量词，占领 AI Overview 引用位；
   - `agentic chatbot`（CPC $13.58，KD 38）：高商业价值词，进入 FAQ 段落布局；
   - `langbot discord setup`、`langbot dify integration`、`langbot ollama`：品牌词衍生教程词，近零量但 GEO 精准。

7. **与既有分析的关联**：LangBot 直接与 Olares 方向 1（Market 应用）、方向 2（场景=Personal Agent with IM access）强关联。从 `olares-500-keywords` 视角看，"self-hosted AI"类词（`self hosted AI` Vol 390）与 Olares 核心叙事重合，LangBot 可作为该词的具体用例展示，为 olares.com 的 SEO 内容补充真实 app 场景证明。
