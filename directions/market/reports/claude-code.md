## 核心洞见

- **CPC 是所有分析产品中最高的赛道**：claude code settings（$35.87）、claude code privacy（$29.57）、claude code memory（$26.28）、claude code vs opencode（$26.16）——开发者在为"私有/离线/配置 Claude Code"的解决方案付高价，Olares + Ollama 就是答案。
- **claude code plan mode（2,400月搜，KD=15）**是新发现的高价值词：量大、KD 低、CPC 高（$15.68），这是 Claude Code 的一个核心功能词，目前几乎没有专门的教程内容。
- **claude code vs aider（KD=6）**：全场 KD 最低的对比词，Olares Market 同时上架 Claude Code，有天然资格写这篇文章，且竞争几乎为零。
- **"离线"和"本地 LLM"是 Olares 最精准的切入角度**：claude code ollama（KD=29）、claude code offline（KD=24）——精准对应 Olares 的核心差异化，CPC 高说明这是开发者真正愿意付钱的场景。
- **意外发现**：claude code hooks（2,400月搜，KD=52，$16.06 CPC）——hooks 是 Claude Code 的高级功能，有大量搜索需求但 KD 较高，Olares 的 settings.json hook 配置是差异化内容角度。
