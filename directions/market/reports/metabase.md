## 项目理解（前置）

Metabase 是业内最受欢迎的开源商业智能（BI）与内嵌分析工具，2015 年创立，总部加利福尼亚州 Spring Valley。核心价值主张是「让非技术人员也能探索数据」——可视化查询构建器大幅降低 SQL 门槛，任何团队成员均可自助建图表、建 Dashboard。同时面向 SaaS 产品商提供内嵌分析能力（Embedded Analytics），支持 iFrame 或 React SDK 白标定制、多租户数据隔离。2026 年新增 MCP Server，打通 AI Agent 数据查询通道。Community Edition（CE）完全免费、AGPL 许可、可无限制自托管；商业版（Pro $575/月起，Enterprise $20K/年起）增加 SSO、行列级权限、白标嵌入等功能，Pro 与 Enterprise 功能完全相同，仅服务层级有别。
