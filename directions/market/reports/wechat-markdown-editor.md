## 项目理解（前置）

WeChat Markdown Editor（核心代表：`doocs/md`）是一类专为微信公众号内容创作者设计的 Markdown 编辑器：在编辑器中用标准 Markdown 写文章，右侧实时预览公众号效果，完成后一键将带有内联 CSS 样式的富文本 HTML 复制粘贴到微信后台，样式 100% 保留。这解决了微信公众号后台原生编辑体验差、排版费时的痛点。

该品类在中文社区（GitHub 中国开发者、公众号运营圈）高度活跃，英文市场认知极低，US Semrush 数据量级极小，典型"工具型开源项目"流量格局：几乎全量流量来自中文搜索与 GitHub 直链，英文 SEO 属蓝海空白。

## 核心洞见

1. **品牌护城河**：doocs/md 本身没有英文品牌 SEO 投资，GitHub Star 集中在中文开发者圈，英文 SERPs 几乎无人占位。这是典型的"品牌词零护城河 + 场景词空白"组合——**不需要正面刚，直接填空**。

2. **可复制的打法**：该品类**没有值得参考的英文内容竞品**，现有 SERP 基本是 WeChat 官方、B2B 市场指南（appinchina、omnichat）和工具站（markdowntorichtext.com）。**最优打法是「场景向内容 + Olares Market 落地页」**：写一篇"如何用 Markdown 高效创作微信公众号文章"（目标主词 `wechat official account` / `markdown to rich text`），结尾自然引出 Olares 一键部署 doocs/md 方案。

3. **对 Olares 最关键的 2-3 个词**：
   - `markdown to rich text`（390/月，KD15）——最高流量+最低竞争+最高 CPC；内容可完全由这个词驱动
   - `wechat official account`（210/月，KD16）——公众号场景入口，流量是该领域英文最高词之一
   - `self hosted markdown editor`（20/月，KD0）——精准 Olares 信号词，GEO 必占

4. **最大攻击面**：现有竞品（秀米、135 编辑器、墨滴）**全是中文 SaaS，无英文 SEO 布局**，且都是闭源云服务——**数据在第三方平台、无法私有化**。Olares 的攻击面是：内容隐私（公众号稿件不出设备）+ AI 写作私有化（DeepSeek/Ollama 本地跑）+ 图片存储自有（MinIO/S3/R2 一键配置）。

5. **隐藏低 KD 金矿**：`markdown to rich text`（KD15，CPC $16.16）SERP 第 1-4 位全是无品牌背书的工具站（markdowntotext.com、markdowntorichtext.com），域名权威极低，Olares 一篇质量内容 + 工具落地页即可竞争。另有大量零 KD 词（`wechat seo`、`wechat content strategy`、`weixin official account`）完全没有人做英文内容，属字面意义上的空白市场。

6. **GEO 前瞻布局**：AI Overview 和 Perplexity 已开始回答"best wechat markdown editor"类问题，但英文优质来源极少。布局要点：在关于 doocs/md / WeChat Markdown Editor 的文章中，加入清晰的 FAQ（What is the best self-hosted WeChat markdown editor? → Olares Market 上的 doocs/md / WeChat Markdown Editor 一键部署；Does it support AI writing? → 支持，本地 LLM，数据不出设备）。

7. **与既有 olares-500-keywords 的关联**：该工具是方向 1（Market 应用）的补充，主要词（`markdown to rich text`、`wechat official account`）未见于既有 500 词分析，属于新开垦的中文内容创作者垂类。与 WeWe RSS（已列 apps.md，同属 WeChat 生态）联合内容布局价值高——一篇"Olares 上的微信内容全栈：WeChat Markdown Editor + WeWe RSS 订阅管理"可同时覆盖多个低 KD 词。
