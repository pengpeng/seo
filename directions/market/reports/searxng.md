## 核心洞见

- **月流量 4,081，估值 $8,218**，SearXNG 的流量质量高于数量，平均 CPC=$2/访客。
- **searxng api（CPC=$49.73）**：是本次所有分析产品中 CPC 最高的单一关键词！说明企业用户在为"私有搜索引擎 API"付非常高的价格，Olares 上的 SearXNG API 服务是极高商业价值内容。
- **searxng mcp（210月搜，KD=16）**：SearXNG 作为 AI Agent 的搜索工具这个场景正在爆发，KD=16 几乎无竞争，Olares 可以做"SearXNG + MCP + OpenClaw/Claude Code on Olares"的完整教程。
- **bt4g 策略**：SearXNG 通过引擎文档页排名 bt4g（2,900月搜，KD=20）——一个 BitTorrent 搜索引擎词，说明 SearXNG 文档页面排名了大量搜索引擎相关的长尾词，这是内容的意外收益。
- **意外发现**：torznab（390月搜，KD=13）通过 SearXNG 文档排名，这是 Prowlarr/Jackett 等媒体管理工具使用的 API 协议词，说明 SearXNG 的用户群和媒体服务器用户高度重叠，Olares 同时支持这些应用是内容联动机会。
