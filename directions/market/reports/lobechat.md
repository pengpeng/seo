## 核心洞见

- **最大战略洞察：LobeHub 的 SEO 流量 90%+ 来自 AI 工具图标/Logo 页面**（lobehub.com/icons），而非 LobeChat 本身。gemini logo（12,100月搜）、claude logo（9,900月搜）、openai icon（6,600月搜）等词都通过图标页排名，是 LobeHub 最大的非品牌流量来源。**Olares Market 应该为每个应用建立标准化 Icon 页面，完全可以复制这个策略。**
- **lobechat 品牌词 KD=15**：极低，Olares 一键部署 LobeChat 的教程可以快速排名。
- **jm官网（6,600月搜，KD=12）通过 MCP Server 目录排名**：MCP Server 聚合列表是新兴的大流量来源，Olares Market 建立 MCP Server 目录可以排名大量类似的中文 MCP 词。
- **gram hub（9,900月搜，KD=30）通过 OpenClaw Skills 页排名**：再次验证 Olares Market 应用页面的 SEO 价值。
- **意外发现**：hermes agent ai icon（1,900月搜，KD=27）通过图标页排名——说明 Hermes Agent 用户群和 LobeChat 用户高度重叠，Olares 可以在 LobeChat 教程中联动提及 Hermes Agent。
