> 开源 Web Office 套件领导者：自托管 MS Office 兼容文档/表格/演示编辑器，兼营云协作平台（DocSpace），目标用户从个人到中大型企业。

## 项目理解（前置）

ONLYOFFICE 由拉脱维亚公司 Ascensio System SIA 开发，2009 年以 TeamLab 起步、2013 年改名 OnlyOffice，是当前**开源自托管 Office 套件中 MS Office 兼容性最强的方案**。核心产品分两条线：①**ONLYOFFICE Docs**（AGPLv3 开源文档服务器，与 Nextcloud/ownCloud/40+ 平台集成，也可 Docker 独立部署）；②**ONLYOFFICE DocSpace**（以"房间"为单位的协作平台，云端免费个人版 + 付费商业版 + 自托管企业版）。官方声称全球 2,100 万用户。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 开源、可自托管、MS Office 格式高兼容 Web Office 套件 |
| 开源 / 许可证 | 是；Docs 核心 AGPLv3；Desktop Editors 免费闭源；DocSpace 服务端有社区/企业双版本 |
| 主仓库 | [ONLYOFFICE/DocumentServer](https://github.com/ONLYOFFICE/DocumentServer)（★ 6,682）；[ONLYOFFICE/DocSpace](https://github.com/ONLYOFFICE/DocSpace-buildtools) |
| 核心功能 | Web 文档/表格/演示编辑（.docx/.xlsx/.pptx 高兼容）、PDF 编辑与表单、实时协作、AI 助手、40+ 平台集成（Nextcloud/ownCloud/Odoo/Moodle 等）、桌面离线版（Win/Mac/Linux） |
| 商业模式 / 定价 | Docs Community Edition 免费开源；DocSpace 云端"管理员收费、用户免费"；Enterprise 自托管付费（连接数计）；Desktop Editors 免费 |
| 差异化 / 价值主张 | MS Office 格式还原度业内最高；唯一能让 Nextcloud/ownCloud 用户获得"Google Docs 级"实时协作体验；完全开源、服务器数据不出境 |
| 主要竞品（初判）| LibreOffice（离线为主）、Collabora Online（LibreOffice Online）、Google Workspace、Microsoft 365 |
| Olares Market | **已上架**（⬜ 待调研状态，本报告覆盖） |
| 来源 | [onlyoffice.com](https://www.onlyoffice.com/)、[GitHub](https://github.com/ONLYOFFICE/DocumentServer)、[developer-edition 页](https://www.onlyoffice.com/developer-edition.aspx) |

## 核心洞见

1. **品牌护城河（能否正面刚？）**
   OnlyOffice 品牌词月量 9,900，KD 88，**无法正面刚**。但非品牌机会词（`google docs alternative`、`onlyoffice vs libreoffice`、`best free office suite`）KD 普遍 14–29，**极可攻击**——OnlyOffice 自身在这些词上排名也很弱（未排名或弱排名），第三方内容有机会超过它自己的页面。

2. **可复制的打法**
   - **博客信息内容漏斗**：OnlyOffice 用`如何在 Excel 中移动列`、`circular formula`、`barbie font` 等通用 Office 教程吸量，然后漏斗进入产品页。Olares 可复制：以"如何在 OnlyOffice 里做 X"+"如何在 Olares 上安装 OnlyOffice"为双轴写教程，同时获通用 Office 教程流量和品牌部署流量。
   - **模板库程序化页面**：`templates.onlyoffice.com` 贡献 10% 流量（3,748/月），靠 1,840 个模板词。Olares 可类比做应用场景落地页（"在 Olares 上自托管 OnlyOffice 模板"）。

3. **对 Olares 最关键的 3 个词**
   - `google docs alternative`（月量 1,300，KD 14）——量大+KD 极低+Olares 叙事完美对接，优先级最高
   - `onlyoffice vs libreoffice` / `libreoffice vs onlyoffice`（合计月量 1,200，KD 22）——比较词，Olares 适合写"哪个更适合在家庭服务器运行"的决策文
   - `onlyoffice docker`（月量 140，KD 22）——Olares Market 直接竞争词，用户搜这个词就是在找自托管安装方式

4. **最大攻击面**
   - **部署门槛痛点**：OnlyOffice Docker Compose 配置复杂（Nginx、SSL、Nextcloud 集成需手动），Olares 一键安装是核心差异。
   - **订阅费用痛点**：`private office suite`（CPC $8.32）、`google workspace alternative`（CPC $6.65）说明用户愿意为"逃离订阅"付费，Olares 叙事"一次性拥有 vs 永久订阅"有力。
   - **开源可信度问题**：`is onlyoffice safe`（70/月）、`is onlyoffice free`（110/月）说明用户对 OnlyOffice 的开源/免费边界不清楚——Olares 内容可以回答"OnlyOffice Community Edition 在 Olares 上是真开源+完全免费"。

5. **隐藏低 KD 金矿**
   - `collabora vs onlyoffice` / `onlyoffice vs collabora`（合计 100 月量，**KD 5**）：近乎零竞争，技术决策词，进入门槛极低。
   - `libreoffice alternative`（140 月量，**KD 14**）：LibreOffice 用户寻找升级路径，OnlyOffice on Olares 是完美答案。
