## 核心洞见

- **月流量 7,042，SEO 重点是 Figma 替代词**——figma alternatives（1,000月搜，KD=21）是最大的非品牌流量词，说明 Penpot 的定位完全是"Figma 的开源替代"。
- **adobe xd alternative（KD=11）和 sketch alternative（KD=10）**：两个极低KD的设计工具替代词，Olares 上部署 Penpot 是这些词的最佳落地场景。
- **penpot mcp（KD=0）**：Penpot 已推出 MCP Server（从 Phase 1 数据可见 penpot mcp server 在排名），但这个词完全没有竞争，Olares 是第一个写"Penpot MCP on Olares"教程的平台。
- **博客内容策略突出**：Penpot 靠博客文章排名 phosphor icons（2,900月搜）、font face css（880月搜）等设计工具相关词，这种"不相关高量词通过内容排名"的策略 Olares 可以学习。
- **意外发现**：penpot mcp server（110月搜，KD=0）通过 penpot.app/ai/mcp-server 排名第一——Penpot 已经在布局 AI/MCP 方向，Olares 的 Penpot + AI 组合内容是独特角度。
