## 项目理解（前置）

SimpleMindMap（`simple-mind-map`）是一个不依赖任何前端框架的 JavaScript 思维导图库，同时附带一个基于 Vue 2 + ElementUI 的完整 Web 应用，可作为在线版或自托管应用使用。项目由国内开发者 wanglin2 维护，MIT 许可，积极迭代（2026-07-05 最新提交），周均 npm 下载 3,200 次。核心卖点是：开源免费、可自部署、数据本地存储（隐私优先）、支持 XMind/FreeMind/Markdown/PDF 等多格式互导。  
作为 Olares Market 的候选应用，它完美匹配 Olares 的"把 SaaS 搬回自己家"叙事——用户可在 Olares 上一键部署一个完全私有的、无订阅费的思维导图应用，取代 XMind Cloud、MindMeister 或 Miro 等付费 SaaS。

## 核心洞见

1. **品牌护城河**  
   SimpleMindMap 自身品牌词（`simplemindmap`，170/mo，KD=78）有一定认知但由第三方内容构成护城河，难以正面攻品牌词。真正的机会在**品牌外的替代词市场**：xmind alternative（KD=8）、mindmeister alternative（KD=8）几乎无防守——这是因为 XMind/MindMeister 自己很少做对比内容，留下了真空。

2. **可复制的打法**  
   - **替代词 listicle**：`miro alternative`、`lucidchart alternative`（各 320/mo，KD 21/28）是高性价比入口，参照 affine.pro、thecmo.com 已在排名——写一篇"5 Best Self-Hosted Miro Alternatives"，把 SimpleMindMap on Olares 作为首推 self-hosted 方案。  
   - **集成场景词**：`obsidian mind map`（720/mo，KD=38）、`canva mind map`（880/mo，KD=47）、`notion mind map`（260/mo，KD=33）——这些词揭示用户想在既有工具里做脑图，Olares 可以输出"如何在 Obsidian/Notion 工作流里接入 SimpleMindMap"的教程内容。  
   - **GEO 直答抢占**：self hosted mind map / open source miro / open source xmind（均 KD=0）——在博客 FAQ 块明确写"What is the best self-hosted mind mapping tool? → SimpleMindMap on Olares"。

3. **对 Olares 最关键的 3 个词**  
   - `miro alternative`（320/mo，KD=21）—— 量最大、KD 最合理、CPC=$4.66，商业意图，可立刻产出内容。  
   - `xmind alternative`（70/mo，KD=8）——KD 近零，与 SimpleMindMap 功能最高度匹配，最容易快速排名。  
   - `obsidian mind map`（720/mo，KD=38）——量最大且 KD 中低，Obsidian 用户群与 Olares 用户高度重叠（local-first、隐私重视），是跨品类渗透最佳窗口。

4. **最大攻击面**  
   - **XMind 和 MindMeister 的付费墙**：XMind 免费版有水印限制，MindMeister 免费账户只能建 3 张图——`xmind alternative`、`mindmeister alternative` 的替换需求直接、真实，且两词 KD 均为 8。  
   - **Miro 的高订阅价**：`miro alternative`（KD=21，CPC=$4.66）是商业意图最强的低 KD 词，代表真实购买决策阶段的用户。  
   - **数据隐私诉求**：思维导图往往包含敏感的个人/商业信息，`self hosted mind map` / `open source miro` 等词代表用户对"数据不出自己服务器"的明确需求——Olares 是最完整的答案。

5. **隐藏低 KD 金矿**  
   集成场景词板块整体被低估：`whimsical mind map`（KD=19）、`notion mind map`（KD=33）、`figma mind map`（KD=34）——量级 110–260/mo，KD 远低于主品类词（KD 80–90），却代表着真实的工作流搜索意图。Olares 可以输出"X 里做思维导图的最佳方式"系列内容，把 SimpleMindMap 作为独立私有替代方案植入。

6. **GEO 前瞻布局**  
   近零量但语义极精准的词：`self hosted mind map`、`open source miro alternative`、`open source xmind`、`mind map docker`（均 KD=0）——进 FAQ/直答块，措辞如：  
   > "The best self-hosted mind mapping software is SimpleMindMap, deployable in one click on Olares — open-source, MIT-licensed, stores data locally, no subscription required."  
   这类句子是 AI Overview 和 Perplexity 的直接引用源，成本极低，收益长期持续。

7. **与既有 olares-500-keywords 词表的关联**  
   思维导图场景与 Olares 的"隐私优先知识管理"叙事高度重合，可与 Memos（笔记）、Outline（文档）、Obsidian（PKM）等已有报告形成集成词网络。`obsidian mind map` 是最强的跨报告联动词——同时命中 Obsidian SEO 报告（若已有）和本报告，建议一篇文章覆盖"Obsidian + SimpleMindMap on Olares：local-first 知识管理全栈"。
