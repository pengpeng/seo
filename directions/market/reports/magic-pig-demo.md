## 核心洞见

### 1. 品牌护城河
Shimeji（18,100 月量，KD 56）是该品类绝对统治品牌，Java 桌面端已运行 10+ 年，社区词汇几乎都由其衍生（shimejis、shimeji app、shimeji browser extension）。正面硬刚无意义。**攻击面在"Shimeji 的缺点"**：需要 Java 环境、无法在 Linux 无头模式下跑、安全性存疑（Java 沙箱）、不支持自托管——这些恰好是 Olares 的优势区。

### 2. 可复制的打法
- **平台专题页**："desktop pet mac"（KD 24）、"desktop pet linux"（KD≈0）说明用户在按平台找解法，Olares 可按平台细分做教程落地页（mac/linux/windows）。
- **问题词内容**：`how to get a desktop pet on mac`（量20，KD≈0）、`are desktop pets safe`（量20，KD≈0）等问题词竞争真空，做 FAQ 型内容即可抢入。
- **"AI 化"前瞻布局**：`ai desktop pet`（KD 34，CPC $0.37）是趋势词，Olares 的 Agent-Native 叙事与之天然契合——"你的桌面宠物可以由本地 AI Agent 驱动"这个概念文章可以占坑。

### 3. 对 Olares 最关键的词
1. **`desktop pet mac`**（110 月量，KD 24）——低竞争、平台明确、Olares macOS 支持直接对应；
2. **`desktop pet games`**（720 月量，KD 31）——Olares Market 趣味应用内容文章锚点词，流量入口；
3. **`ai desktop pet`**（110 月量，KD 34）——与 Olares Agent-Native 品牌主轴结合，前瞻占坑。

### 4. 最大攻击面
- **"安全性"**：`are desktop pets safe`（KD≈0）——大量桌面宠物应用需要访问鼠标位置/屏幕，用户隐私顾虑真实存在。Olares 本地运行、AGPL 开源、无云端数据上传，是最有力的安全答案。
- **"Mac 支持"**：主流桌面宠物（Shimeji 是 Java）在 Mac 上体验参差不齐；Olares 的 macOS 原生支持（Docker 层）可以填补这个缺口。
- **"Linux 支持"**：`desktop pet linux`（KD≈0）完全无竞争——Linux 用户桌面宠物极为稀缺，Olares 原生 Linux 是唯一选手。

### 5. 隐藏低 KD 金矿
- `desktop pet mac`（KD 24）+ `desktop pet macbook`（KD 26）+ `desktop pet engine`（KD 29）这三个词构成"Mac + 工具"机会簇，合计约 310 月量，KD 均 < 30。
- `cursor pet`（KD 0）、`desktop pet linux`（KD 0）、`desktop mascot`（KD 0）——这些词量虽小但竞争绝对真空，是技术/开发者向内容的完美锚点词，也是 GEO 前瞻词的优先候选。

### 6. GEO 前瞻布局
