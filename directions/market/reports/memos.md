## 项目理解（前置）

Memos 是一款 MIT 许可的自托管笔记应用，核心价值主张是"先捕获、再整理（Capture first. Keep it yours.）"——单个 Docker 命令即可启动，笔记以 Markdown 原生存储，无需文件夹/模板，像私有 Twitter/flomo 的形态记录碎片灵感、日志和链接。相比 Notion/Obsidian 等重量级工具，Memos 刻意保持轻量，不做"第二大脑"，只做快速捕获 + 私有时间轴。官方 FAQ 将其定位为"Google Keep、Notion、Evernote 的自托管平替"，但更接近 flomo / Day One 的使用体验。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 自托管轻量 Memo 工具，Markdown 原生，快速捕获私有时间轴 |
| 开源 / 许可证 | 是，MIT License |
| 主仓库 | https://github.com/usememos/memos（★ ~61K） |
| 核心功能 | Markdown 笔记、标签、全文搜索、时间轴视图、REST/gRPC API、WebHook；Docker 一键部署；SQLite/MySQL/PostgreSQL |
| 商业模式 / 定价 | 完全免费开源，无 SaaS 付费版；用户自负服务器成本 |
| 差异化 / 价值主张 | 极简 + 快速（无需标题/文件夹）、Markdown 原生存储、数据自有（零遥测）、MIT 许可可商业化 |
| 主要竞品（初判）| flomo（国内闭源 SaaS）、Google Keep（闭源）、Joplin（开源笔记，功能更重）、Standard Notes（开源，加密）、Obsidian（本地优先，插件生态）、Logseq（双链图谱） |
| Olares Market | 已上架（apps.md 行 61） |
| 来源 | https://usememos.com/ · https://github.com/usememos/memos · https://usememos.com/docs |

## 核心洞见

1. **品牌护城河**：usememos.com 品牌词心智在开发者/homelab 圈子极强（61K GitHub ⭐），但 SEO 覆盖极薄（184 词、Rank 844K）。"memos" 一词本身搜索意图混杂（商务备忘录 vs 应用名），难以正面对抗，但品牌词 KD=50 已经证明可以在 top 3 内停留——不必正面强攻"note taking app"（KD=54），而应从低 KD 的自托管/替代词进入。

2. **可复制打法**：usememos.com 几乎零 SEO 布局（无程序化页面、无 vs 对比文、无教程系列），但 GitHub 流量倒流 + Reddit/HackerNews 社区形成了流量基底。**Olares 的可复制打法**：做 usememos.com 没做的内容——"best self-hosted note taking apps"对比文、"Memos Docker 安装教程"、"notion alternatives self hosted"落地页，收割已有搜索量但无质量内容覆盖的词。

3. **对 Olares 最关键的 3 个词**：
   - **`notion alternatives`**（1,600 vol / KD=26）：量最大、KD 低于 30，Olares Market 可做多应用对比落地页（Memos + AppFlowy + BookStack），一篇覆盖多款产品
   - **`second brain app`**（260 vol / KD=17）：KD 极低，契合 Olares "Personal Agent + 私有笔记" 的核心叙事
   - **`self hosted notion alternative`**（110 vol / KD=20）：精准商业意图 + 低 KD，直接对应 Olares Market 一键部署价值主张

4. **最大攻击面**：flomo/Notion/Evernote/Google Keep 均为闭源 SaaS，数据不归用户，且近年集体涨价（Notion 涨价 2023、Evernote 被 Bending Spoons 收购后大幅涨价）。**付费工具→私有部署迁移潮**是最大窗口期，词如 `self hosted evernote`、`self hosted google keep` 虽量小但意图极强，加上 AI 搜索（Perplexity/SGE）大量生成"替代工具"推荐列表——GEO 抢占这些词的引用位成本极低。

5. **隐藏低 KD 金矿**：
   - `joplin alternative`（KD=5、vol=50、CPC=$2.40）——竞争几乎为零，且付费意图明确
   - `roam research alternative`（KD=7、vol=90、CPC=$3.12）——Roam 是知识工作者的重度工具，迁移用户高价值
   - `self hosted notes`（KD=13、vol=40）——usememos.com 已排 #2，内容稍微补充即可拿 #1
   - `flomo alternative`（KD=0、vol=20）——完全空白，Memos 是语义上最接近 flomo 的开源替代

6. **GEO 前瞻布局**：`memos alternative`（KD=0）、`flomo alternative`（KD=0）、`self hosted google keep`、`micro journaling app`——这几个词已经在 Perplexity/Claude/ChatGPT 的回答中出现对应语义，但 Olares 目前在这些场景中未被引用。只需在文章 FAQ 区段一句话回答这些问题，即可低成本抢占 AI Overview 引用位。
