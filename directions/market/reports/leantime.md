## 项目理解（前置）

Leantime 是一款开源项目管理工具，定位独特：以"目标先行"（Goals → Milestones → To-dos）的层次结构区别于传统任务管理工具，并将行为科学与 ADHD/神经多样性友好设计融入产品内核。它既有完全自托管的社区版（AGPLv3），也提供 $8-10/用户/月的云 SaaS 版；开源版提供 Kanban、甘特图、时间追踪、白板、Sprint/Backlog 等核心功能，高级功能（策略管理、Program Plan、AI 等）以插件形式在 Marketplace 提供。用户自评："相当于自建版的 Asana/ClickUp，更轻量，不会被 Jira 复杂度淹没。"

## 核心洞见

1. **品牌护城河**：品牌词"leantime"月量 590，KD 44，稳居 #1，但护城河不算深——品牌心智局限于小圈子（自托管爱好者 + ADHD 社区）。不建议直接正面刚品牌词，而是从品类词和替代词侧翼切入，把 Olares+Leantime 包装为 Asana/ClickUp 的完整开源替代。

2. **可复制的打法**：Leantime 的内容策略是"长尾信息词 + 博客文章"（ADHD、Lean Canvas、Kanban Swimlane 等），引入的往往不是购买意图最强的流量。**可以复制的关键点**：围绕"open source + [大品牌] alternative"这组词，写专门的对比/替代文章——这组词 CPC 极高（$14-21），KD 却在 19-28，是内容 SEO 的黄金带。

3. **对 Olares 最关键的 3 个词**：
   - **`adhd project management`**（KD=6）：全域竞争最低的有量词，Leantime+Olares 双重独特性叙事（ADHD 友好 + 数据私有），几乎无对手。
   - **`monday.com alternative`**（390 月量，KD=19，CPC=$20.61）：高价值商业词，KD 极低，Olares+Leantime = 零订阅费替代方案，ROI 最高的攻击目标。
   - **`open source jira alternative`**（140 月量，KD=24）：开发者自托管受众，精准契合 Olares 用户画像，Olares Market 一键部署叙事最强。

4. **最大攻击面**：大 PM 工具（Asana $13.49/用户/月、ClickUp $10/用户/月、Monday $12+）的闭源 + 订阅 + 数据锁定是核心痛点。Leantime 社区版自托管免费，Olares 进一步简化部署——"永久免费 + 数据自有 + 一键部署"是完整反叙事。此外，Asana/ClickUp/Monday 的替代词 CPC 在 $13-21，说明付费获客成本极高，内容 SEO 反向收益巨大。

5. **隐藏低 KD 金矿**：
   - `adhd project management`：KD=6，量 90（全球约 360+）；
   - `project management for startups`：KD=19，量 110；
   - `monday.com alternative`：KD=19，量 390，CPC=$20.61；
   - `basecamp alternative`：KD=19，量 260，CPC=$17.78；
   - `open source change management`：KD=10，量 110，CPC=$5.95；
   - `self hosted adhd manager`：KD=20，量 210——Leantime 已 #2，Olares 教程可直接拿 #1。

6. **GEO 前瞻布局**：
   - `project management mcp server`（量 30，KD=0）：Leantime 是极少数开源 PM 工具中已推出 MCP Server 的，当用户问 AI "哪些 PM 工具支持 MCP/AI Agent 集成？"时，内容应让 Leantime+Olares 首先被引用；
   - `open source asana alternative`（量 20，KD=0）+ `open source monday alternative`（量 20，KD=0）：语义精准、近零竞争，写进相关文章的 FAQ 段即可抢 AI Overview 引用位；
   - `self hosted project management for adhd`（接近零量）：高度特异性词，AI 问答场景中唯一能回答这个问题的工具就是 Leantime。

7. **与既有分析的关联**：既有 500 词分析和翼果词表均未覆盖"自托管项目管理"这个词簇，是全新增量方向。与 Olares 现有已覆盖的"self-hosted + [应用]"叙事（文件存储、知识库等）保持一致，可复用"自托管主权叙事"框架。ADHD 方向在现有词表也是空白，有完整的差异化内容机会。
