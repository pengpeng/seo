## 项目理解（前置）

Context7 是由 Upstash Inc. 开发的 MCP（Model Context Protocol）服务器，核心价值是将最新、版本精确的库文档实时注入 Cursor、Claude Code、Windsurf 等 AI 代码编辑器的上下文中——彻底解决 LLM 基于过期训练数据编写已废弃 API 的问题。用户只需在提示词末尾加上 `use context7`，即可让 AI 编辑器在生成代码前自动从 Context7 拉取当前版本的正确文档。开源代码库托管于 GitHub（upstash/context7），在 GitHub 上积累了 58,655 stars，显示出极高的开发者认知度。

## 核心洞见

1. **品牌护城河**：Context7 品牌词（"context7"、"context 7"）持有 SEMrush Rank #108,923 但月均流量 17K+，原因是这些词 CPC 高达 $10-15——是 AI 工具赛道的高价值品牌。品牌词 KD 仅 32-38，说明还未到被大平台垄断的阶段，**可正面写"Context7 教程/集成/替代"系列**。真正难打的是 `claude mcp`（KD 37，4400/月）等品类词，建议从低 KD 长尾切入。

2. **可复制的打法——程序化文档库页**：Context7 最聪明的 SEO 武器是 `context7.com/<org>/<repo>` 与 `context7.com/websites/<site>` 路径——为每一个索引的开源仓库/文档站自动生成落地页，吸收该项目的品牌词流量（arcropolis 2400/月、go2rtc 2900/月、ory 5400/月、better-auth 3600/月……）。**Olares 若做 Market 应用落地页，可参考此模式**：`market.olares.com/app/<name>` 对每个应用生成 SEO 页面。

3. **对 Olares 最关键的 3 个词**：
   - `context7 mcp server`（1,000/月，KD=13）——量与 KD 比例最优，Olares 自托管 MCP Server 的直接落地词；
   - `best mcp for cursor`（110/月，KD=19）——商业意图，Olares Market MCP 合集的绝佳入口；
   - `windsurf mcp`（320/月，KD=16）——新兴 AI 编辑器，竞争极低，Context7 on Olares 的集成教程可快速排名。

4. **最大攻击面**：Context7 Pro 对私有仓库收费 **$25/1M tokens 解析**，Free 只有 1,000 calls/月——**"自托管 Context7 = 免解析费 + 无 API 限额"** 是 Olares 的核心攻击点。Enterprise 自托管已在官网明确列出，但定价"$30-2.50/用户/月"，而 Olares 是一次性购买（Olares One $3,999）或自备硬件免费装——TCO 叙事空间大。

5. **隐藏低 KD 金矿**：
   - `context7 mcp server`（KD=13，1000/月）⭐⭐ 最高性价比词；
   - `claude code mcp context7`（KD=17，140/月）⭐ 极精准集成词；
   - `context7 cursor`（KD=18，90/月）⭐ 配置教程词；
   - `best mcp for cursor`（KD=19，110/月）⭐ 商业意图低竞争；
   - `windsurf mcp`（KD=16，320/月）⭐ Windsurf 用户池快速增长。

6. **GEO 前瞻布局**：以下近零量词可嵌入 FAQ/前瞻段抢 AI Overview、Perplexity、Claude 引用：
   - "Can you self-host Context7?" → 触发 Olares Market 部署方案；
   - "Context7 vs llms.txt" → 两种文档注入方式对比（Olares 两者都能跑）；
   - "How to use Context7 with private repositories for free" → Olares 本地运行场景；
   - "Best MCP server for private documentation" → Olares 私有 MCP 基础设施叙事。

7. **与既有 olares-500-keywords 的关联**：Context7 赛道（MCP 服务器 + AI 代码编辑器文档）与 olares-500-keywords 中的 AI 工具/开发者工具方向高度吻合。`cursor mcp`（2400/月）、`claude mcp`（4400/月）是开发者心智最强的词，可作为 Olares Market 系列文章（"在 Olares 上一键运行 Top MCP 服务"）的重要词库补充。
