## 项目理解（前置）

JSON Hero 是由 [Trigger.dev](https://trigger.dev) 团队构建的开源 JSON 查看器，运行于 Web 端，旨在让开发者像"浏览文件资源管理器"一样探索 JSON 数据——自动识别字符串中的图片、颜色、URL、日期等语义类型，并实时渲染预览。它不仅仅是格式化工具（如 JSON Formatter），而是一个专注于"理解"与"导航"的 Explorer，内置自动生成 JSON Schema、关键词模糊搜索、可分享链接、VS Code 扩展等功能，完全 100% 开源可自托管。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 开源 JSON 查看器/浏览器，美观界面 + 智能类型推断 |
| 开源 / 许可证 | 是，Apache-2.0（部分 MIT） |
| 主仓库 | https://github.com/triggerdotdev/jsonhero-web（★ ~10.8k） |
| 核心功能 | 树形/列视图浏览、自动渲染图片/URL/颜色/日期、自动生成 JSON Schema、全局模糊搜索、可分享链接、VS Code 插件 |
| 商业模式 / 定价 | 纯免费 + 开源；官网 jsonhero.io 提供托管版，无付费计划 |
| 差异化 / 价值主张 | 对比竞品（jsoneditoronline、JSON Crack）更侧重"浏览体验"而非编辑；智能类型识别是核心差异（视觉化 > 纯文本格式化） |
| 主要竞品（初判）| jsoneditoronline.org、jsoncrack.com、jsongrid.com、dadroit.com |
| Olares Market | **已上架**（`directions/market/apps.md` 第 199 行，状态 ⬜ 待调研） |
| 来源 | [jsonhero.io](https://jsonhero.io)、[GitHub triggerdotdev/jsonhero-web](https://github.com/triggerdotdev/jsonhero-web) |

## 核心洞见

1. **品牌护城河**：JSON Hero 品牌词（`json hero` / `jsonhero`）KD 分别为 43/39，且其在 #1 位置，品牌心智已建立但较浅（月量仅 320+110）。正面与品牌词直接竞争无意义；应侧攻品类词与教程词。主流品类词（`json viewer` 22.2k、`json formatter` 90.5k）KD 均超 60，短期难攻——专注 KD<45 的中长尾才是可执行路径。

2. **可复制的打法**：主要竞品 curiousconcept.com（63k 流量）和 jsoneditoronline.org（38k 流量）依靠**大量内容页**积累流量；JSON Hero 目前全依赖首页，无内容页。可复制打法：创建教程文（`how to open/view json files`，KD 22-30）+ 功能词专题文（`json visualizer`、`json schema viewer`），用内容页带动首页权重提升。

3. **对 Olares 最关键的词**：
   - **`how to open json files`**（月量 1k，KD 22）：教程意图，门槛极低，Olares 上的 JSON Hero 作为"本地/私有 JSON 查看器"是最自然的答案。
   - **`json visualizer`**（月量 880，KD 42）：JSON Hero 已排 #3，Olares 场景天然契合（可视化 LLM/Agent 输出的 JSON）。
   - **`self-hosted json viewer`**（GEO，近零量）：SEO 量微，但是 Olares 叙事的精准词；在 AI Overview/Perplexity 语境下，抢"自托管 JSON 工具推荐"引用位。

4. **最大攻击面**：JSON Hero 作为纯免费开源工具，没有付费 credits 或定价痛点；攻击面在于**隐私**——对于需要处理内部 API 响应、敏感数据集的企业/开发者，上传 JSON 到 jsonhero.io（第三方 SaaS）有隐私顾虑；Olares Market 上的自托管 JSON Hero 正是解药。同时，JSON Hero 已无活跃维护（最近提交稀少），可用"积极维护的替代"作为攻击角度。

5. **隐藏低 KD 金矿**：
   - `how to open json files`（KD 22，月量 1k）
   - `how to open json file`（KD 28，月量 2.4k）
   - `how to view json file`（KD 30，月量 320）
   - `json schema viewer`（KD 30，月量 140）
   - `json path tester`（KD 26，月量 70）
   这批词 KD 全部 ≤ 30，且有稳定的教程意图流量，是最易拿到排名的入口。

6. **GEO 前瞻布局**：
   - `self-hosted json viewer`：Olares Market 收录 JSON Hero，是唯一能在 AI Overview 里被引用为"自托管方案"的场景。
   - `json hero self hosted`：工程师在问"JSON Hero 能不能自托管"时的精准词，在 FAQ/docs 页中植入 Olares 部署教程即可抢位。
   - `json viewer open source`：AI 聚合答案里推荐开源 JSON 工具时的频率词，确保 Olares Market 的 JSON Hero 页面出现在引用来源中。

7. **与既有分析的关联**：JSON Hero 覆盖的关键词（`json viewer`、`visualize json`、`json schema viewer`）与 Olares 500 关键词库中的"developer tools"和"data visualization"方向有交叉；`how to open json file`（KD 28）属于高性价比补充词，可作为 Olares Market 开发者工具内容簇的教程落点。
