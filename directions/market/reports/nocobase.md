## 项目理解（前置）

NocoBase 是一个开源 AI + no-code 平台，面向企业/团队快速构建业务系统（CRM、ERP、工单、审批等内部工具）。其核心差异在于**微内核插件架构**——一切皆插件，可无限扩展而不失控；同时以**数据模型驱动**分离数据与 UI，支持连接外部数据库，用户数据不锁定在平台内。近期主推 AI 能力：与 Claude Code、Cursor、Codex 等 coding agent 协作构建，以及内置 AI 员工处理业务流程。

## 核心洞见

1. **品牌护城河**：`nocobase` 品牌词 KD 仅 22，但搜索量也只有 720——心智仍属早期。NocoBase 无力阻止我们在对比类内容上切走流量；品牌词正面刚意义不大，应走"X alternative"和垂直场景词侧翼。

2. **可复制的打法**：NocoBase 的 SEO 核心策略是**内容比较文**——以"open source 工单系统 / 资产管理 / ERP / ticketing"等场景词写博客，通过"最佳开源 X 工具"列表型内容带流量。数量多（约 1,949 个主域关键词），但单词流量极低，证明**长尾列表内容有效但 SEO 稀薄**。我们可复制此打法但专注更高 CPC 的词（$8–14 段，如工作流自动化、no-code CRM）。

3. **对 Olares 最关键的词**：
   - **`airtable alternative`**（320/月，KD 18，$7.23 CPC）——体量最大的低 KD 商业意图词，NocoBase + Olares 是"自托管 Airtable"的完整答案
   - **`nocodb self hosted`**（110/月，KD 10）——极低 KD 的自托管信号词，对应 Olares 一键部署叙事
   - **`retool open source`**（70/月，KD 8，$3.63 CPC）——Retool 用户寻找开源替代，完美切入 NocoBase on Olares

4. **最大攻击面**：NocoBase **买断制授权** 是双刃剑——Community 免费吸引评估者，但 Professional $8,000 的门槛让中小团队退缩。攻击词：`nocobase pricing`（已排 #1，可在内容中插入 Olares 一键部署成本对比）；以及竞品的"subscription pricing"vs"buy once use forever"对比叙事。

5. **隐藏低 KD 金矿**：
   - `no code erp`（KD 1！月量 70，ERP 决策词 CPC 预期高）——近乎零竞争
   - `retool open source`（KD 8，70/月）——Retool 仍是最大竞品，用户找开源替代
   - `nocodb self hosted`（KD 10，110/月）——自托管意图明确，Olares 平台差异化最清晰
   - `self hosted no code platform`（KD 0，40/月）——零竞争入口词

6. **GEO 前瞻布局**：
   - `plugin based no code platform` / `extensible no code platform`——NocoBase 微内核架构的差异化词，当前搜索量近零但 AI Overview 正在索引这类精准概念词
   - `ai no code platform on premise` / `self-hosted AI business system`——AI + no-code + 本地部署三重交集，Olares + NocoBase 的未来搜索场景
   - `nocobase docker compose setup`——安装/教程前瞻词，抢 Perplexity 引用位

7. **与既有分析的关联**：与 olares-500-keywords 分析相比，NocoBase 为 Olares 补充了**no-code/low-code 垂直**中的低 KD 机会词（`airtable alternative`、`retool open source`、`nocodb self hosted`），以及**企业业务系统构建**方向（CRM、ERP、工作流自动化），这些在既有词表中属于补白方向。另，NocoBase 中文流量可观（`工单系统`、`低代码平台`、`mcp服务器推荐`），暗示有中文用户群体搜索此类工具，可考虑中文内容联动。
