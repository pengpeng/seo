## 模型理解（前置）

TRELLIS.2 是微软研究院出品的开源 3D 生成模型，以"结构化 3D 潜空间（Structured 3D Latents）"架构实现图像→3D 资产的高质量转换，直接输出 mesh + PBR 材质（粗糙度/金属度贴图），无需后处理即可导入 Blender、Unity、Unreal。4B 参数规模，在开源 3D 生成领域持续保持 state-of-the-art 水准。MIT 许可、无用量限制，相比 Meshy/Tripo/Rodin 等按次计费的订阅 SaaS 产品，本地跑的成本壁垒为零。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 图生 3D（image-to-3D）+ PBR 材质输出，开源 3D 生成天花板 |
| 许可证 | **MIT**（商用友好，无地区限制，可商业化自托管） |
| 主仓库 / 分发 | [microsoft/TRELLIS](https://github.com/microsoft/TRELLIS)（GitHub ~20k ★）；HuggingFace：JeffreyXiang/TRELLIS-image-large；ComfyUI 社区节点可用 |
| 参数 / VRAM 可跑性 | 4B 参数；推理需 ~12–16GB VRAM；**Olares One（RTX 5090 Mobile 24GB）完全可跑**；消费级 RTX 3090/4090（24GB）亦可 |
| 变体 / 型号 | TRELLIS（原版）→ TRELLIS.2 / TRELLIS 2.0（当前世代，主推）；官方 Gradio Demo 及 HF Space 可在线试用 |
| 闭源对标 | **Meshy AI**（主要竞品，按 credit 收费）、**Rodin / Hyper by Deemos**、**Tripo AI**、CSM AI |
| Olares Market | ComfyUI 已在 Olares Market 上架（TRELLIS ComfyUI 节点可一键叠加） |
| 来源 | [GitHub](https://github.com/microsoft/TRELLIS)、[HuggingFace](https://huggingface.co/JeffreyXiang/TRELLIS-image-large)、[论文 arXiv:2412.01506](https://arxiv.org/abs/2412.01506) |

## 核心洞见

**1. 搜索心智已部分建立，但主要靠版本词而非品牌词**

TRELLIS 3D（880/月）、TRELLIS 2.0（1,300/月）有稳定量，说明模型出圈程度超过多数同类开源 3D 工具；但 TRELLIS AI（KD 73）竞争较激烈，品牌独占还未形成。搜索行为仍以"直达 GitHub/HF"为主（trellis github 210/月、trellis huggingface 210/月 均为导航意图），SEO 机会集中在"图文教程+对比"内容层面而非官网排名。

**2. Olares One 完全可跑——本地叙事成立**

TRELLIS.2 推理需 ~12–16GB VRAM，Olares One 搭载 RTX 5090 Mobile（24GB GDDR7）有充裕余量；消费级 RTX 3090/4090 亦可。ComfyUI 已在 Olares Market，叠加 TRELLIS ComfyUI 节点即可一键使用。**本地部署叙事完全成立，无需为 VRAM 打折扣。**

**3. MIT 许可证是主推卖点——无限制商用**

MIT 许可证无地区限制、允许商业用途，可直接用于商业游戏资产、产品渲染等场景。竞品 Meshy/Tripo/Rodin 均为 SaaS 按量计费，无法本地化。**许可证 + 本地化是 Olares + TRELLIS 的双重护城河，可主推。**

**4. 对 Olares 最关键的 3 个词**

- `meshy ai alternative`（KD 16，月量 70）：**最低竞争的商业调研词**，对比内容落地首选，截取正在比价 Meshy 替代品的用户。
- `trellis comfyui`（KD 45，月量 70）：**引擎组合词前哨**，Olares Market 上有 ComfyUI，一步操作指南文可直接排名。
- `meshy ai pricing`（KD 28，月量 110）：**商业调研词 + 价格敏感用户**，对比文"Meshy vs TRELLIS（本地部署）"的核心锚词，Olares 免订阅角度攻击面最强。

**5. GEO 抢发窗口**

以下词近零搜索量，但语义确定、问答型，是 Perplexity/AI Overview 引用位的早期卡位机会：
- `run trellis locally` / `trellis vram` / `trellis gpu requirements`：硬件教程 FAQ
- `trellis vs meshy`：对比问句，尽早发布对比分析文
- `trellis 3d api`：开发者调用教程（GEO 抢发）
- `game asset generator ai` / `pbr material generator`：垂类场景，TRELLIS PBR 直出的差异化切入
