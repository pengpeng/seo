## 核心洞见

1. **流量/估值比严重失调**：75K 月流量但估值 $27.7 万，说明 Groq 关键词 CPC 极高（groq pricing = $16.03！），每个访客的商业价值比其他产品高 5-10 倍——这批开发者是最优质的目标用户。

2. **"grok"与"groq"严重混淆**：grok api（6,600/mo）、grok cloud（210/mo）等 X/Twitter 的 Grok 词流量有相当一部分流向 groq.com——Groq 意外受益于名字相似性，这也说明 LLM API 市场用户教育不足。

3. **groq api free tier（KD=22）是最低竞争 API 词**：开发者在寻找免费推理 API，Ollama on Olares 完全免费且无限制——Olares 写一篇"免费 LLM 推理 API 对比：Groq vs Together AI vs 本地 Ollama"可以获取多个品牌词流量。

4. **groq stock（6,600/mo）出现在 Groq 流量词里**：Groq 做了股票分析 demo，靠金融词获取流量——这是内容借力的典型策略，与 Perplexity 的做法类似。

5. **Groq 的护城河是推理速度（Groq LPU）而非模型**：所有模型词（llama、kimi k2、whisper）都只是 Groq 的"租用"内容——一旦本地硬件（Ryzen AI Max、RTX 5070）速度足够快，Groq 的优势就会消失，Olares 正处于这个临界点。
