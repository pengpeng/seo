## 项目理解（前置）

Adobe Firefly 是 Adobe 公司推出的商业 AI 生成模型套件（图像/视频/向量/音频），核心差异化为"100% 基于授权内容训练（Adobe Stock + 公有领域）+ 向付费用户提供 IP 赔偿保障"，定位为专业设计师与企业团队的合规生成工具。2026 年当前旗舰图像模型为 **Firefly Image 5**（即品牌营销名"Nano Banana 2"），并已扩展至文本到视频、图像到视频、生成式音频。其最大战场是与 Photoshop 的深度整合（Generative Fill / Generative Expand），而非单独的文本到图像 Web 产品。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 面向专业创意团队的商业级 AI 生成套件，以授权训练数据 + IP 赔偿为核心护城河 |
| 开源 / 许可证 | **闭源**，Adobe 商业 SaaS |
| 主仓库 | 无公开 GitHub（产品型 SaaS） |
| 核心功能 | 文本到图像（Firefly Image 5）、Generative Fill/Expand（Photoshop）、文本到向量（Illustrator）、文本到视频（5s 片段）、生成式音频、Custom Models API（品牌微调） |
| 商业模式 / 定价 | 免费（25 credits/月）；Standard $9.99/月（2,000 credits）；Pro $19.99/月（4,000 credits + IP 赔偿）；Pro Plus $49.99/月（10,000 credits）；Premium $199.99/月（50,000 credits + 无限视频）；或随 Creative Cloud All Apps ~$60/月 捆绑 |
| 差异化 / 价值主张 | IP 赔偿（唯一主要 AI 图像工具）+ Photoshop Generative Fill 深度整合 + 多模态一站式（图/视频/向量/音频） |
| 主要竞品（初判）| Midjourney（艺术质量）、DALL-E / GPT Image（提示准确）、Google Imagen（生态整合）、Stable Diffusion / ComfyUI（本地开源）、Canva Magic Studio、Leonardo AI |
| Olares Market | **未上架**（闭源商业产品）；开源平替 ComfyUI 已上架 ✅ |
| 来源 | [adobe.com/products/firefly](https://www.adobe.com/products/firefly.html)、[aitrendtool.com/tools/adobe-firefly](https://aitrendtool.com/tools/adobe-firefly)、[pick-right.com/tools/adobe-firefly/](https://pick-right.com/tools/adobe-firefly/) |
