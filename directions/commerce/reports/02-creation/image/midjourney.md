## 项目理解（前置）

Midjourney 是独立 AI 研究实验室（2022 年，David Holz 创立）做的**文生图**产品，2022 下半年以 Discord 机器人形态引爆生成式图像浪潮，2024 起推出 web 端（midjourney.com / alpha.midjourney.com），2025 起加入 V1 视频模型。它以**出图美学质量**著称，靠订阅制变现，**没有永久免费档、没有公开 API、内容审查严格（无 NSFW）**——这三点正是开源自托管方案（ComfyUI + FLUX）的天然攻击面。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 闭源 SaaS 文生图/视频，主打出图美学，订阅制 |
| 开源 / 许可证 | ❌ 闭源专有，不可自托管 |
| 主仓库 | 无（专有；模型权重不公开）|
| 核心功能 | 文生图（v7）、图生图/风格参考、V1 视频、web + Discord 双端 |
| 商业模式 / 定价 | 纯订阅：Basic $10 / Standard $30 / Pro $60 / Mega $120 每月；无永久免费、偶尔放开 free trial |
| 差异化 / 价值主张 | 美学质量 + 社区（Discord）+ 易用；无需本地算力 |
| 主要竞品（初判）| DALL·E、Leonardo AI、Ideogram、Google "nano banana"、Stable Diffusion / FLUX（开源）|
| Olares 平替 | ✅ [ComfyUI](/Users/pengpeng/seo/directions/market/reports/comfyui.md) 已上架 Olares Market + FLUX.2 本地权重 |
| 来源 | midjourney.com、docs.midjourney.com、公开报道（ARR/估值由需求方提供）|

## 核心洞见

1. **品牌护城河极深，通用大词全被大平台把持，正面刚无胜算。** midjourney 品牌词（246K，KD85）+ 数十个误拼变体全部霸榜第 1，自然流量估值 $818K/月，几乎全靠品牌心智；而 `ai image generator`（823K，KD92）、`nano banana`（368K，Google 新模型）、`ai photo generator`（165K）等品类超级大词 KD 均在 80+，且 SERP 被 YouTube/Reddit/巨头占据。品牌词与品类大词都不是 Olares 的战场。

2. **可复制的打法不是它的（它几乎没有 SEM/程序化打法），而是它留下的结构性空档。** Midjourney 只买自家品牌 + 一个防御词 `dalle`，付费仅 157 词、`/explore` 单页扛所有广告，**没有 guides/对比/教程落地页矩阵**。这正是 Olares 可低成本反超的地方：用"X alternative / X vs Y / how to run X locally"的对比+教程页矩阵，去接住它自然流量之外、意图更明确的替代/本地/无审查人群。

3. **对 Olares 最关键的 3 个词：`comfyui`（49,500, KD63）、`uncensored ai image generator`（6,600, KD44）、`midjourney free`（8,100, KD62）。** 第一个是平替本体大词（ComfyUI 已在 Olares Market，另有独立报告可协同）；后两个直击 Midjourney 的两大痛点——审查与付费——本地 ComfyUI+FLUX.2 天然全部满足。三词量都大、KD 明显低于品类大词，是主攻方向。

4. **最大攻击面＝审查 + 定价 + 无 API，三者皆本地方案强项。** 审查：`uncensored ai image generator`(6,600)、`ai image generator no filter`(1,900)、`midjourney nsfw`(320,$4.17)；定价：`midjourney free`(8,100)、`is midjourney free`(4,400,$7.53)、`midjourney pricing/price/cost`(合计近 1 万,$5+ CPC)；接口：`midjourney api`(1,300) 官方不开放。叙事应直击："本地出图 = 无限、免费、无审查、有 API、数据归你"。

5. **隐藏低 KD 金矿：ComfyUI 部署词族。** `comfyui docker`(480,KD21)、`comfyui workflow`(1,900,KD27,$4.26)、`comfyui vs automatic1111`(260,KD22)、`comfyui local`(90,KD0)——这批词量不小、竞争极低、意图就是"想把 ComfyUI 跑起来/托管起来"，与 Olares 一键部署 + GPU 就绪的能力精准咬合，是转化最高的隐藏矿脉。

6. **GEO 前瞻布局：** `open source midjourney`、`self hosted image generation/generator`、`open source image generation`、`flux local/locally`、`self hosted stable diffusion` 一批 KD=0、近零量的词，语义与 Olares 完美契合。提前发权威内容占位，抢 AI Overview / Perplexity 引用位。
