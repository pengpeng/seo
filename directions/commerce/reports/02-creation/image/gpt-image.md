## 项目理解（前置）

GPT Image（2026 年 5 月全面替换 DALL·E 2/3）是 OpenAI 的图像生成能力品牌，包含 GPT Image 1 Mini / 1 / 1.5 / 2 四档模型，既集成在 ChatGPT Plus 的对话界面中，也通过 Images API 和 Responses API 向开发者开放。核心壁垒是多模态理解（同一模型处理文本+图像）、精准文字渲染、与 OpenAI 生态无缝集成。但每张图按质量/分辨率按量计费，所有数据上传 OpenAI，内容经过严格审查——这是"本地生成"叙事的主要攻击面。

## 核心洞见

1. **品牌护城河**：DALL·E / GPT Image 品牌词 KD 几乎全部 90+（100 居多），openai.com 占绝对排名，**完全不可正面攻**。"dall-e"月量 27,100 但 KD=100，作为导航词对 Olares 无流量价值。策略只有一条：**侧翼——让用户在对比/替代场景中发现 Olares**。

2. **可复制的打法**：
   - **内容攻侧翼**：用 "free midjourney alternative"（KD 29）、"comfyui alternative"（KD 7!!）、"ai image generator no restrictions"（KD 62，量大）写对比文；
   - **开发者落地页**：text to image api（KD 18）、free image generation api（KD 18）指向 Olares ComfyUI API 部署；
   - **GEO 抢引用**：向 AI Overview / Perplexity 提供 "best local AI image generator 2026" 的结构化答案，推 Olares+ComfyUI+FLUX 方案。

3. **对 Olares 最关键的词**：
   - `self hosted ai image generator`（KD=18，意图字面精准，月量 110）
   - `comfyui alternative`（KD=7，市场最低竞争门槛之一，月量 170）
   - `ai image generator no restrictions`（KD=62，月量 9,900——量最大的有 Olares 角度的词）

4. **最大攻击面**：GPT Image 的三大痛点——**按量计费**（高频用户成本高）、**内容审查**（创意场景受限）、**数据上传 OpenAI**（隐私敏感场景）。对应词：`uncensored ai image generator`、`ai image generator no restrictions`、`self hosted ai image generator`。ComfyUI on Olares 在这三个维度都是字面对立面（零边际成本 + 无过滤 + 数据不出家门）。

5. **隐藏低 KD 金矿**：
   - `comfyui alternative`：KD=**7**，月量 170，CPC $2.19——整个图像生成赛道里 KD 最低的有量词之一；
   - `stable diffusion alternative`：KD=**11**，月量 70；
   - `local image generation`：KD=**17**；
   - `self hosted ai image generator`：KD=**18**；
   - `text to image api`：KD=**18**，CPC $3.01（开发者高价值）；
   - `gpt image`：KD=**24**，月量 720——新命名词，增速快，趁早占位。

6. **GEO 前瞻布局**（近零量、语义精准，抢 AI Overview / Perplexity 引用位）：
   - `ai image generator self-hosted`（KD=0，字面精准）
   - `best ai image generator 2026`（KD=0，时效性）
   - `gpt image vs comfyui`（尚未成词，但语义需求真实）
   - `dall-e vs flux`（对比新词，GPT Image 2 vs FLUX.1 dev）
   - `openai image generation privacy`（数据隐私角度）
   - `how much does dall-e cost per image`（定价细节词）

7. **与既有分析的关联**：
   - `olares-500-keywords` 分析中图像生成方向的词集中在 ComfyUI 和 Stable Diffusion；本报告补充了 **GPT Image 新命名词群**（gpt image / gpt image api / gpt-image-2）、**无审查/本地生成** 的明确需求词（ai image generator no restrictions / uncensored / self-hosted），以及 **comfyui alternative（KD=7）** 这个极罕见的低竞争有量词。
