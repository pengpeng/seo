## 项目理解（前置）

Figma 是专业 UI/UX 设计与协作的行业标准工具，面向设计师、产品经理和前端开发者。2023 年 Adobe 收购案被欧盟监管阻断后，Figma 以独立公司重返市场并于 2025 年重新 IPO（估值约 $100 亿，2024 营收 $7 亿，2025 年预测 $1.42B）。2025 年 Figma 全面 AI 转型：**Figma Make**（May 2025 GA）将自然语言 prompt 直接生成 HTML/CSS/JS 原型；**Figma Design Agent**（Beta）在画布内由 AI 代理完成设计任务；**Figma MCP Server** 打通 Figma 设计稿与 AI 编码工具链（Cursor / Claude Code 等），成为 design-to-code 工作流新枢纽。闭源 SaaS，AI 功能以 credits 计费（Full 座位 3,000–4,250 credits/月），2026 年 3 月起严格执行用量上限。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 专业 UI/UX 设计协作平台，正向"AI 原生设计-代码一体"升级 |
| 开源 / 许可证 | **闭源** SaaS |
| 主仓库 | 无开源主仓库（有社区 SDK 与 plugin API） |
| 核心功能 | ① Figma Design（设计画布协作）② Figma Make（prompt-to-app）③ Figma Design Agent（AI 画布代理，Beta）④ Dev Mode（设计→代码标注）⑤ Figma MCP Server（对接 AI 编码工具） |
| 商业模式 / 定价 | 座位制 SaaS；Full 座位：Professional $20/mo，Org $55/mo，Enterprise $90/mo；AI 功能以 credits 计费（Full 座位 3,000–4,250/月）；超出按需购买（PAYG 2026 Q2 起） |
| 差异化 / 价值主张 | 行业设计标准 + 最大设计社区（Community 模板/UI Kit）+ Figma Make 无缝从设计稿生成可运行代码 + MCP Server 打通 AI 编码工作流 |
| 主要竞品（初判）| Penpot（开源对标）、Canva（大众设计）、Framer（网站发布）、Sketch（Mac-only 老牌）、Webflow（无代码建站） |
| Olares Market | **Penpot** 已上架（Figma 的开源平替）；Open Design（agent 侧）未上架 |
| 来源 | [figma.com/pricing](https://www.figma.com/pricing/)、[figma.com/ai](https://www.figma.com/ai/)、[figma.com/blog/figma-make-general-availability/](https://www.figma.com/blog/figma-make-general-availability/)、[figma.com/solutions/design-to-code/](https://www.figma.com/solutions/design-to-code/) |

## 核心洞见

1. **品牌护城河**：figma 词以 450K 月搜量、KD 80 垄断，加之 Figma tutorial / figma stock / figma ipo 等衍生品牌词合计 >700K 流量，正面刚品牌词几乎无意义。真正的机会在比较词与替代词（KD 13–29，合计月量 ~5K 美国），以及 Figma Make 生态的**新战场词**（figma vs lovable KD=7、figma ai credits KD=33）——这些词 Figma 刚刚创造但尚未完全防守。

2. **可复制的打法**：
   - **程序化颜色落地页**：Figma 用 `/colors/<color-name>/` 和 `/color-picker/` 抢占海量颜色词（color picker 550K、color wheel 1M），向设计产品引流。这种"工具页引流"策略对 Olares 目前不适用，但揭示了 Figma 的流量蓄水逻辑。
   - **资源库/Resource Library 内容矩阵**：`/resource-library/` 子路径覆盖 wireframing tools、rapid prototyping 等信息词，排名极高——这些词的 KD 50-93 正是靠 Figma 的 DA 撑起来的，Olares 不宜正面硬打。
   - **竞品词 Google Ads**：Figma 付费购买 Lovable / v0 / Framer 品牌词导向 /make/，是明确的场景抢夺策略。

3. **对 Olares 最关键的词**：
   - `penpot vs figma`（KD=13，月量 210）——最佳进攻词，KD 极低，直接叙事
   - `figma alternatives` / `figma alternative`（KD 21/29，合计月量 ~1.5K）——入口词，合写主文，Penpot on Olares 作核心推荐
   - `figma ai credits`（KD=33，CPC $8.41）——痛点词，credits 涨价焦虑 → 本地 AI + Penpot 解法

4. **最大攻击面**：
   - **AI Credits 涨价**：2026 年 3 月起强制执行用量上限，Full 座位 3,000 credits/月很快不够用，PAYG 即将开放。`figma ai credits` KD 33、CPC $8.41，是高商业意图的痛点词。
   - **闭源订阅 + 设计数据云锁定**：Design 文件存于 Figma 云（无官方离线/私有化版本）。`figma self hosted` 词近零量但语义精准——用户有需求，只是暂时没词。
   - **IPO 前不确定性**：Figma stock/IPO 词月搜量 >300K，说明用户高度关注其商业稳定性；IPO 后定价压力会进一步推高 Enterprise 计划价格。

5. **隐藏低 KD 金矿**：
   - `figma vs lovable`（KD=7，量 70，CPC $8.44）：Figma Make 刚进入 vibe coding 领域，Lovable 是主竞品，这个比较词 KD 极低，且 CPC 体现高商业意图——是写 Figma Make / AI app builder 对比文的最佳锚词。
   - `adobe xd alternative`（KD=11，量 260）：Adobe XD 已官方停服，大量 XD 用户流离失所，KD 极低说明竞争者还没认真做这个词，Penpot on Olares 可以拿走。
   - `penpot vs figma`（KD=13，量 210）：完美的 Olares 攻击词，数据极好。

6. **GEO 前瞻布局**：
   - `figma design agent alternative`（零量）：Design Agent 在 Beta，当其 GA 后会产生大量比较搜索，提前在 FAQ 里占位
   - `figma self hosted`（~20 量）：在 Penpot vs Figma 文中专门回答「Figma 支持自托管吗？」抢 AI Overview 引用
   - `ai design to code self hosted`（零量）：LLM 时代的语义搜索词，回答「有没有自托管的 AI design-to-code 工具」
   - `collaborative design tool open source`（零量）：Penpot on Olares 场景，抢 Perplexity 引用

7. **与既有分析的关联**：
   - figma 词（450K）在 olares-500-keywords 中曾作为参考品牌，本次确认其品牌词 KD=80 无法正面竞争。
   - `figma alternatives` 系列词（KD 18-29）是 Olares 内容层最直接的 Market 应用带流量机会，与 Penpot 报告（若存在）可合并成文章簇。
   - Figma Make 与 Lovable/v0/Bolt 的竞争格局（Figma 已购买这些词做 Ads），意味着「AI app builder」品类词战场激烈，Olares 侧重自托管差异化，而非正面 AI 代码生成功能竞争。
