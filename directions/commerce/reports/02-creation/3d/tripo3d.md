## 项目理解（前置）

Tripo3D 是 Tripo AI（前身 VAST / 深圳）推出的 AI 3D 生成平台：输入一段文字或一张图片，几十秒生成带贴图、可直接进创作/打印/游戏管线的 3D 模型。它对标 Meshy、Rodin(Deemos)、Sloyd，是当前文/图生 3D 商业赛道两强之一。核心变现是 credits 订阅 + API（platform.tripo3d.ai）。对 Olares 的意义在于：它做的事，开源侧的 **TripoSR / Hunyuan3D / TRELLIS** 已能在本地 GPU 上自托管跑通——同样是"文/图→3D"，但无 credits、模型与数据归用户。

| 项目 | 内容 |
|------|------|
| 一句话定位 | AI 文/图生 3D 模型生成 SaaS（生成即带贴图、可导入 Blender/游戏/3D 打印） |
| 开源 / 许可证 | **闭源商业**（平台/模型不开源；早期 TripoSR 权重曾以研究许可放出，非产品本体） |
| 主仓库 | 无公开产品仓库；开源平替见下 |
| 核心功能 | 文生 3D、图生 3D、AI 贴图/纹理、DCC Bridge（导入 Blender）、在线 3D 工具（STL/FBX 查看器、格式转换）、API |
| 商业模式 / 定价 | Freemium + credits 订阅 + 按量 API（platform.tripo3d.ai） |
| 差异化 / 价值主张 | 生成质量/速度领先、贴图好、创作管线集成（Blender/游戏资产）、活跃 UGC 模型库 |
| 主要竞品（初判）| Meshy、Rodin(Deemos/hyper3d.ai)、Sloyd、3D AI Studio、Poly.cam |
| 开源平替 | **Hunyuan3D**（腾讯混元，✅ 已在 [Olares Market](/Users/pengpeng/seo/directions/market/reports/hunyuan3d.md)）、**TripoSR**、**TRELLIS**、Stable Fast 3D、InstantMesh；配合 **ComfyUI**（✅ 已上架）跑工作流 |
| 来源 | tripo3d.ai / platform.tripo3d.ai / market.olares.com、SEMrush us |
