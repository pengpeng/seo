## 项目理解（前置）

Meshy（Meshy AI）是一款闭源 SaaS，用文字提示或单张图片在几十秒内生成带贴图的 3D 模型（含自动 PBR 纹理、自动绑定/rigging、多格式导出 GLB/FBX/OBJ/STL/USDZ），面向独立游戏开发者、3D 打印玩家、AR/VR 与设计师。它的价值主张是"不会建模也能出 3D 资产"——把传统需要数小时的建模压缩到分钟级。差异化在于成品度高（贴图 + 拓扑 + rigging 一条龙）、格式全、有社区素材库与 Blender/Unity/Unreal 插件。软肋是**按 credits 订阅计费**且生成受额度约束，商用授权与数据都在云端。

| 项目 | 内容 |
|------|------|

| 主仓库 | 无（提供 REST API + 插件，非开源） |
| 核心功能 | 文/图生 3D、自动 PBR 贴图、自动 rigging、多格式导出、素材社区、DCC 插件 |

## 核心洞见

1. **品牌护城河：中等，正面刚品类词无胜算，但品类尚未定型。** Meshy 的自然流量高度依赖品牌词（meshy ai 27,100 / meshy 18,100 及一堆误拼变体全霸榜第 1）与通用品类词（ai 3d model generator 6,600、image to 3d model 6,600）。品牌词抢不动也不必抢；但整个 "AI 3D 生成" 品类词的 KD 普遍偏高（55-66）且各家未真正垄断——Meshy 自己也只是排到第 1-4 且靠首页硬撑，说明品类心智仍在争夺期，**内容有机会插队**。

2. **可复制的打法有两条，都很适合 Olares 复刻：**
   - **免费 3D 工具页矩阵**：`/3d-tools/file-converter/glb/to/stl`（KD8）、`3mf to stl`（KD19）、`.obj viewer`（KD25）、`stl file viewer`（KD34）——这些不是核心产品，却用近零竞争的工具词把精准 3D 用户（打印玩家、开发者）钓进站。Olares 完全可做同构的"在线/本地 3D 文件转换与查看"工具页。
   - **用户素材 `/tags/` 页 + 多语言子目录**：`batmobile free glb model`（KD12）、`3d model roman gear roblox`（KD0）、`dreadlocks 3d`（KD12）等游戏/打印素材长尾，加上 `/cs` `/zh` `/tr` 多语言页借力大词（捷克语 `3d modelář` 借 27,100 的量排到第 6）。

3. **对 Olares 最关键的 3 个词：`image to 3d model`（6,600, KD43）、`混元3d`（4,400, KD47）、`meshy ai alternative`（70, KD16）。** 分别对应三种打法：品类主词做 **TRELLIS/Hunyuan3D 本地图像转 3D 深度教程**（KD 是所有大词里最低的）；中文词吃 **Hunyuan3D（腾讯出品）中文人群**（Meshy 都在花钱买 `混元3d`）；替代词做 **"开源自托管 Meshy 替代"总览页**（KD16、几乎零竞争）。

4. **最大攻击面：credits 订阅计费。** `meshy cancel subscription`（260, KD15）、`meshy pricing`（30, KD0）、`meshy ai pricing`（110, KD28）、`is meshy ai free`（170）密集出现，退订词甚至挤进自然流量前列——用户对"按 credits 计费 + 额度限制"高度敏感。Olares 差异化直击此处："Hunyuan3D / TRELLIS 自托管在你自己的 Olares 上，本地无限次生成、不烧 credits、不订阅、模型权重与作品都归你；TRELLIS 为 MIT 许可，商用最省心。"

5. **隐藏低 KD 金矿：开源模型的版本/工具长尾几乎无人做。** `hunyuan3d-polygen`（880, KD24）、`hunyuan3d-3`（260, KD22）、`hunyuan3d 3.0`（140, KD20）、`dreamgaussian`（170, KD21）、`comfyui-hunyuan3d-2-1`（260, KD32），以及一整排 KD0 的 `hunyuan3d comfyui` / `trellis 3d comfyui / free / api`。谁先发"在 Olares 上跑 X 模型/ComfyUI 工作流"的权威教程，谁就吃下这批高意图、零竞争的部署词。新竞品 `hitem3d`（2,400, KD14）同理。

6. **最强新信号——Meshy 花钱买自己的开源替代品名字。** 付费词里赫然出现 `hunyuan3d-2`、`trellis 2`、`shap-e`、`混元3d`——Meshy 在 Google Ads 上竞价截流这些开源模型的搜索者。这等于官方盖章："用户在主动搜开源本地 3D 生成，且这批流量值得掏广告费。" 对 Olares 是免费的市场验证：**做 Hunyuan3D/TRELLIS 自托管内容，需求真实存在、且被闭源方视为威胁。**

7. **GEO 前瞻布局：** `open source text to 3d`、`open source image to 3d`、`generative 3d`、`3d asset generator`、`hunyuan3d comfyui`、`trellis 3d free/api` 目前近零量但语义与 Olares 完美契合。提前发权威内容占位，抢 AI Overview / Perplexity 在"开源本地 3D 生成"话题上的引用位。

8. **与既有分析的关联：** 本报告延续上一版 Meshy 洞见（工具页低 KD 取量、image-to-3d 是最大机会、中文市场信号、meshy pricing KD0），并新增三点——(a) Meshy 付费买开源竞品名的防御信号；(b) Hunyuan3D 版本/ComfyUI 长尾金矿；(c) `meshy cancel subscription` 退订词作为迁移入口。建议在 `olares-500-keywords` 词表中补入 "AI 3D 生成 / 本地 3D 模型" 子品类，与「本地图像/视频生成」并列。
