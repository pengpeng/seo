## 项目理解（前置）

Granola 是一款 AI 会议笔记工具，运行于用户本机（macOS/Windows），通过捕获系统音频实现会议转录，**无需 bot 入会**——这是它与 Otter、Fireflies 等主流竞品最核心的差异。Granola 采用"人机协作笔记"模式：用户在会议中手写要点，AI 结合转录自动结构化补全，产出远胜纯转录的可用笔记。2026 年 3 月完成由 Index Ventures 领投的 $125M C 轮，估值 $1.5B（较 10 个月前 6 倍增长），企业客户已包含 Vanta、Cursor、Mistral AI、Lovable 等。

|------|------|

| 开源 / 许可证 | **闭源商业产品**（非开源） |
| 主仓库 | 无公开 GitHub |
| 核心功能 | 本地音频转录（无 bot）、AI 结构化笔记、日历集成、会议简报（Brief）、多平台（Zoom/Meet/Teams/面对面）、MCP 集成（Claude/ChatGPT/Cursor）、CRM 推送（HubSpot/Attio/Linear） |
| 商业模式 / 定价 | Free（25 条历史上限）/ Business $14/用户/月（无限历史+集成）/ Enterprise $35+/用户/月（SSO+组织级隐私控制） |

| 主要竞品（初判）| Otter.ai、Fireflies.ai、Fathom、tl;dv、Read AI、Notta、MeetGeek、Jamie AI |

## 核心洞见
