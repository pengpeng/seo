## 核心洞见

1. **STT API CPC=$125.23 是全部报告中最高 CPC 词**：只有 20 次/月搜索但每次点击价值 $125，说明企业为语音 API 采购愿意付出极高代价——Whisper on Olares 的零成本方案对每个 STT API 用户都是巨大的成本节约。

2. **Deepgram 从 STT 扩展到完整"语音 AI 平台"**：deepgram tts（TTS）、deepgram voice agent api（语音 Agent）、deepgram flux（新产品）——Deepgram 正在构建完整的语音 AI 工具链，与 ElevenLabs 形成竞争，Olares 的本地语音方案可以覆盖同样的场景。

3. **deepgram api key（CPC=$14.91）和 deepgram vs assemblyai（CPC=$18.56）都是高 CPC 词**：STT API 市场的所有词 CPC 都异常高，这是因为企业 STT 服务的年合同价值很高——写"降低 STT 成本：Whisper on Olares"这类内容，即使排名靠后也有极高的 ROI。

4. **Deepgram 构建了 Voice AI App 目录（speechgen、podcastle、ttmaker）**：这是一个典型的"借合作伙伴词获取流量"策略——Olares Market 上的语音 AI 应用（TTS、STT、转录）可以用类似策略，用热门语音工具名词来获取流量。

5. **pipecat（1,900/mo，KD=39）是语音 AI Agent 框架词**：Deepgram 集成了 Pipecat 开源框架——说明实时语音 Agent 是下一个热门场景，Olares 如果提供 Pipecat + Whisper + Kokoro TTS 的完整本地语音 Agent 方案，可以在这个新兴市场占先位。
