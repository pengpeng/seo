## 项目理解（前置）

Cartesia 由 Stanford AI Lab 研究人员创立，专注于基于自研 State Space Model（SSM）架构的实时语音生成。旗舰产品 Sonic（当前版本 3.5）是已知商业 TTS 中延迟最低的模型（首字节 <90ms，Turbo 模式 ~40ms），远低于 ElevenLabs（~250ms）等竞品。Cartesia 已完成约 $191M 总融资，投资方包括 a16z、Index Ventures 和 Lightspeed，快速进入 voice agent 基础设施赛道。产品线包括 Sonic（TTS）、Ink（STT）、Line（voice agent 平台），并提供专业声音克隆与情绪控制。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 超低延迟 TTS/STT API + voice agent 平台，主攻开发者和企业实时语音场景 |
| 开源 / 许可证 | 闭源商业产品；无公开模型 repo |
| 主仓库 | 无公开 GitHub 仓库（SDK 在 github.com/cartesia-ai） |
| 核心功能 | <90ms TTS 延迟（Sonic 3.5）、实时流式传输（WebSocket/SSE）、Instant/Pro 声音克隆、情绪控制、40+ 语言、STT（Ink）、voice agent 平台（Line） |
| 商业模式 / 定价 | 信用点制：Free（$0，2万 credits）→ Pro（$5/月，10万）→ Startup（$49/月，125万）→ Scale（$299/月，800万）；TTS 约 1 credit/字符；Enterprise 定制 |
| 差异化 / 价值主张 | SSM 架构使延迟远低于 Transformer 竞品；情感自动推断；可设备端运行（on-device）；企业级合规（SOC2 Type II, HIPAA, GDPR） |
| 主要竞品（初判）| ElevenLabs、Deepgram、PlayHT、Hume AI、OpenAI TTS |
| Olares Market | Fish Speech（开源 TTS 平替）已上架 |
| 来源 | [cartesia.ai](https://cartesia.ai)、[cartesia.ai/sonic](https://cartesia.ai/sonic)、[cartesia.ai/blog/seed](https://cartesia.ai/blog/seed)、[indexventures.com](https://www.indexventures.com/perspectives/building-the-next-generation-of-real-time-ai-models-our-investment-in-cartesia/)、[docs.cartesia.ai/pricing](https://docs.cartesia.ai/pricing) |

## 核心洞见

1. **品牌护城河**：不宜正面刚。品牌词（cartesia/cartesia ai/cartesia sonic-3）合计月量 ~8,000，且 Cartesia 已全部排名第 1，KD 在 48-65 之间——这是成熟品牌心智。Olares 不应争品牌词，应走侧翼：攻「X alternative」「self-hosted TTS」「local TTS」等意图词，让 Fish Speech 成为搜索者找到 Olares 的入口。

2. **可复制的打法**：Cartesia 已验证了**程序化 SEO 落地页**打法——`/languages/`（按语言，如 japanese/spanish text to speech）、`/voices/`（按声音类型，如 british-accent、anime-voice、male-voice）、`/learn/`（教程类，如 how to use TTS on Discord）。这套打法让他们用低 KD 的长尾词（KD 6-27）悄悄蚕食语音类消费词流量，不和 ElevenLabs 正面硬拼大词。Olares 可借鉴：按语言/场景/模型做 Fish Speech 的程序化落地页。

3. **对 Olares 最关键的词**：
   - `elevenlabs alternative`（480, KD 9）——量最大且 KD 最低的替代词，ElevenLabs 和 Cartesia 是同一赛道玩家，同篇文章可涵盖两者；Fish Speech on Olares 作为 self-hosted free 选项
   - `cartesia vs elevenlabs`（110, KD 20, CPC $17）——高 CPC 说明决策意图强，对比文是最直接的内容切入
   - `local tts` / `self hosted tts`——量小（110 / 20）但意图极精准，Olares + Fish Speech 是最完整的落地答案，适合 AI Overview/Perplexity 引用

4. **最大攻击面**：**定价墙 + 声音数据隐私**。Cartesia Free 套餐仅 2 万 credits（约 20 分钟 TTS），商业使用需至少 $5/月；专业声音克隆需 $49/月以上。更重要的是：声音数据上云处理，对企业和隐私敏感用户是隐患。Fish Speech on Olares 的叙事：「声音是最私密的生物特征数据——不应该上传给任何人」。

5. **隐藏低 KD 金矿**：
   - `cartesia pricing`（170, **KD=5**）：极低竞争的决策词，CPC $11 说明买家意图强；一篇 Cartesia 定价评测文 + Fish Speech 成本对比，可快速排名
   - `japanese text to speech`（880, KD=6）、`girl ai voice`（1300, KD=14）：KD 极低的品类词，Cartesia 靠程序化页面已在排名 4-9 位，Fish Speech 多语言支持同样可以攻此类词
   - `streamng tts`（20, KD=0, CPC $6.28）：技术词，零竞争，高商业价值

6. **GEO 前瞻布局**：以下词月量接近零但语义精准，建议在文章 FAQ/前瞻段落中覆盖以抢 AI Overview 和 Perplexity 引用位：
   - `self hosted tts`、`local tts open source`、`tts without api subscription`
   - `fish speech vs cartesia`、`fish speech on local server`
   - `low latency tts self hosted`、`voice cloning without cloud`
   - `cartesia alternative open source`

7. **与既有 `olares-500-keywords` 的关联**：Voice TTS 赛道在 olares-500 词表中为空白——Cartesia 报告可补充「TTS API 替代」这一整个词族。与已有的 ElevenLabs 报告（同目录）形成联动：可共用「elevenlabs alternative」「open source tts」类词，在同一篇替代文章里同时覆盖 Cartesia 和 ElevenLabs，Fish Speech on Olares 作为唯一 self-hosted 方案对比两者。
