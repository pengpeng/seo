## 核心洞见

1. **speech to text api CPC=$126.55 是所有报告集里最高 CPC**：这个词的广告价值极其高，说明企业为 STT API 服务付出极高成本——Whisper on Olares 的零成本方案对这批用户有极大吸引力，每个转化的价值远超其他场景。

2. **AssemblyAI 的 SEO 极弱**（rank 157,137，只有 11.6K 月流量）：几乎完全靠品牌词，没有任何功能/场景内容营销。这意味着 STT 领域的内容空间极大，Olares 发布相关内容几乎没有 AssemblyAI 的竞争。

3. **assemblyai pricing（KD=11）是 STT 领域里最低竞争的定价词**：70/mo 搜索量很小，但这批用户在做采购决策，转化价值极高——一篇"AssemblyAI vs Whisper 本地部署成本对比"完全可以排第一。

4. **AssemblyAI 靠博客"借力排名"获取非品牌词流量**：DALL-E 2（2,400/mo）、diffusion models（4,400/mo）、diarization（2,400/mo）——他们用 AI 技术解说文章截获了完全不相关领域的流量，这是内容多元化的典型策略。

5. **本地 STT 市场目前几乎没有内容**：whisper self hosted（30/mo，0量）、speech to text self hosted（20/mo）——这类词量极小是因为市场教育不足，不是因为需求不存在——Olares 发布"本地 Whisper 安装指南"可以成为这个品类的第一定义者。
