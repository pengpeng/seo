## 项目理解（前置）

SoundHound AI（Nasdaq: SOUN）是一家双线经营的语音 AI 公司：**消费端**做"哼唱识歌"应用（Shazam 竞品，月活数千万，`music.soundhound.com`）；**企业端**提供 Houndify 白标 Voice AI 平台与 Amelia 企业 AI 代理，深耕汽车、餐饮（电话/驾车点餐）、金融、医疗垂直行业。其核心技术主张是"Speech-to-Meaning"——一步完成语音转意图，跳过传统 ASR→NLU 两步流程，宣称速度与准确率领先。FY2025 总收入 $168.9M（YoY +99%），但仍亏损（GAAP 净亏 $14M，调整后 EBITDA 亏 $58.4M），现金储备 $269M。竞品 SEO 角度的关键矛盾：它是**闭源商业 API，无本地部署选项，企业定制定价不透明**——恰好是 Olares + CosyVoice/Whisper 的切入缝隙。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 闭源企业级对话 AI 平台（ASR + NLU + TTS + Voice Commerce），兼有消费级音乐识别 App |
| 开源 / 许可证 | 闭源商业产品；Houndify SDK 部分开放但核心引擎专有 |
| 主仓库 | 无主开源仓库（GitHub 官方账号零实质开源，star 级别忽略不计） |
| 核心功能 | Speech-to-Meaning 引擎（ASR+NLU 合一）、自定义 TTS（30+ 语言）、Houndify 白标平台、Amelia 企业 AI 代理（原 IPsoft Amelia）、Smart Answering（电话 AI）、Smart Ordering（餐饮点单）、Dynamic Drive-Thru |
| 商业模式 / 定价 | 纯 B2B；三条线：版税（OEM 硬件集成）+ SaaS 订阅（Smart Answering 等）+ 语音商业交易分成；无公开价格表，按部署规模/行业/定制复杂度议价 |
| 差异化 / 价值主张 | 白标（客户品牌不被 Google/Amazon 淡化）、噪音环境鲁棒（车内/餐厅）、跨垂直行业集成（POS/CRM/DMS）、端云一体、数据主权声称（私有域可选） |
| 主要竞品（初判）| Cerence（车载）、Microsoft Nuance（医疗/企业）、Google Dialogflow、Amazon Alexa for Business；开源层：OpenAI Whisper（STT）、CosyVoice / Coqui / Piper（TTS） |
| Olares Market | 未上架（闭源商业，不可自部署）；Olares 上的平替：Whisper（STT）、CosyVoice（TTS） |
| 来源 | [官网 Platform](https://www.soundhound.com/voice-ai-products/platform/)、[FY2025 业绩稿](https://investors.soundhound.com/news-releases/news-release-details/soundhound-ai-reports-record-annual-revenue-169-million-nearly)、[10-K 2025](https://investors.soundhound.com/static-files/4cfd0abf-2ce8-485e-bf17-dc3167ef3513)、[CheckThat.ai](https://checkthat.ai/brands/soundhound-ai) |
