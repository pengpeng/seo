## 项目理解（前置）

Luma AI 是硅谷（Palo Alto）2021 年成立的前沿生成式 AI 公司，最初做手机端 NeRF / 高斯泼溅的照片级 3D 捕捉，2024 年 11 月推出旗舰平台 **Dream Machine**（Web + App，30M+ 用户）转向 AI 视频生成而爆发。核心是"reasoning video model"系列 Ray（Ray2 → Ray3 → Ray3.14），号称质量 benchmark 高于 Sora 2、与 Google Veo 3 同级；另有 **Genie**（文本秒级生成 3D 物体）、**Photon**（图像生成）、**Luma API**（开发者接入）。它给创作者/影视广告工作室提供"从文字/图片到电影级视频"，走订阅 + 按秒计费 API 的闭源模式。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 多模态生成 AI（视频 Dream Machine / 3D Genie / 图像 Photon），面向创作者与影视广告工作室 |
| 开源 / 许可证 | ❌ 闭源，模型不公开权重；仅通过 Web / App / API 使用 |
| 主仓库 | 无开源本体（历史有 `lumalabs/luma-api` 等 SDK 客户端库，非模型）|
| 核心功能 | ① 文/图生视频（Ray3.14，1080p）② 视频改写 video-to-video ③ 文本秒级生成 3D（Genie）④ 图像生成（Photon）⑤ NeRF/高斯泼溅 3D 捕捉 |
| 商业模式 / 定价 | 订阅制（免费额度 + 付费档）+ 按秒计费 API；Dream Machine 2024/12 已 $8M 年化收入 |
| 差异化 / 价值主张 | "reasoning video model"、原生 HDR、时序稳定；30M+ 用户品牌心智；Adobe/AWS 集成 |
| 主要竞品（初判）| Runway、OpenAI Sora、Kling、Google Veo、Pika、Vidu；3D 侧 Meshy/Tripo/TRELLIS |
| Semrush 印证竞品 | invideo.io、leonardo.ai、synthesia.io、veed.io、vidu.com、renderforest、steve.ai、deevid.ai |
| Olares Market | 无 Luma 本体（闭源）；平替 **Wan / TRELLIS 经已上架的 ComfyUI 本地跑** |
| 来源 | lumalabs.ai 官网、CNBC 2025-11-19、BusinessWire 2026-01-26、Silicon Valley Investclub |

## 核心洞见

1. **品牌护城河极强，正面刚品牌词无胜算，但内容侧几乎全空。** Luma 自然流量 11 万/月、估值 $181K/月，但 90%+ 来自品牌词与海量误拼变体（luma ai / dream machine / lumalabs / lumalabai…全部霸榜第 1），且付费投入近乎为零（78 词）。这是纯品牌心智型结构——Olares 无法也不必去抢 `luma ai`（KD81）、`luma`（KD88）这类词。

2. **可复制的打法＝补 Luma 的"内容空白"，而非模仿它的打法。** 与 Lovable 那种 `/guides/` + 买大词的程序化打法不同，**Luma 根本没有内容矩阵**（docs 站 0 流量、无 blog/use-case 落地页贡献流量）。这反而是机会：Luma 把品类信息词（`luma dream machine text to video` 4,400、`ai video generator` 品类大词）都靠首页硬排，Olares 可用"Wan/TRELLIS 本地部署教程 + X alternative 对比页"矩阵去吃**开源/本地/替代**这批 Luma 完全没覆盖的长尾。

3. **对 Olares 最关键的 3 个词：`wan 2.2`（14,800, KD61）、`local ai video generator`（390, KD26）、`luma ai alternative`（40, KD14）。** 三层递进——`wan 2.2` 是本地视频生成的头号引擎词（量最大），`local ai video generator` 是"本地生成"意图的品类词（KD 低、契合满分），`luma ai alternative` 是直接替代词（KD14）。配合已上架的 ComfyUI，Olares 能同时吃"模型部署词 + 本地品类词 + 替代词"三层。

4. **最大攻击面＝Luma 的按秒计费 + 额度 + 水印。** `luma ai free`（260）、`luma dream machine pricing`（90）、`free/unlimited ai video generator`（590/480）、`ai video generator without credits`（30）密集出现，说明视频用户对"计费/额度/水印"高度敏感。Olares 的叙事直击此处：**本机独显跑 Wan＝无 credits、无水印、无时长限制、生成不出设备**——这是对"按秒付费云 API"模式的正面颠覆。

5. **隐藏低 KD 金矿集中在 ComfyUI 部署词与竞品替代词。** `comfyui wan`（210, KD29）、`wan video wrapper`（480, KD27）、`wan 2.2 install`（170, KD33）、`wan video to video`（140, KD19）、`ltx video`（2,400, KD31）、`kling ai alternative`（170, KD5）、`sora alternative`（210, KD11）——量不小且 KD 远低于品类大词（`ai video generator` KD92 打不动），是最该优先落地的一批。3D 侧 `luma ai 3d`（KD17）、`luma genie`（KD27）同理。
