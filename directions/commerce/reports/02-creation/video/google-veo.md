## 项目理解（前置）

Google Veo（现 Veo 3.1）是 Google DeepMind 的旗舰视频生成模型，于 Google I/O 2025 发布 Veo 3，业界首款原生同步音频（对话、音效、环境音）的商业视频模型，支持文本生成视频、图像生成视频、场景延伸与相机控制，输出分辨率最高 4K。配套工具 **Google Flow**（前身 VideoFX）是专为 Veo 定制的 AI 影视创作平台，提供 8 秒片段拼接的时间线界面。Veo 集成在三个入口：①Google Flow（付费订阅）、②Gemini App（AI Pro/Ultra 订阅）、③Gemini API / Vertex AI（按秒 API 计费）。所有视频均嵌入 SynthID 数字水印。

## 核心洞见

### 1. 品牌护城河（能否正面刚）

**无法正面刚，但侧攻有大量低 KD 机会。** `veo 3`（KD=73）、`google veo 3`（KD=67）、`ai video generator`（KD=92）等核心词 Google 自有域名（aistudio、deepmind、gemini、labs）垄断前 4 名，第三方内容很难进入 Top5。但替代词和对比词的 KD 极低：`veo 3 alternative`（KD=8）、`veo 3 vs kling`（KD=14）、`sora alternative`（KD=11）——这些词量小但商业意图极强，代表已决策替代的用户。

### 2. 可复制的打法

- **替代词 + 对比词文章**：`veo 3 alternative`（KD=8）、`veo 3 vs kling`（KD=14）、`veo vs sora`（KD=26）——低 KD 对比文是最大攻击面，标准三段式（X vs Y vs Wan on Olares）。
- **定价痛点文**：`veo 3 price`（KD=38）、`veo 3 cost`（KD=44）——定价对比页（Veo 3 $0.40/秒 vs 本地 Wan 零成本）可以配合高 CPC 词抓商业意图流量。
- **"免费路径"教程**：`how to get veo 3 for free`（480 量，KD=41，CPC=$6.45）、`how to use veo 3 outside us`（KD=21）——这类问题词说明用户在寻找低成本/无地区限制的方案，Wan on Olares 是直接答案。
- **本地/开源部署教程**：`local ai video generator`（KD=26）、`comfyui text to video`（KD=16）——技术用户群，ComfyUI + Wan on Olares 组合内容。

### 3. 对 Olares 最关键的 2-3 个词

1. **`veo 3 alternative`**（KD=8，260/月）——零竞争的直接替代意图词，Wan/HunyuanVideo on Olares 完美命中，一篇文章可收割全部替代意图流量。
2. **`local ai video generator`**（KD=26，390/月）——自托管意图核心词，Olares 一键部署叙事直接命中。
3. **`veo 3 price` / `veo 3 cost`**（KD=38/44，合计 2,900/月）——定价痛点对比页，凸显 Veo 3 按秒计费 vs 本地运行零边际成本的对比。

### 4. 最大攻击面（痛点词）

- **闭源按秒计费**：`veo 3 price`、`veo 3 api pricing`、`how much does veo 3 cost`——每条 8 秒视频约 $3.20，重度用户成本极高。
- **地区限制**：`how to use veo 3 outside us`（KD=21）——Google Flow 初始仅限美国，大量全球用户被排在门外。
