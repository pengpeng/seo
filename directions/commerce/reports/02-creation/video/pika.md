## 项目理解（前置）

Pika 是 Pika Labs（斯坦福 AI 实验室背景，Demi Guo / Chenlin Meng 创立于 2023）推出的 **AI 视频生成 App**，主打**消费级短视频特效**——一句话/一张图生成视频，靠 Pikaffects、Pikadditions、Pikaframes 等病毒式特效在社媒（尤其 Discord/TikTok）传播。它是**闭源 SaaS**，按积分（credits）+ 订阅收费，最新世代 Pika 2.2。它的护城河不是 SEO，而是社媒模因与品牌心智。对 Olares 而言，真正对位的不是"复刻 Pika"，而是**"本地私有、无积分、不限时长的视频生成"**——用 Olares Market 已上架的 **ComfyUI** 跑开源的 **Wan（阿里 Wan 2.1/2.2）** 以及 CogVideoX / LTX-Video / HunyuanVideo 等模型。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 消费向 AI 视频生成 App，短视频特效 + text/image-to-video |
| 开源 / 许可证 | 闭源 SaaS（模型与权重不公开） |
| 主仓库 | 无（非开源）；产品入口 pika.art |
| 核心功能 | text-to-video、image-to-video、Pikaffects 特效、Pikadditions/Pikaframes、Pika 2.2 |
| 商业模式 / 定价 | Freemium + 积分（credits）+ 订阅分档；免费额度有限 |
| 差异化 / 价值主张 | 特效好玩、上手快、社媒传播力强（模因视频） |
| 主要竞品（初判）| Runway、Sora、Kling、Luma、Pollo、Hailuo |
| Olares Market | Pika 本身不可自托管；对位方案 = ComfyUI（✅ 已上架）跑 Wan / CogVideoX / LTX-Video |
| 来源 | pika.art（官网）、公开融资报道（$135M / $700M 估值） |
