## 项目理解（前置）

Mega（品牌 MEGA，运营公司 Mega Limited，总部新西兰）是老牌**端到端加密云存储**，2013 年由 Kim Dotcom 创立后已易主，主打"零知识加密——连 Mega 自己都解不开你的文件"。产品线覆盖网盘、文件分享、聊天、密码管理，2024 起加推 **MEGA VPN**。它把**大免费额度 + 隐私**作为获客钩子，是隐私圈里"免费加密网盘"的默认答案之一。核心域名分工特殊：**mega.nz 是 App / 登录 / 分享链接域名**，营销与品类内容主要跑在 **mega.io**（见下方竞品发现）。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 端到端加密（零知识）云存储 / 网盘 / 文件分享 |
| 开源 / 许可证 | 闭源商业产品（部分客户端代码公开可审计，非自托管）|
| 主仓库 | 无自托管发行版；github.com/meganz 仅客户端/SDK |
| 核心功能 | 加密网盘、加密分享链接、MEGA Chat、MEGA VPN、Pass 密码管理 |
| 商业模式 / 定价 | Freemium：免费 20GB 起；付费 Pro I/II/III + VPN，按容量/月订阅 |
| 差异化 / 价值主张 | 端到端零知识加密 + 大免费额度；"连平台都看不到你的数据" |
| 主要竞品（初判）| Proton Drive、pCloud、Sync.com、Tresorit、Icedrive、Dropbox/Google Drive |
| Olares Market | Olares 内置 **Olares Drive**（自托管加密网盘）；Market 另有 Nextcloud / Immich / Seafile |
| 来源 | mega.io / mega.nz / help.mega.io / github.com/meganz |

**Olares 平替定位**：Olares Drive = 端到端加密的**自托管**网盘，数据与密钥都在自己硬件上，**没有容量墙、没有月费**——把 Mega 的"隐私"承诺从"信任一家新西兰公司"升级为"根本不用信任任何人"。

## 核心洞见

1. **品牌护城河：正面刚 mega.nz 的品牌/链接词毫无意义。** mega.nz 的 16,474 月流量本质是**分享链接分发 + 登录词**（`https://mega.nz/file/…` 海量变体全排第 1），不是内容 SEO，无法迁移也不必抢。Mega 真正的品类 SEO 跑在姊妹域 **mega.io（299,968 流量）**——要研究"Mega 打法"应看 mega.io，而 mega.nz 对 Olares 的价值只在于**它证明了"免费+加密网盘"是真实、巨大的需求入口**。

2. **可复制的打法：评测榜单 + 对比页，而非买大词。** Mega 付费投放几乎可忽略（$290/月）；真正的品类流量掌握在 **cyberinsider / cloudwards 这类评测榜单**手里（"best encrypted cloud storage"）。Olares 的可复制动作是：产出高质量 **"best self-hosted / encrypted cloud storage" 榜单**与 **X alternative / X vs Y 对比页矩阵**（google drive / dropbox / nextcloud / mega 各一），进这些榜单并自建对比页。

3. **对 Olares 最关键的 3 个词：`google drive alternative`（1,300, KD23）、`self hosted cloud storage`（390, KD19）、`encrypted cloud storage`（880, KD34）。** 三词构成从"大众替代→自托管精准→加密品类"的漏斗，KD 都在可打范围，且全部指向 Olares Drive 的核心卖点（自托管 + 加密 + 无月费）。次一层 `dropbox alternative`（KD29）、`nextcloud alternative`（KD24）补齐替代词面。
