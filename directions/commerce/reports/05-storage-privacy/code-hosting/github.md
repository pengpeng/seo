## 项目理解（前置）

GitHub 是微软旗下的全球最大代码托管与协作平台，托管超过 4.2 亿个代码仓库，注册用户超过 1 亿名开发者（截止 2024 年）。它不仅是 Git 托管服务，更是集代码审查（Pull Request）、CI/CD（GitHub Actions）、包管理（Packages）、AI 编码助手（Copilot）、项目管理（Projects）、安全扫描（Advanced Security）于一体的完整 DevOps 平台。GitHub 的闭源 SaaS 属性与微软云锁定是其最大痛点：企业数据上传至微软服务器、Actions minutes 超出套餐即计费、GitHub Copilot 按座位订阅、Enterprise Server 自建许可费用高昂。Olares 可在本地一键部署开源平替 Gitea / GitLab CE，实现数据自主可控 + 零云费用。

## 核心洞见

1. **品牌护城河：无法也无需正面刚。** github.com SEMrush Rank 130，自然词 530 万，月流量 1,814 万，流量估值 $3,400 万/月——这是全球排名前 200 的超级权威域名。它 92% 的流量来自用户开源项目 repo URL（程序化内容），**任何品牌词（github / github copilot 等 KD 90-100）都是禁区**。Olares 的战场必须严格限定在"自托管替代"这个 GitHub 不参与竞争的细分空间。
