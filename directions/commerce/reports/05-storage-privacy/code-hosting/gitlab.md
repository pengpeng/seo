## 项目理解（前置）

GitLab 是目前市场上少有的"一站式 DevSecOps 平台"：把代码仓库、CI/CD 流水线、安全扫描（SAST/DAST）、项目管理、AI 助手（Duo）整合在单一应用中，是 GitHub 在自托管市场的最强竞品。GitLab 采用**开源核心（open-core）**模式——CE 版（Community Edition）以 MIT 许可完全开源，可免费自托管；EE 版（Enterprise Edition）加入闭源企业功能，按 Premium/Ultimate 付费订阅。FY2026 ARR 突破 $10 亿，50M+ 注册用户，是全球 >10 万家组织的主力代码平台。2026 年新增 GitLab Duo AI 套件（含代码补全、安全审计助手、MCP Server），战略向 AI-native DevSecOps 演进。
