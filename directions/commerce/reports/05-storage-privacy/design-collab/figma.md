## 项目理解（前置）

Figma 创立于 2012 年，是浏览器原生、实时多人协作的 UI/UX 设计与产品开发平台。它把"设计文件变成 Google Doc 一样可以多人同时编辑"这一理念做到极致，随后从单一设计工具扩展成涵盖原型、白板（FigJam）、开发交接（Dev Mode）、AI 生成（Figma Make）、幻灯片（Figma Slides）、网站发布（Figma Sites）的全栈产品平台。2022 年 Adobe 试图以 $200 亿收购，被英国/欧盟监管机构否决；Figma 于 2025 年 7 月独立 IPO，当日市值逼近 $680 亿（现回落至约 $110 亿）。核心壁垒是**生态锁定**：~4500 万设计师使用 `.fig` 专有格式，95% 的 Fortune 500 是其客户，大量第三方插件/社区资源绑定在 Figma 平台内。AI 积分制（Figma Make 从 2026 年 3 月开始强制计费）是其最大的用户摩擦新风险。**Olares 侧的平替是 Penpot**（MPL-2.0，已上架 Olares Market，Docker/K8s 可自托管，支持 SVG/CSS/HTML 开放标准，~55K GitHub Stars）。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 协作设计平台绝对霸主，从 UI/UX 设计向全栈产品开发平台演进 |
| 开源 / 许可证 | 闭源商业 SaaS；`.fig` 专有格式，无自托管选项 |
| 主仓库 | 无开源主仓库（Figma 全闭源）|
| 核心功能 | 实时多人协作设计、原型、Dev Mode 开发交接、FigJam 白板、Figma Make（AI 生成）、设计系统/变量/组件库 |
| 商业模式 / 定价 | 免费 Starter；Professional $16/mo（Full 席）/ $12/mo（Dev 席）；Organization $55/mo；Enterprise $90/mo（年付）；AI 积分单独计费 |
| 差异化 / 价值主张 | 唯一在浏览器里实现生产级实时协作的设计工具，插件生态 + 社区资产库无可比肩，95% Fortune 500 背书 |
| 主要竞品（初判）| Penpot（开源自托管）、Sketch（macOS Only）、Adobe XD（已停止开发）、Framer（发布网站向）、Canva（非专业向）、Miro（白板向） |
| Olares Market | Penpot 已上架（Figma 本身不可自托管）|
