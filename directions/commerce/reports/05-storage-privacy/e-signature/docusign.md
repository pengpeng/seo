## 项目理解（前置）

DocuSign 是全球电子签名（e-signature）的绝对头部，诞生于 2003 年，2018 年 IPO，如今已成为协议管理的事实标准。超过 18 亿用户在全球使用其服务签署和管理合同。2024 年以来，DocuSign 将叙事从单纯的"电子签名"扩展为"Intelligent Agreement Management（IAM）"平台，引入 AI 对合同进行分析、提取洞察，但核心盈利仍来自 eSignature 订阅。其最大痛点：**按信封（envelope）计费 + 高单价 + 完全闭源**，催生了以 Documenso 为代表的开源替代品。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 全球第一电子签名 + 智能协议管理平台 |
| 开源 / 许可证 | 闭源商业 SaaS |
| 主仓库 | 无（闭源） |
| 核心功能 | eSignature（电子签名）、CLM（合同生命周期管理）、Notary（在线公证）、IAM 平台（AI 协议分析）、API/SDK、Web Forms、Bulk Send |
| 商业模式 / 定价 | 订阅制；Personal $10/月（5封/月）、Standard $25/用/月（100封/年）、Business Pro $40/用/月（100封/年）；企业定制；eSignature 仅 API 另计费 |
| 差异化 / 价值主张 | 市场份额 ~42%（美国）、合规覆盖全球（ESIGN/eIDAS/HIPAA/21 CFR）、深度集成生态（Salesforce/SAP/Microsoft）、IAM AI 洞察新卖点 |
| 主要竞品（初判）| Adobe Sign、PandaDoc、HelloSign（Dropbox Sign）、Documenso（开源）、DocuSeal（开源） |
| Olares Market | Documenso 已上架（DocuSign 的开源平替，[报告](../../../../../market/reports/documenso.md)） |
| 来源 | [docusign.com](https://docusign.com)、[SEC FY26 Annual Report](https://www.sec.gov/Archives/edgar/data/1261333/000126133326000044/docufy26annualreportformupa.pdf)、[定价页](https://ecom.docusign.com/plans-and-pricing/esignature) |

## 核心洞见

1. **品牌护城河**：`docusign` 单词月量 100万，品牌心智极强，正面刚品牌词毫无意义。但护城河是双刃剑：用户 Google `docusign` 而非 `e-signature` 说明品牌词本质是导航词，**竞品机会在品牌词外围**——替代词（`docusign alternatives`）、对比词（`vs adobe sign`）和痛点词（`how to cancel`）是可攻击的缺口。
