## 项目理解（前置）

Slack 是**企业团队协作与即时通讯 SaaS**——用「频道（channel）+ 消息 + 集成」重构了团队沟通，让工作从邮件迁到实时聊天。2020 年被 Salesforce 以 $277 亿收购，如今是 Salesforce「工作操作系统」叙事的入口。它定义了"team chat / workplace communication"这个品类，护城河来自海量企业工作区（含 UCLA / Stanford / MIT / Kubernetes / Chromium 等公开工作区子域名）与深度第三方集成生态。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 企业团队协作/即时通讯平台（频道 + 集成 + 工作流），团队沟通品类定义者 |
| 开源 / 许可证 | ❌ 闭源，纯 SaaS |
| 主仓库 | 无开源本体；`slack.dev` 为其 SDK/API 开发者站 |
| 核心功能 | 频道消息、Huddles 语音/视频、Slack Connect 跨组织、Workflow 自动化、Slack AI、2600+ 集成 |
| 目标用户 / 场景 | 各规模企业团队、开源社区、高校/组织；实时协作沟通 |
| 商业模式 / 定价 | SaaS **按席位/月**：Free / Pro (~$8.75) / Business+ (~$15) / Enterprise Grid（数据存 Salesforce 云）|
| 差异化 / 价值主张 | 集成生态最广 + 品牌心智 + Salesforce 生态绑定；"工作从这里开始" |
| 主要竞品（初判）| Microsoft Teams、Discord、Mattermost、Rocket.Chat；Semrush 印证：Asana / monday / Zapier / Wrike / Atlassian / Pumble / Chanty |
| Olares Market | Slack 本身不可上架（闭源 SaaS）；平替 **Mattermost 已上架 ✅**，Rocket.Chat 可自托管运行于 Olares |
| 来源 | slack.com / slack.com/pricing / Salesforce 公告 / SEMrush |

## 核心洞见

1. **品牌护城河极深，正面硬刚品牌词无胜算。** SEMrush Rank 1,478、月自然流量 177 万、流量估值 $650 万/月，`slack`（823K）+ `slack login`（135K）+ 海量误拼变体全霸榜第 1，还有高校/开源社区公开工作区子域名（stanford / mit / kubernetes.slack.com）在自然扩散。品牌心智不是 Olares 的战场——要打的是它**够不到的地方：开源、自托管、数据主权**。

2. **可复制打法有两条，且都被 Slack 亲自验证过：** ① **内容带流量**——Slack 靠 `/blog/collaboration` 与 `/blog/productivity` 程序化内容排到 `virtual meeting etiquette`（183 万，KD26 第 6）、`team collaboration tools`（183 万，第 9）、`communication skills`（274 万）、`team building activities`（150 万）等超级信息大词，这是纯内容工程；② **买泛词导向落地页**——Google Ads 买 `messenger`（673K）、`messages for web`、`textnow app` 等通用聊天词，全导向 `/chat-instant-messaging` 程序化落地页。Olares 可对每个自托管应用做同构的 "X on Olares / self-hosted X" 落地页矩阵。

3. **对 Olares 最关键的 3 个词：`mattermost vs slack`（140, KD8）、`self hosted slack`（90, KD14）、`open source slack alternative`（110, KD24）。** 这三个词共同指向一个真实且明确的需求——用户在主动搜"能自己托管的 Slack 替代"。KD 全部极低（8/14/24），Slack 本身不可能去排，Mattermost/Element 报告已印证这条赛道几乎无竞争内容。Olares Market 已上架 Mattermost、可自托管 Rocket.Chat，正好把"应用词 + 对比词 + 自托管词"三层流量一网打尽。

4. **最大攻击面 = 按席位收费 + 数据存 Salesforce 云。** `slack pricing`（6,600）、`slack cost`（4,400）、`is slack free`（1,900）、`slack security`（KD46, CPC$17.9）密集出现，说明用户对**席位费与数据归属**高度敏感。Olares 差异化叙事直击此处："自托管 = 无席位费、团队规模无关成本、数据留在自己家、还能接本地 AI"——正呼应 `mattermost pricing`（KD26）、`rocket chat pricing`（KD10）这些价格对比词。
