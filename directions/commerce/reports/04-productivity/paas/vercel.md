---

## 项目理解（前置）

Vercel 由 Guillermo Rauch 于 2015 年创立（原名 ZEIT，2020 年更名），是全球领先的前端云平台：Push 代码即部署，全球边缘网络自动分发，Preview URL 每次 PR 自动生成。它同时是 Next.js 框架的商业赞助方，并正快速向 AI 基础设施转型——v0（AI 代码/UI 生成器）、Vercel AI SDK（3M+ 周下载量）、AI Gateway、Vercel Agent 已成为其第二增长曲线。**Vercel 是闭源 SaaS，不可自托管**；用户付费换零运维，但面临用量计费不可控、单项目团队费用快速堆高的定价痛点。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 前端云 + AI 基础设施平台，"push to deploy" 体验标杆 |
| 开源 / 许可证 | 闭源 SaaS；Next.js 框架本体开源（MIT），Vercel 平台不可自托管 |
| 主仓库 | 无公开主仓库；Next.js: [github.com/vercel/next.js](https://github.com/vercel/next.js)（★ 130k+）|
| 核心功能 | Git 一键部署 / 全球 CDN + Edge Network / Serverless & Fluid 函数 / Preview Deployments / v0 AI 生成器 / AI SDK + AI Gateway |
| 商业模式 / 定价 | Hobby 免费；Pro $20/席位/月（含 $20 用量信用额）；Enterprise 定制；按用量计费（带宽 / CPU 时间 / Edge 请求），隐藏费用多 |
| 差异化 / 价值主张 | Next.js 原生最优化体验；Preview Deployments 工作流；Fluid Compute（较传统 Serverless 低 85% 成本）；v0 是同类最强的 AI UI 生成工具 |
| 主要竞品（初判）| Netlify、Railway、Render、Cloudflare Pages、AWS Amplify；自托管侧：Coolify、Dokploy、Dokku |
| Olares Market | 未上架（闭源 SaaS，不可自托管）；Coolify / Dokploy 为主力平替（待上架）；Coder 已上架（Cloud IDE 方向）|
| 来源 | [vercel.com](https://vercel.com)、[sacra.com/c/vercel/](https://sacra.com/c/vercel/)、[research.contrary.com/company/vercel](https://research.contrary.com/company/vercel)、[vercel.com/pricing](https://vercel.com/pricing) |
