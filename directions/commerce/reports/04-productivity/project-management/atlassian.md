## 项目理解（前置）

Atlassian 是澳大利亚软件公司，旗下三大产品覆盖团队协作的核心场景：Jira（敏捷/DevOps 项目追踪）、Confluence（团队 Wiki 与知识库）、Trello（轻量看板）。公司服务超过 300,000 家客户，包括大量 Fortune 500 企业，营收主体正从历史 Server 许可向 Cloud SaaS 切换。2026 年 3 月 30 日 Atlassian 正式停售 Data Center 新许可，Data Center 将于 2029 年完全 EOL——这意味着数万家原本依赖本地部署的企业正被迫迁移，"自托管"用户迁移窗口已打开。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 企业敏捷/DevOps 项目管理 + 团队 Wiki 套件；Jira 占全球 bug 追踪市场份额约 87% |
| 开源 / 许可证 | 闭源 SaaS；部分组件开源，但核心产品不可自托管（Server EOL，Data Center 2029 停止） |
| 主仓库 | 无 |
| 核心功能 | Jira：Issue 追踪、Sprint 规划、看板/Scrum、DevOps 集成；Confluence：Wiki、知识库、文档协作；Trello：轻量看板；Loom：异步视频 |
| 商业模式 / 定价 | 按用户/月计费，Free（≤10 用户）→ Standard（Jira $7.91/用户/月）→ Premium（$14.54/用户/月）→ Enterprise（定制） |
| 差异化 / 价值主张 | 产品间深度集成（Jira + Confluence + Bitbucket 生态）；海量 Marketplace 插件；企业合规/安全（SSO、数据驻留） |
| 主要竞品（初判）| Asana、Monday.com、Linear、ClickUp、Notion、GitHub Issues、Azure DevOps |
| Olares 平替 | **Plane**（Jira 开源替代，★53,883，已上架 Olares Market ✅）+ **Docmost**（Confluence 开源替代，★20,000+，已上架 Olares Market ✅） |
| 来源 | [atlassian.com](https://atlassian.com)、[atlassian.com/licensing/cloud](https://www.atlassian.com/licensing/cloud)、[plane.so/open-source](https://plane.so/open-source)、[github.com/makeplane/plane](https://github.com/makeplane/plane) |

## 核心洞见

1. **Atlassian 的护城河不是品牌词，是方法论内容帝国。** `jira`（201K, KD 69）、`confluence`（135K, KD 58）的品牌词有大量竞品在拼，正面硬刚成本极高。但 Atlassian 真正的护城河是凭借 `/agile`、`/devops`、`/git/tutorials` 等内容页霸占 `agile methodology`（2.24M）、`git commands`（1.22M）等方法论大词——这类词 Olares 无法也不必争，但可以反向利用（在"Agile with self-hosted tools"内容里把 Plane/Docmost 作为 Atlassian 替代嵌入）。

2. **可复制的打法：对比落地页矩阵 + 买竞品名词。** Atlassian 持续购买竞品词（asana, monday, miro）导向 `/comparison/jira-vs-<竞品>` 对比页。Olares 可以低成本复制这个结构：针对每个 Atlassian 产品建一套"Plane vs Jira"/"Docmost vs Confluence"对比落地页，KD 普遍 < 30，且 Atlassian 自己不会防守"对比自己的开源版本"。

3. **对 Olares 最关键的三个词：`jira alternatives`（1,900, KD 33）、`self hosted wiki`（260, KD 14）、`open source knowledge management`（30, KD 4, CPC $155.60）。** 第一个是量最大的主词候选；第二个极低 KD 却撑一篇独立文；第三个量小但 CPC 异乎寻常（$155 = 比知识库软件词的 $41 高 4 倍），说明真正寻找开源知识管理方案的用户**购买意愿极强**——Docmost 可以精确命中此词。

4. **Atlassian Data Center EOL 是 2026-2029 的时间窗口攻击面。** Atlassian 已停售新 DC 许可（2026-03-30），现有 DC 用户面临 2029 年 EOL。这批用户天然倾向于寻找"可继续自托管但不烧 Atlassian 云钱"的方案，关键词包括：`atlassian data center`（210, KD 42）、`jira data center alternative`（当前量 0 但 CPC $19.10）、`jira self hosted`（140）。Olares + Plane 是这批用户的完美替代叙事。

5. **隐藏低 KD 金矿：vs 系列词全面低竞争。** `jira vs notion`（KD 4）、`confluence vs sharepoint`（KD 12）、`jira vs azure devops`（KD 11）、`jira vs monday`（KD 14）、`jira vs linear`（KD 19）——这些对比词不仅 KD 低，而且带有明显的**商业决策意图**（用户在比较工具时是最接近切换的时候）。Olares/Plane 作为"开源第三选项"可以在这些对比文章中占据位置。

6. **GEO 前瞻布局：`plane vs jira`（KD 0）、`jira data center alternative`（量 0，CPC $19）、`open source atlassian`（量 0）。** 这些词当前搜索量几乎为零，但随着 Atlassian DC EOL 窗口到来和 Plane 品牌增长，预计 2027 会快速放量。建议在 Plane on Olares 的主文章中提前埋入这些段落，抢 AI Overview/Perplexity 引用位。
