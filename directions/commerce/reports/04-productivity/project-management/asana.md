## 项目理解（前置）

Asana 是一款面向企业与团队的 SaaS 工作管理平台，帮助团队跨部门协调任务、项目进度、目标与工作流，是项目管理 SaaS 市场头部玩家之一。它以"结构化任务管理 + 目标对齐 + 组合集报告"为核心差异化，在企业 IT 采购中常与 Monday.com、ClickUp、Jira 竞争。Asana 是纯闭源 SaaS，没有自托管选项；这是 Olares 的关键切入口——Plane（AGPL-3.0，53K GitHub stars）和 Leantime（目标导向型项目管理）已在 Olares Market 上架，提供开源自托管替代方案。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 企业级任务/项目/目标/工作流管理，SaaS-only |
| 开源 / 许可证 | 闭源（Asana 本体无开源版本） |
| 主仓库 | 无公开仓库（闭源商业产品） |
| 核心功能 | 任务 & 子任务、时间线（Gantt）、看板、组合集（Portfolio）、目标（Goals）、工作负载、AI Studio、自动化规则 |
| 商业模式 / 定价 | SaaS，4 档：Personal（免费，≤15 人）、Starter $10.99/用户/月、Advanced $24.99/用户/月、Enterprise（询价）；AI Studio 按 credits 计费 |
| 差异化 / 价值主张 | 最成熟的目标→项目→任务层级打通；Fortune 100 认可；干净 UI 快速上手 |
| 主要竞品（初判）| Monday.com、ClickUp、Notion、Trello、Jira、Wrike |
| Olares Market | 未上架（闭源）；开源替代：Plane ✅、Leantime ✅ 已上架 |
| 来源 | [asana.com/pricing](https://asana.com/pricing)、[workmanagementhub.com/asana-pricing-2026](https://workmanagementhub.com/asana-pricing-2026/)、[plane.so/plane-vs-asana](https://plane.so/plane-vs-asana)、[github.com/makeplane/plane](https://github.com/makeplane/plane)（★53.8K） |

## 核心洞见

1. **品牌护城河极深，正面硬刚无胜算。** SEMrush Rank 3,055，月自然流量 82.5 万（估值 $394 万），且 Asana 同时购买 zoom/trello/n8n/claudeai 等超级大词及竞品词（月付费流量 54 万，花费 $175 万/月），说明 Asana 主动建立多层防御——品牌词、类别词、竞品词、内容词全面覆盖。Olares 不应在高 KD 的品类词（project management software KD65）上硬刚。

2. **可复制的打法：内容营销 + "uses/" 程序化落地页 + 竞品广告买词。** Asana 靠 `/resources/` 内容词（team building activities 月流量 9.75 万、agile methodology 2 万）吸引大量非品牌流量；靠 `/uses/` 和 `/compare/` 页面覆盖品类意图；还购买竞品词（zoom/trello/n8n）导向对比落地页。这套"内容矩阵 + 竞品对比页"模式 Olares 完全可以复制——以 Plane/Leantime 为主体做 `best open-source asana alternative` + `plane vs asana` + `asana pricing vs plane` 的落地页矩阵。

3. **对 Olares 最关键的三个词：`clickup vs asana`（KD17）、`asana alternatives`（KD27）、`plane self hosted`（KD28，无竞价）。** `clickup vs asana` 是意外发现的超低 KD 对比词，可写"ClickUp vs Asana vs 开源自托管（Plane on Olares）"三方评测文，借高流量词带出 Olares 叙事。`plane self hosted` 目前无任何竞争（CPC $0），是 Olares Market 最高效的 Plane 安装教程落地页词。

4. **最大攻击面：定价模式。** `asana pricing`（5,400 量，KD38）、`asana cost`（2,900）、`how much does asana cost`（260，KD28）、`is asana free`（1,300）等定价查询词密集出现，说明用户对 Asana 的**按用户/月计费**高度敏感。Asana Starter $10.99/用户/月 × 20 人 = $220/月 = $2,640/年，而 Plane 社区版自托管完全免费（无用户上限）。这是 Olares 最强的叙事：**一次性硬件成本 + Olares = 数年内 TCO 远低于 SaaS**。

5. **隐藏低 KD 金矿：`basecamp alternative`（KD19）、`asana for nonprofits`（KD24）、`clickup vs asana`（KD17）。** 这三个词都在 KD<25 范围内，且有实际流量（260/210/1300），是被大玩家忽视的切入口。其中 `basecamp alternative` CPC 达 $17.78，说明商业意图极强——Plane 可作为 Basecamp 替代列入清单文。
