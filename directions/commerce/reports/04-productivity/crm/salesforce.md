## 项目理解（前置）

Salesforce 是全球最大的 CRM（客户关系管理）平台，1999 年以"No Software"云交付理念颠覆行业，现已覆盖销售自动化、客户服务、营销自动化、电商、分析与 AI（Einstein/Agentforce）全链路。产品按 Cloud 分模块，以「每用户每月」订阅计费，价格从 $25/用户（Starter）到 $550/用户（Agentforce 1 Sales）不等，AI 功能额外按对话次数或 Flex Credits 叠加收费。**闭源 + 高定价 + 复杂实施 + 厂商锁定**是其最大的用户痛点，也是开源自托管方案的切入点。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 全球第一企业级云 CRM，覆盖销售/服务/营销全链路 |
| 开源 / 许可证 | 闭源 SaaS；无开源版本 |
| 主仓库 | 无开源仓库；GitHub 仅有 SDK/工具 |
| 核心功能 | 销售自动化（Sales Cloud）、客服（Service Cloud）、营销自动化（Marketing Cloud）、AI/Agent（Agentforce）、分析（Tableau）、平台扩展（AppExchange）|
| 商业模式 / 定价 | 年度订阅 $25–$550/用户/月；AI 另收 $0.10/action 或 $2/对话；Agentforce + Data Cloud 捆绑 |
| 差异化 / 价值主张 | 行业最成熟生态（AppExchange 7,000+ 应用）、全球最大 Salesforce 社区/认证体系（Trailhead）、AI/Agent 原生集成 |
| 主要竞品（初判）| HubSpot、Microsoft Dynamics 365、Zoho CRM、Oracle CRM、Pipedrive、Attio、Monday CRM |
| Olares Market | Twenty（已上架，[报告](../../../../../../market/reports/twenty.md)）；SuiteCRM 待上架 |
| 来源 | [salesforce.com/pricing](https://www.salesforce.com/sales/pricing/)、[GitHub salesforce](https://github.com/salesforce)、[atonementlicensing.com 2026 Guide](https://atonementlicensing.com/blog/salesforce-licensing-complete-guide/) |

1. **品牌护城河极深，正面刚品牌词无意义。** salesforce.com SEMrush Rank 660、月自然流量 415 万、流量估值超 $3,191 万/月——品牌词 `salesforce` 单词月量 55 万且 KD=90，排名第 1。这类词 Olares 不应正面竞争。Salesforce 的真正护城河还在于 **130+ 万 Trailhead 认证用户**和 **AppExchange 7,000+ 应用**构成的生态锁定，内容层面无法复制。
