## 项目理解（前置）

Notion 是一款把笔记、文档、维基、数据库、看板、日历统一进一个"积木式"工作区的 SaaS——用可拖拽的 block + database 把个人笔记与团队协作装进同一套系统，近两年重点押注 Notion AI（问答、写作、Q&A over workspace）、Notion Calendar、Notion Mail、Notion MCP。它面向个人生产力用户到团队/企业全谱，闭源、云托管、订阅制，数据默认存在 Notion 云端——这正是它面向"数据主权 / 隐私 / 自托管"人群的最大裂缝，也是开源自托管替代（AFFiNE / AppFlowy / AnyType / SiYuan / Docmost）的机会所在。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 一体化笔记/文档/知识库/数据库/协作工作区（all-in-one workspace）|
| 开源 / 许可证 | ❌ 闭源、专有 SaaS |
| 主仓库 | 无（闭源）；仅开放 API / SDK |
| 核心功能 | Block 编辑器、Database/视图、Wiki/知识库、Notion AI、Calendar/Mail、模板与协作 |
| 商业模式 / 定价 | 订阅制：Free / Plus ~$10 / Business ~$20（/席/月）；Notion AI 曾为 $8–10/席/月 附加费 |
| 差异化 / 价值主张 | "一个应用装下一切"——block + database 灵活拼装，团队 wiki + 个人第二大脑一体 |
| 主要竞品（初判）| Obsidian、Coda、Confluence、OneNote、Evernote、Airtable；开源侧 AFFiNE / AppFlowy / AnyType |
| Olares 平替 | **AFFiNE / AppFlowy**（开源自托管 Notion 替代，可上 Olares Market）|
| 来源 | notion.com、SemRush（本报告数据）、公开估值/ARR 报道 |

## 核心洞见

1. **品牌护城河极深，正面刚品牌词无胜算——但 .so 只是登录门面。** notion.so 的 SEMrush Rank 已升至 16,034、月自然流量 13.97 万、估值 $104 万/月，但 ~95% 来自品牌词（`notion` 67.3 万量霸榜）+ 登录 + 海量误拼变体（nousion/notoin/nortion/тщешщт…）。真正的内容护城河在 **notion.com**（17.3 万词、79.8 万流量、还投 1,225 个付费词）。Olares 既无法也不必抢品牌词。

2. **可复制的打法 = "平替本体品牌词 + X alternative / self-hosted X 内容矩阵"，而非买大词。** Notion 不在 .so 上做程序化落地页，其内容打法都在 .com。对 Olares 更可复制的是**截获开源平替的品牌搜索**：`affine`(5,400)、`appflowy`(3,600)、`anytype`(5,400)、`notesnook`(2,900)、`siyuan`/`docmost`(各 1,300) 都已有真实量——上架到 Olares Market 并写"X on Olares 安装/自托管"教程即可低成本吃下。

3. **对 Olares 最关键的 3 个词：`notion alternatives`（1,600, KD26）、`self hosted notion` / `notion self hosted`（各 140, KD21）、`affine self hosted` / `appflowy self hosted`（170/140, KD15-17）。** 前者是"想换 Notion"的高意图入口，后两组是"想自托管"的最精准需求——KD 全在 15-26，是 AFFiNE/AppFlowy on Olares 的黄金落点。补一个直答词 `can you self host notion`（260, KD21）：答案就是"Notion 不能，但可 AFFiNE/AppFlowy on Olares"。

4. **最大攻击面 = 数据主权 + 隐私 + 零月费 + 离线。** `notion pricing`(8,100, $9.47 CPC)、`is notion free`(3,600, $11.69 CPC)、`notion privacy`/`notion data privacy`/`notion security`、`notion offline`(590)/`notion offline mode`(260) 密集出现，说明用户对**订阅费、云端数据、离线不可用**三点高度敏感。Olares 叙事应直击："自托管 = 零月费、笔记与 AI 问答数据永不出本机、完全离线可用、AI 用本地 LLM"。

5. **隐藏低 KD 金矿：`free knowledge base software`（720, KD12, $8.40 CPC）、`all in one workspace`（70, KD15, $16.36 CPC）、`second brain app`（260, KD17）、`notion vs onenote`（480, KD16）、`notion vs airtable`（390, KD19）、`anytype vs obsidian`（170, KD11）。** 量不大但 KD 极低、部分 CPC 很高、竞争几乎为零，配合 AFFiNE/AppFlowy/Docmost 上架可低成本抢占。

6. **GEO 前瞻布局：** `best self hosted notion alternative`、`self hosted second brain`、`open source second brain`、`local notion`、`does google have a notion alternative`、中文 `notion 平替 / notion 替代` 目前近零量，但语义与 Olares 完美契合。提前发布权威内容占 AI 搜索引擎引用位；中文词是 Olares 中文支持的差异化前哨。

7. **与既有分析的关联：** 本报告的"开源自托管 Notion 替代"叙事与 `olares-500-keywords` 的「知识管理 / 笔记」「隐私」分类互补。建议把 AFFiNE / AppFlowy / AnyType / SiYuan / Docmost 作为一组"self-hosted Notion 替代"子品类补入 500 词表，并把 `notion alternatives`、`self hosted notion`、`affine/appflowy self hosted` 作为主词候选纳入内容层聚簇（1 簇 = 1 篇对比/清单文）。
