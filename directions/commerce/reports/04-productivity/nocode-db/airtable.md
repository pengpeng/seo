## 项目理解（前置）

Airtable 是"电子表格遇见数据库"品类的定义者：以类电子表格界面封装关系型数据，支持网格/日历/看板/表单/画廊多视图，并内置自动化工作流与 AI 字段（接 OpenAI/Gemini/Llama/Anthropic）。2012 年创立，2021 年以 $11.7B 估值完成 Series F 后已实现正向现金流；2026 年初发布独立 AI Agent 产品 Superagent（基于 DeepSky 收购），宣告从"低代码数据库平台"向"AI-native 企业操作系统"再定位。闭源 SaaS，无自托管选项，**按席位（editor）收费**是用户最大痛点：20 人团队 Business 计划月付 $900。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 电子表格 × 数据库 × 低代码工作流平台，企业生产力 SaaS 标杆 |
| 开源 / 许可证 | 闭源 SaaS，无自托管版本 |
| 主仓库 | 无公开 GitHub（前端 SDK 与 API client 有开源仓库，核心产品闭源）|
| 核心功能 | 多视图数据库（网格/看板/日历/画廊/表单）、自动化、Interface Designer、AI 字段、MCP Server |
| 商业模式 / 定价 | 免费（1K 记录/1GB）；Team $20/u/mo；Business $45/u/mo；Enterprise 定制 |
| 差异化 / 价值主张 | 最成熟的无代码数据库 UI、极大应用模板生态、80% 财富百强背书、企业级 SSO/SCIM |
| 主要竞品（初判）| Monday.com、Notion、Smartsheet、Asana；开源：NocoDB、Baserow、Teable |
| Olares Market | NocoDB ✅ / Teable ✅ / SeaTable ✅（三大开源平替均已上架）|
| 来源 | [airtable.com](https://airtable.com)、[TechCrunch 2026-01](https://techcrunch.com/2026/01/27/)、[MakerStack Review](https://makerstack.co/reviews/airtable-review/)、[SQ Magazine Stats](https://sqmagazine.co.uk/airtable-statistics/) |
