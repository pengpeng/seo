## 项目理解（前置）

Superhuman Mail 是一款键盘优先的 AI 邮件客户端，架设于 Gmail 和 Outlook 之上，以"让专业人士以最快速度清空收件箱"为核心主张——快捷键、AI 摘要、自动草稿、已读追踪、日程调度，全套面向高效率商务用户。2021 年估值 $825M，当年 ARR 约 $35M。2025 年 7 月被 Grammarly 以未公开价格收购；同年 10 月，Grammarly 母公司整体改名为 Superhuman，将旗下 Superhuman Mail、Coda（协作工作区）、Grammarly（AI 写作）、Superhuman Go（跨 App AI 助手）打包为 Superhuman Suite。目前 Starter $30/用户/月、Business $40/用户/月，定价远高于市场均值。

| 项目 | 内容 |
|------|------|
| 一句话定位 | AI 邮件客户端，键盘优先 + AI 提速，现属 Superhuman Suite 旗下产品 |
| 开源 / 许可证 | 闭源 SaaS，无开源版本 |
| 主仓库 | 无公开 GitHub |
| 核心功能 | AI Replies / Summaries / Auto Drafts / Auto Labels / Split Inbox / Read Receipts / Keyboard Shortcuts / Snooze & Follow-up / CRM 集成（HubSpot/Salesforce，Business 版） |
| 商业模式 / 定价 | Starter $30/用户/月；Business $40/用户/月（$33 年付）；Enterprise 定制；可 a la carte 购买 Mail；30 天免费试用 |
| 差异化 / 价值主张 | 全平台最快的邮件操作体验 + 本人声音 AI 草稿 + 企业销售场景深度集成；2025 年后叠加 Coda/Grammarly/Go 套件协同 |
| 主要竞品（初判）| Spark Mail（sparkmailapp.com）、Missive（missiveapp.com）、Shortwave、HEY、Mimestream（Mac）、Canary Mail、Gmelius |
| Olares Market | 未上架（Olares Mail 为系统内置 Webmail，可对标；开源邮件服务器 Mailcow 等可补充）|
| 来源 | [superhuman.com](https://superhuman.com)、[Reuters 收购报道](https://www.reuters.com/business/grammarly-acquires-email-startup-superhuman-ai-platform-push-2025-07-01/)、[help.superhuman.com](https://help.superhuman.com)、Gmelius/VentureBurn 2026 评测 |

## 核心洞见

1. **品牌护城河中等，但护城河正在重构。** superhuman.com 主站品牌词集中（前 5 位自然关键词均为品牌），月搜量 27,100（KD66）不算高，护城河比 Lovable 弱。更关键的是：Grammarly 完成改名后，`grammarly`（月量 183 万，KD90）的巨量流量现在由 superhuman.com/products/grammarly 截获（自然流量 10,980），造成"虚高"的感知规模——这部分流量是品牌叙事转型期的临时红利，不代表邮件产品的真实竞争力。正面硬刚品牌词无意义。

2. **可复制的打法：博客 + 广义信息词 + 竞品名买词。** 博客以"email no phone number"类非品牌文章获得大量免费流量，验证了"用泛化邮件话题收割搜索量"的模式。付费端大量购买竞品品牌名（missive、spark、mimestream、canary mail、shortwave）并导入对比/控制页，其中 shortwave 有专属 `/vs-shortwave` 落地页——这套"竞品名 → 对比页"打法可以被 Olares 完整复制，成本极低。

3. **对 Olares 最关键的三个词：`email triage`（110, KD16, CPC $17.51）、`superhuman alternatives`（210, KD8）、`email management software`（4,400, KD30, CPC $21.64）。** `email triage` 是最被低估的机会词：竞争度极低（KD16）、CPC 极高（$17.51）、直接映射"用本地 AI 帮我分类邮件"——Olares Mail + Ollama 正是此场景的完美自托管答案。`superhuman alternatives` KD=8 可直接出排名。`email management software` 量大且 CPC 最高，代表付费意愿强烈的商业受众。

4. **最大攻击面：$30/月的价格门槛 + 数据隐私。** `is superhuman worth it`（KD24, CPC $12.62）、`superhuman pricing`（320, CPC $5.32）、`how much does superhuman email cost`（量小但高 CPC）密集出现，说明用户对价格敏感度极高。更深层的攻击面是隐私：Superhuman 连接 Gmail/Outlook 账户并在云端处理邮件，数据归 Superhuman/Grammarly 所有。Olares Mail 完全本地运行，AI 处理也在本地（Ollama），"每封邮件只有你能看到"是最强差异化叙事。

5. **隐藏低 KD 金矿：**
   - `email triage`（110, KD16）+ `inbox management`（590, KD25）+ `email productivity`（40, KD20）构成一个紧密词簇——合计月量 ~740，KD 均在 25 以下，高 CPC 证明商业价值；
   - `superhuman alternative`/`superhuman email alternative`（KD4/KD1）是 Olares 可 100% 抢占的词，目前几乎无人竞争；
   - `self hosted email server`（KD19）、`self hosted email`（KD22）是专属自托管词，Olares 教程文章即可排名。

6. **GEO 前瞻布局：** `self hosted email client`（KD0）、`email client privacy`（KD0）、`missive alternative`（KD0, CPC $12.37）目前近零量，但语义高度契合 Olares 隐私叙事。建议提前发布权威内容占位，抢 AI Overview / Perplexity 在"私有 / 自托管邮件客户端"场景的引用位——这个赛道在 AI 搜索生态中几乎空白。
