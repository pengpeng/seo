## 项目理解（前置）

Mailchimp 创立于 2001 年，2021 年被 Intuit 以 120 亿美元收购。它从单纯的邮件营销工具演变为"一体化营销平台"，提供邮件自动化、受众分群、落地页、社媒广告、CRM、短信营销、事务邮件（原 Mandrill）等功能，服务全球数百万中小企业与品牌。其主要竞争力在于极低的技术门槛（拖拽编辑器）、庞大的模板库和教程生态系统，以及巨头背书的品牌信任度。2025 年 Mailchimp 大幅缩水免费计划（从 2,000 联系人降至 250），并持续对未订阅用户收费，这成为近年用户流失与替代工具搜索量激增的直接导火索。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 全球最大邮件营销 SaaS，拖拽式编辑 + 自动化 + 受众管理 + CRM 一体化平台 |
| 开源 / 许可证 | 闭源商业 SaaS；事务邮件 Mandrill API 也是闭源 |
| 主仓库 | N/A（闭源） |
| 核心功能 | 邮件自动化、受众分群/A/B 测试、拖拽模板、落地页、短信营销、CRM、报告分析 |
| 商业模式 / 定价 | 按联系人数阶梯计费：Free（250 联系人/月 500 封）、Essentials $13/月起、Standard $20/月起、Premium $350/月起；对未订阅/退订联系人也计费 |
| 差异化 / 价值主张 | 品牌知名度极高、教程/模板生态成熟、与 Shopify/WooCommerce 等深度集成、一站式营销能力 |
| 主要竞品（初判）| Klaviyo、Brevo（原 Sendinblue）、Constant Contact、ActiveCampaign、MailerLite |
| Olares Market | 开源平替 Listmonk 已上架（[市场报告](../../../../../market/reports/listmonk.md)） |
| 来源 | [mailchimp.com/pricing](https://mailchimp.com/pricing/marketing/)、[emailtooltester.com Mailchimp 2026 定价](https://www.emailtooltester.com/en/reviews/mailchimp/pricing/)、[marketerschoice.com 2026](https://marketerschoice.com/mailchimp-pricing-2026/) |
