## 项目理解（前置）

Outlook 是微软邮件产品线的核心品牌，横跨两个场景：**Outlook.com**（面向个人的免费 Web 邮件，前身 Hotmail）和**Microsoft 365 中的 Outlook 客户端**（企业订阅版，含桌面 App + Exchange Online 后端）。两者共用品牌但产品形态差异极大——前者是 Gmail 的直接竞品，后者是企业内网邮箱系统（Exchange）的托管化版本。Outlook 的真正护城河在于与 Microsoft 365 生态的深度绑定：Active Directory 单点登录、Teams 日历共享、SharePoint/OneDrive 集成，使其在中大型企业 IT 采购中几乎不可绕开。自托管替代方向主要有两条路：一是自建邮件服务器（Mailcow / Stalwart / iRedMail）替换 Exchange Online，二是使用 Thunderbird / Nextcloud Mail 等客户端替换桌面 Outlook App。

1. **品牌护城河**：outlook.com 的品牌护城河极深——99% 流量来自品牌导航词（login/hotmail/outlook email），KD 全部 95-100，**无任何正面刚的可能**。`/owa/<domain>` 的机构邮箱入口是独特的流量结构，外部无法复制。Olares 完全不应尝试在品牌词上正面竞争。

2. **可复制的打法**：Outlook 本身不做内容 SEO（也不买广告），其流量完全靠品牌认知驱动。可学习的是其 `/owa/<domain>` 的**程序化白标落地页**逻辑——Olares 若支持多租户邮件域名，可以类似结构为不同组织/社区建立专属入口页。更直接可复制的是竞品（Mailcow/Stalwart）的内容策略：做技术教程 + 对比文 + 自托管指南。

3. **对 Olares 最关键的词**：
   - `self hosted email server`（KD 19，月量 390，同族合计约 1,500+）——最重要的品类攻击词
   - `outlook alternative`（KD 20，月量 320）——最直接的品牌替代词
   - `protonmail alternative`（KD 22，月量 480）——覆盖隐私邮件场景，Olares 自建邮件服务的最强叙事词

4. **最大攻击面**：
   - **价格痛点**：微软 2026-07 再度提价（Business Standard 涨至 $14/用户/月，E3 至 $39），`microsoft 365 alternative`（$8.07 CPC）、`exchange server alternative`（$6.83 CPC）等词的 CPC 反映用户正在主动寻找替代，付费意愿高。
   - **闭源锁定**：Outlook/Exchange 数据格式（PST/OST）制造迁移壁垒，但反过来说，`outlook pst`（KD 26）迁移词可帮助捕获在找迁移方案的用户。
   - **隐私顾虑**：Microsoft 扫描邮件以提供 Copilot 功能，"数据主权"叙事是 Olares 自托管邮件的核心差异化。

5. **隐藏低 KD 金矿**：
