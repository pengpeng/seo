## 项目理解（前置）

Microsoft 365（原 Office 365）是微软的云订阅办公套件，包含 Word、Excel、PowerPoint、Outlook、Teams、OneDrive、SharePoint 等全家桶应用，面向个人、家庭、中小企业到企业客户。全球约 446 million 付费席位，75% 的 Fortune 500 使用 Microsoft 365。2025 年 12 月微软宣布 2026 年 7 月 1 日起最大规模涨价（自 2022 年以来），Business Basic +17%、E3 +8%、F3 +25%，同时捆绑 Defender for Office P1 / Intune / Copilot Chat 等 AI 功能向涨价做包装。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 云订阅制全家桶办公套件，企业市场事实标准 |
| 开源 / 许可证 | 闭源 SaaS；部分 SDK/工具有 MIT 开源 |
| 主仓库 | 无（闭源产品） |
| 核心功能 | Word / Excel / PowerPoint / Outlook / Teams / OneDrive / SharePoint / Copilot AI |
| 商业模式 / 定价 | 订阅制；Business Basic $7/用户/月（涨后）；Business Standard $14；E3 $39；E5 $60（均 2026-07-01 起） |
| 差异化 / 价值主张 | 生态锁定（Active Directory / Azure AD 深度集成）、Excel 护城河、Teams 视频协作、Copilot AI 全家桶 |
| 主要竞品（初判）| Google Workspace、Zoho Workplace、OnlyOffice（开源）、LibreOffice（开源桌面）|
| Olares Market | OnlyOffice ✅ 已上架；Nextcloud ✅ 已上架；Microsoft 365 本体为闭源云服务 |
| 来源 | [microsoft.com/en-us/microsoft-365](https://www.microsoft.com/en-us/microsoft-365)、[Microsoft 涨价公告](https://www.microsoft.com/en-us/licensing/news/2026-m365-packaging-pricing-updates)、[medhacloud M365 Statistics 2026](https://medhacloud.com/blog/microsoft-365-statistics-2026) |
