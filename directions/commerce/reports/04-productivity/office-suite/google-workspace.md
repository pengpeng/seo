## 项目理解（前置）

Google Workspace（前身 G Suite）是 Alphabet 旗下的云端生产力套件，将 Gmail、Google Drive、Docs、Sheets、Slides、Meet、Calendar、Chat 等工具打包为统一订阅服务，核心价值主张是"浏览器原生、零安装、AI 加持（Gemini 原生集成）"。截至 2026 年，Google Workspace 拥有超过 900 万家付费企业客户、300 亿全球 Gmail 用户，在总域名份额上以约 50% 居全球第一（尤其统治初创公司与教育市场）。主要对手是 Microsoft 365（企业份额约 58% 企业席位），两者合计覆盖全球 ~96% 商业生产力市场。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 谷歌云办公套件：Gmail+Drive+Docs+Meet 一体订阅，面向企业/教育/个人 |
| 开源 / 许可证 | 闭源 SaaS，不可自托管 |
| 主仓库 | 无（闭源商业产品） |
| 核心功能 | Gmail（企业邮件）、Drive（云存储）、Docs/Sheets/Slides（在线文档）、Meet（视频会议）、Calendar、Chat、Gemini AI 助手 |
| 商业模式 / 定价 | SaaS 订阅；Business Starter $7/用户/月（年付）、Business Standard $14、Business Plus $22、Enterprise 定制；另有免费 Essentials Starter（15GB） |
| 差异化 / 价值主张 | 浏览器原生协作、深度 Gemini AI 集成、教育免费生态积累（1.7 亿+ 学生）、全球基础设施可靠性 |
| 主要竞品（初判）| Microsoft 365、Zoho Workplace、Zoho Mail、Notion、Slack（单点） |
| Olares Market | 未上架（闭源 SaaS，不可自托管）；Olares 平替：Nextcloud ✅ + OnlyOffice ✅ 已上架 |
| 来源 | [workspace.google.com](https://workspace.google.com)、[gwsave.com/blog/google-workspace-statistics](https://gwsave.com/blog/google-workspace-statistics)、[cloudwards.net](https://www.cloudwards.net/google-workspace-plans-and-pricing/) |
