## 项目理解（前置）

Arize AI 是面向 AI 工程师的**全链路可观测与评估平台**，核心任务是让 AI Agent/LLM 应用"持续自我改进"——从生产追踪（Trace）、自动评估（Eval）到迭代优化（Improve）形成闭环。旗下两大产品：**Phoenix**（Apache-2.0 开源，本地/自托管，10.4k ⭐，LLM 评估与可观测基础工具）和 **Arize AX**（托管 SaaS，企业级，adb 数据库支持，在线 Eval + Agent Swarms 调试）。公司还孵化了 **OpenInference** 标准（基于 OpenTelemetry 的 GenAI 语义约定），是业内开放标准主导者。估值约 $742M，融资 $131M（Coatue / Foundation Capital 领投）。

| 项目 | 内容 |
|------|------|
| 一句话定位 | AI 工程团队的持续学习平台：Trace → Eval → Improve |
| 开源 / 许可证 | Phoenix: Apache-2.0 |
| 主仓库 | [github.com/Arize-ai/phoenix](https://github.com/Arize-ai/phoenix)（★ ~10.4k） |
| 核心功能 | LLM/Agent Trace 追踪、自动 Eval（LLM-as-judge）、RAG 评估、Prompt 实验、在线监控告警 |
| 商业模式 / 定价 | Phoenix 免费自托管；AX Free $0（25k spans/月）；AX Pro $50/月（50k spans）；AX Enterprise 定制（≈$60k/年）|
| 差异化 / 价值主张 | 创始人团队建立了 OpenInference 开放标准；adb 是业内最快最大规模 GenAI trace 存储；Alyx AI 工程 Agent 内嵌调试 |
| 主要竞品（初判）| LangSmith（LangChain 生态）、Langfuse（开源）、Braintrust、MLflow、Weights & Biases |
| Olares Market | Langfuse ✅ 已上架（[报告](../../../market/reports/langfuse.md)）；Arize Phoenix 尚未上架（可自托管） |
