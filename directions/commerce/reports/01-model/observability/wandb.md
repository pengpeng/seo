---

## 项目理解（前置）

Weights & Biases（W&B）是一家 MLOps 平台公司，成立于 2017 年，核心产品覆盖 ML 实验跟踪、超参数优化（Sweeps）、模型注册（Models）与 LLM 可观测性（Weave）。OpenAI、NVIDIA、Meta 等顶级 AI 机构均是其用户。2025 年 5 月 W&B 被 CoreWeave（Nasdaq: CRWV）以约 $1.4B 完成收购，双方合并后形成"GPU 算力 + 模型开发平台"垂直一体化布局。旗舰客户端 SDK（wandb Python 库）以 MIT License 开源，但 **W&B Server（自托管版）需要许可证密钥**，实质上是"开源 SDK + 闭源服务器"模式，数据默认流向 wandb.ai 云端，对数据主权敏感的团队存在顾虑。2026 年 6 月 CoreWeave 在 W&B 平台内推出了 AI 研究 Agent ARIA，进一步加深平台绑定。

| 项目 | 内容 |
|------|------|
| 一句话定位 | MLOps + LLM 可观测一体平台（实验跟踪 → 模型注册 → LLM 评估） |
| 开源 / 许可证 | SDK 开源 MIT（github.com/wandb/wandb，★11k）；Server 闭源，需授权 |

| 商业模式 / 定价 | Free（个人，5 个 Model Seats，5 GB）；Pro $60/月；Enterprise 定制；Server 个人版免费许可 / 企业版需付费 |
| 差异化 / 价值主张 | 丰富的可视化与协作报告、深度框架集成（PyTorch/TF/HF/Lightning）、ML→LLM 全栈覆盖、CoreWeave GPU 生态打通 |
| 主要竞品（初判）| MLflow（开源自托管）、Langfuse（开源 LLM 可观测）、ClearML、Comet ML、Arize AI |
| Olares Market | 未上架（Olares 平替：Langfuse ✅ 已上架；MLflow 可 Docker 部署） |
| 来源 | [官网](https://wandb.ai/site/)、[定价页](https://wandb.ai/site/de/preise/)、[GitHub](https://github.com/wandb/wandb)、[CoreWeave PR](https://investors.coreweave.com/news/news-details/2025/CoreWeave-Completes-Acquisition-of-Weights--Biases/default.aspx) |
