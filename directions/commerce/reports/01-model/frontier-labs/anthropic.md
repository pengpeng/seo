## 项目理解（前置）

Anthropic 是与 OpenAI、Google DeepMind 并列的第一梯队闭源大模型厂商，主打"安全对齐"叙事。它把业务拆成三块：面向开发者的 **Claude API**（按 token 计费）、面向个人/团队的 **claude.ai 消费应用**（Free/Pro/Max/Team/Enterprise 订阅）、以及一系列产品（Claude Code 终端 Agent、Claude Desktop、Claude Cowork 协作空间）。它还发明并推动了 **MCP（Model Context Protocol）**——让 LLM 接外部工具/数据的开放标准，现已成事实标准。对 Olares 而言，Anthropic 是"闭源、订阅制、数据上云、有内容审查"的典型对立面。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 前沿闭源大模型厂商（Claude API + 消费应用 + MCP 标准） |
| 开源 / 许可证 | ❌ 闭源（模型权重不公开）；MCP 协议本身开源（推动生态） |
| 主仓库 | 无模型仓库；MCP 规范 github.com/modelcontextprotocol（开放标准） |
| 核心功能 | Claude 模型 API、claude.ai 聊天、Claude Code（终端 Agent 编码）、Claude Desktop、MCP |
| 商业模式 / 定价 | 订阅（Pro $20/mo、Max、Team、Enterprise）+ API 按量计费；无免费自托管 |
| 差异化 / 价值主张 | 顶尖推理/编码能力（Opus/Sonnet）、"安全对齐"、MCP 生态、Claude Code 编码体验 |
| 主要竞品（初判）| OpenAI/ChatGPT、Google Gemini、DeepSeek、Qwen；开源侧本地方案 |
| Olares 平替 | **LibreChat + Ollama**（自托管聊天前端接本地/云模型，LibreChat 原生支持 MCP、Agents、多模型）——均已在 Olares Market：[librechat](/Users/pengpeng/seo/directions/market/reports/librechat.md)、[ollama](/Users/pengpeng/seo/directions/market/reports/ollama.md)、[open-webui](/Users/pengpeng/seo/directions/market/reports/open-webui.md) |
| 来源 | anthropic.com / docs.anthropic.com / modelcontextprotocol.io |

> **关键背景（本报告口径要点）**：`anthropic.com` 是 Anthropic 的**公司站**（news / research / careers / learn + 模型发布公告），**Claude 产品自身的流量不在这个域名上**——它在 `claude.ai`（US 自然流量 **4,843,349/mo**）和 `claude.com`（**707,399/mo**）。因此本报告分析的是"公司/品牌 + 模型公告 + MCP"这一层的 SEO，产品词的真实体量还要更大（见竞品发现）。
