## 项目理解（前置）

Reka AI（2022 年成立，前 Google DeepMind / Meta 研究员团队）是一家多模态 AI 研究公司，核心产品为闭源 API 服务（Core / Flash / Edge 三档）及 Nexus 企业 AI Worker 平台。2025 年 7 月完成 $110M B 轮（Nvidia + Snowflake 领投），估值突破 $1B。当前战略正从纯 LLM API 供应商转向"物理 AI 时代基础智能"——面向机器人与可穿戴设备的实时 Omni 模型。值得注意的是：Reka Flash 3 / 3.1（21B，Apache 2.0）已作为开放权重在 HuggingFace 发布，可通过 Ollama 在本地运行，这与常识"Reka 完全闭源"不同；Core 和 Edge 仍为专有 API。

## 核心洞见

1. **品牌护城河**：reka.ai 的 SEO 护城河几乎为零——357 个关键词几乎全是品牌变体/拼写错误，零非品牌有机排名。对 Olares 来说，**无法通过攻击其品牌词获量**（KD 47 且用户明确找 Reka），但正因为 Reka 自己也没有在非品牌词上建立内容防线，该赛道的空白极大。

2. **可复制的打法**：Reka 的付费广告策略值得学习——**直接用竞品域名做词**（synthesia.io、twelvelabs、hive ai、aws rekognition），把竞品用户引到自己的落地页。这证明了"X alternative"打法在 AI 领域的可行性；Olares 可用同样逻辑，在 Reka / Claude / OpenAI alternative 词上布局内容。

3. **对 Olares 最关键的 3 个词**：
   - **`self hosted llm`**（320/mo, KD 22）：KD 极低，量充足，Olares + Ollama 最直接的落地词
   - **`private ai`**（1,900/mo, KD 29）：量大+中低 KD，Reka 闭源 API vs Olares 本地 AI 的核心对比叙事
   - **`claude alternative`**（260/mo, KD 18）：KD 18 全场最低之一，可切入"不想付 Claude 订阅费，在 Olares 上运行开放 LLM"叙事
