## 项目理解（前置）

OpenAI 是闭源前沿模型的领跑者：面向消费者的 **ChatGPT**（订阅 Plus/Team/Enterprise）与面向开发者的 **GPT API / Codex**，是"AI 助手"这个品类的代名词。ARR ~$25B、估值 ~$852B，品牌词体量在整个互联网都属顶级（"chatgpt" 单词月搜索量 6,800 万）。它的护城河是**品牌心智 + 模型能力**，不是 SEO 技巧——正面抢它的品牌词无意义；能切入的只有"替代 / 本地 / 自托管 / 隐私 / API 成本"这类**意图缝隙词**。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 闭源前沿大模型厂商，ChatGPT 是 AI 助手品类的事实标准 |
| 开源 / 许可证 | ❌ 闭源（模型权重、训练数据、推理栈均不公开）；仅早期 gpt-2、Whisper、部分工具开源 |
| 主仓库 | 无核心产品开源仓库；github.com/openai 多为 SDK / cookbook |
| 核心功能 | ChatGPT 对话、GPT-5 系列模型、GPT/Responses API、Codex（编码）、Sora（视频，已调整）、DALL·E（图像）、Realtime/语音 |
| 商业模式 / 定价 | 订阅（Plus $20/mo、Team、Enterprise）+ API 按 token 计费 + 企业合同 |
| 差异化 / 价值主张 | 最强通用模型、最大生态、最低使用门槛；用户"要 AI 就想到 ChatGPT" |
| 主要竞品（初判）| Anthropic Claude、Google Gemini、DeepSeek；本地侧 Ollama / Open WebUI / LM Studio |
| Olares Market | ✅ 平替已上架：**Open WebUI**（聊天前端）+ **Ollama**（本地推理引擎） |
| 来源 | openai.com、platform.openai.com、SemRush |
