## 项目理解（前置）

Poe 由 Quora 于 2022 年底推出，2023 年 2 月正式对外开放，是一个多模型 AI 聊天聚合平台：用户购买单一订阅，即可通过统一界面切换 GPT-5、Claude Opus/Sonnet、Gemini 2.5 Pro、Llama 4、Mistral、Grok 4 等 100+ 个文本、图像、视频、音频模型，以及超过 100 万个社区构建的 Bot。Poe 不训练自有模型，定位是"AI 超市货架"——对上游模型厂商支付 API 计算费用，对用户收订阅费，并将部分收入分给 Bot 创作者。2024 年 1 月 a16z 领投 $75M Series E，估值约 $500M；2026 年分析师估计 ARR ~$65M，月独立访问 ~1100 万。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 多模型 AI 聊天聚合平台，单订阅访问 100+ 模型 |
| 开源 / 许可证 | 闭源商业 SaaS |
| 主仓库 | 无公开主仓（Poe API SDK 有开源辅助库，规模小） |
| 核心功能 | 多模型切换聊天、社区 Bot 创建与分享、图/视频/音频生成模型接入、Poe API（2025-07 上线）、Bot 创作者变现 |
| 商业模式 / 定价 | 订阅 + Points 制：Free（300 pts/日）/ Entry $4.99/月（10K pts/日）/ Standard $19.99/月（100万 pts/月）/ Pro $49.99/月 / Ultra $249.99/月；加购点数 $30/百万 |
| 差异化 / 价值主张 | 一个账号替代多个 AI 订阅（省 ChatGPT Plus+Claude Pro 费）；Bot 创作者生态；声称比原厂 API 便宜 10-30% |
| 主要竞品（初判）| OpenRouter、ChatGPT（直接）、Claude.ai、You.com、LibreChat/LobeChat（开源平替） |
| Olares Market | LibreChat ✅ 已上架；LobeChat ✅ 已上架（均可接 Ollama 本地模型） |

## 核心洞见

1. **品牌护城河**：Poe 的品牌词（"poe ai"月量 22K、KD=76）几乎不可正面竞争——品牌导航流量占自然流量 90%+ 均是硬壁垒。但 Poe 的品类词（"chatgpt alternative"、"openrouter alternative"）KD 仅 47/8，是可打空间；Poe 在"poe ai alternatives"（KD=10）上没有防守优势，竞争者可直接切入。

2. **可复制的打法（程序化 Bot 落地页）**：Poe 靠 `poe.com/Grok-4`、`poe.com/Sora-2-Pro`、`poe.com/Homeworkify` 等 Bot 专属 URL 捕获模型名词流量（homeworkify 月量 22K、KD=25），逻辑是：每个热点模型/应用发布时自动多一个 SEO 落地页。**对 Olares 的启示**：LibreChat/LobeChat 的模型支持页（如 `/models/qwen3`、`/models/llama4`）可以复制同一打法。

3. **对 Olares 最关键的 3 个词**：
