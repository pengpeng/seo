## 项目理解（前置）

Consensus 是一家纽约初创公司（2022 年成立），提供 AI 驱动的学术文献搜索与综合平台。用户可用自然语言提问，系统从 250M+ 篇同行评审论文中检索、阅读并合成证据，并用"共识仪表盘（Consensus Meter）"可视化展示学界支持/反对某一观点的比例。2026 年 5 月完成 $30M 新一轮融资（总融资约 $45.75M），发布了基于 GPT-5 + OpenAI Responses API 的多智能体系统 Scholar Agent，已服务 1000 万+ 研究者。与 Elicit（结构化数据提取/系统综述）、Semantic Scholar（免费索引）等构成学术 AI 搜索赛道主要玩家。

| 项目 | 内容 |
|------|------|
| 一句话定位 | AI 学术文献搜索引擎，250M+ 论文，一句话提问→同行评审证据合成 |
| 开源 / 许可证 | 闭源商业 SaaS |
| 主仓库 | 无开源仓库 |
| 核心功能 | 自然语言学术搜索、Consensus Meter（证据共识仪表盘）、Deep Search（1000 篇筛查）、Scholar Agent（GPT-5 多 Agent 综述）、Citation Graph、Medical Mode |

| 差异化 / 价值主张 | 仅索引同行评审文献（无噪音网页）、低幻觉引用追溯、Consensus Meter 直观呈现学界分歧度、Scholar Agent 将"数周研究"压缩到分钟级 |
| 主要竞品（初判）| Elicit、Semantic Scholar、SciSpace、Research Rabbit、Scite.ai、Google Scholar |
| Olares Market | 未上架（Consensus 为闭源 SaaS；Olares 开源平替为 Open Notebook，GitHub：lfnovo/open-notebook） |

## 核心洞见

1. **品牌护城河极深，但方式特殊——靠"问题词矩阵"而非品牌词。** `consensus`（KD 98）、`consensus ai`（KD 70）完全由自身主导，无需争夺。真正撑起 20 万词、17.7 万月流量的是数十万条 `/questions/` 程序化 Q&A 页面（医疗缩写、星球知识、药物名等），这套内容护城河极难复制——Olares 无需也无力正面竞争 Consensus 的 250M 学术论文数据库与品牌词。

2. **可复制的打法是"程序化 Q&A + 竞品品牌词拦截"组合。** Consensus 一方面靠 `/questions/` 路径的长尾问答获取了绝大多数自然流量，另一方面用 Google Ads 精准购买竞品（Anara / Elicit / ResearchRabbit / SciSpace / Scite）品牌词，以低 CPC（$0.04-$3.94）截获竞品流量。对 Olares 的启示：可以在对比文章中覆盖"Elicit vs Consensus vs Open Notebook"三方格局，用内容形式替代 Ads 拦截。

3. **对 Olares 最关键的 3 个词：`notebooklm alternative`（KD=7！）、`best ai for research`（KD39）、`literature review ai`（KD35）。** 这三个词覆盖了"自托管研究工具"的核心意图，且竞争相对温和——尤其 `notebooklm alternative`（KD=7，月量 320）是罕见的高意图低竞争词，Open Notebook on Olares 在语义上完美匹配，应优先布局。

4. **最大攻击面：Consensus 的订阅门槛 + 公共数据库局限。** `consensus pricing`（CPC $15.52，KD 16）显示用户对价格极其敏感；Deep 计划 $45/月对高频研究者压力不小。更关键的是：Consensus 只能搜公开学术数据库，**无法处理私有/机密文档**（企业内部研究报告、临床试验数据、专有数据库）——这是 Open Notebook on Olares 的核心叙事切入点："私密研究场景只能自托管"。

5. **隐藏低 KD 金矿：`notebooklm alternative`（KD=7）+ `google scholar alternative`（KD=27）+ `literature review tool`（KD=29）。** 这三个词都在 KD30 以下，且带有明确"寻找替代工具"的商业意图，是 Olares SEO 最现实的机会窗口。特别是 `notebooklm alternative` 的 KD=7 在所有学术 AI 词中极为罕见，建议立即布局。

6. **GEO 前瞻布局：`consensus alternative`（KD=0）、`open source literature review`（量≈0）、`self hosted research tool`（量≈0）。** 这些词当前量几乎为零但语义与 Olares 高度契合，随着 AI 搜索引擎（AI Overview / Perplexity / ChatGPT Search）发展，学术研究者在 AI 回答中寻找"开源/自托管学术工具"时，这些词将是引用位争夺重点，建议提前发布权威内容。

7. **与既有分析的关联：** Consensus 所在的学术 AI 搜索赛道在 `olares-500-keywords` 中尚属空白（目前主要覆盖 AI 编码/本地 LLM/通用 Agent 等方向），本报告揭示了一个新的切入点——"**学术研究 + 自托管 + 私有文档**"内容簇，建议将 `notebooklm alternative`、`literature review ai`、`google scholar alternative` 作为独立子品类补入关键词优先级队列。Open Notebook 的应用属性（NotebookLM 竞品）与 Consensus 所在的学术 AI 赛道有部分重叠，但叙事角度应明确区分：**Consensus = 公共论文搜索 SaaS；Open Notebook on Olares = 私有文档研究 + 自托管 + 多模型选择**，不做功能对等声明。
