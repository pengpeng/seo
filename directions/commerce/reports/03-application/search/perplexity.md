## 项目理解（前置）

Perplexity（公司 Perplexity AI，2022 成立）是一个 AI 答案引擎：用户提问，它实时检索网页、用 LLM 综合出带引用的答案，取代传统"十个蓝链接"。核心卖点是"回答 + 来源"，2024–2025 靠此建立起 answer engine 心智，并快速把边界扩张到 Comet（AI 浏览器）、Perplexity Computer（Agent）、Finance（实时股价页）、Labs、Sonar API。它是闭源 SaaS，靠 Pro 订阅 + 企业 + API 变现。

| 项目 | 内容 |
|------|------|
| 一句话定位 | AI 答案引擎——带引用的对话式搜索，向 AI 浏览器/Agent 扩张 |
| 开源 / 许可证 | **闭源**（专有 SaaS），Sonar 模型/后端不开放 |
| 主仓库 | 无公开主仓库（仅 SDK/示例） |
| 核心功能 | 引用式答案 / Comet 浏览器 / Perplexity Computer(Agent) / Finance 实时股价 / Sonar API / Spaces / Labs |
| 商业模式 / 定价 | 免费档 + Pro（约 $20/mo）+ Max + Enterprise + Sonar API（按量计费） |
| 差异化 / 价值主张 | 实时联网 + 引用 + 多模型可选；速度与"直接给答案"体验 |
| 主要竞品（初判）| ChatGPT Search、Google AI Overview/Gemini、You.com、Phind、Grok；开源侧 Perplexica/Vane、SearXNG、Khoj、Morphic |
| Olares Market | ✅ 平替已上架：**SearXNG**（无追踪搜索）+ **Vane**（原 Perplexica，AI 答案引擎） |
| 来源 | perplexity.ai 官网 / 公开报道（ARR/估值）/ Olares [market/apps.md](/Users/pengpeng/seo/directions/market/apps.md) |

**Olares 平替叙事：** SearXNG（本地无追踪元搜索，做检索底座）+ Vane / Perplexica（开源答案引擎，做 LLM 综合层）在 Olares 上一键组合 = **私有 AI 搜索**：查询与推理都在自己设备上、无订阅费、搜索历史永不外泄——正对 Perplexity 的闭源 + 订阅 + 数据收集三处软肋。
