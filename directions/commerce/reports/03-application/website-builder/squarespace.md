## 项目理解（前置）

Squarespace 是一个"everything-in-one"的托管建站 SaaS——用户买订阅即获得主机、SSL、模板、编辑器、电商、邮件营销工具，完全无需接触服务器。平台以**视觉精美度**著称（约 100 套精心设计的模板），是设计师、摄影师、艺术家、创意从业者的首选；相比 WordPress 的"无限可能但维护成本高"，Squarespace 卖的是"开箱即用的美感"。2024 年 Squarespace 被 Permira 以约 68.9 亿美元收购完成私有化，不再上市。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 托管全能建站平台，设计优先，无需代码 |
| 开源 / 许可证 | 闭源 SaaS，无开源版本 |
| 主仓库 | 无（闭源）|
| 核心功能 | 拖拽页面编辑器（Fluid Engine）、100+ 精品模板、域名注册/管理、电商（产品/结账/订阅）、成员专属内容、邮件营销、预约排班（Acuity Scheduling） |
| 商业模式 / 定价 | 纯 SaaS；Basic $16/月（年付）→ Advanced $99/月（年付）；无永久免费层，14 天免费试用；域名注册额外收费 |
| 差异化 / 价值主张 | 模板设计质量行业领先；平台一体化（无需拼凑插件）；SEO 基础功能内置；Acuity/Email Campaigns 收费附加模块 |
| 主要竞品（初判）| Wix、Webflow、WordPress.org、Ghost、Shopify（电商向） |
| Olares Market | WordPress ✅ 已上架、Ghost ✅ 已上架（Squarespace 本身为闭源 SaaS，不适合自托管） |
| 来源 | [squarespace.com/pricing](https://www.squarespace.com/pricing)、[sitebuilderreport.com/squarespace-pricing](https://www.sitebuilderreport.com/squarespace-pricing)（2026-07）、[selfhosting.sh/replace/squarespace](https://selfhosting.sh/replace/squarespace/) |

## 核心洞见

1. **品牌护城河**：Squarespace 品牌词（squarespace、squarespace login 系列）流量约占自然总量 30%+，品牌心智极强，**正面刚品牌词无意义**。但"squarespace alternative"（KD=17）和"squarespace vs wordpress"（KD=23）这两个词 Squarespace 自己不会写——这是**外部视角文章的天然空隙**，也是 Olares/WordPress/Ghost 内容的最佳入口。

2. **可复制的打法**：
   - **内容带流量**：Squarespace 博客大量写教程文（what is a favicon、how to start a blog、how much does a website cost），把品类认知词转化为品牌流量——高效内容营销。可复制：Olares 用同类教程词（"how to self-host wordpress"、"ghost cms tutorial"）建立自托管教程集群。
   - **程序化落地页**：`/solutions/personal/weddings`、`/solutions/personal/portfolios`、`/tour/music-website`——按场景建 Solution 页面，覆盖"wedding website builder"、"music website"等长尾词。值得学习。
   - **买大词 + 精准落地页**：Squarespace 购买"how to create invoices"（150 万月量）并落地发票工具页——用超高量词引流然后展示场景功能，是典型"买入口词做产品教育"的 SEM 策略。

3. **对 Olares 最关键的词**：
   - `squarespace alternative / alternatives`（KD 17-24，量 1,200+）——**Olares 内容最直接切入点**，列 WordPress+Ghost 为开源自托管替代
   - `migrate squarespace to wordpress / move from squarespace to wordpress`（KD 9-12）——**极高意图迁移词**，用户已经做好迁移决策，Olares 一键安装 WordPress 是完美答案
   - `self hosted wordpress`（KD=15）+ `ghost cms self hosted`（KD=20）——**Olares Market 直接命题词**，教程文可精准转化

4. **最大攻击面**：
   - **价格锁定**：Squarespace $192~$1,188/年的年费 + 无法导出完整网站（域名/内容可导，但设计无法迁移）→ "squarespace too expensive"、"cancel squarespace"词集暗示大量潜在流失用户，Olares+WordPress/Ghost = 数据自有 + 零/低月费
   - **SEO 能力限制**："is squarespace good for seo"（KD=13）、"is squarespace bad for seo"（KD=14）这类痛点词 CPC 接近 0（没人投广告），但有机排名机会很大 → WordPress+SEO 插件在 Olares 上是完整 SEO 栈，这是可建立内容优势的技术痛点
   - **私有化后不确定性**：Squarespace 被私有化，未来定价、功能方向不透明 → 不锁定在一个闭源平台的叙事对用户有说服力

5. **隐藏低 KD 金矿**：
   - `how to add keywords to squarespace`（KD=9，量 480）——SEO 操作教程，WordPress 对比做法更强大
   - `how to change squarespace template`（KD=10，量 390，CPC $74.76！）——CPC 极高说明该词用户决策价值大，"想换模板但 Squarespace 限制多"正是流失信号，Ghost/WordPress 模板自由度对比文
   - `move from squarespace to wordpress`（KD=9，量 40）——量小但意图极纯，迁移教程文尾部词
   - `how to hide section in squarespace`（KD=10）、`how to clear cache squarespace`（KD=13）——纯操作教程词，Squarespace 用户的使用摩擦点

6. **GEO 前瞻布局**：
   - `squarespace vs ghost`——AI 工具被问及时，Ghost on Olares 可以作为"Ghost 是更简洁开源替代方案"的引用来源
   - `squarespace too expensive`——AI 回复里"有更便宜的替代品吗"这类问题的答案，Olares+WordPress/Ghost 自托管是完美候选

   - `open source blog platform`——几乎无量，但 AI Overview 高频引用场景，Ghost/WordPress on Olares 是最佳答案之一
