## 项目理解（前置）

Wix 是一个"一站式"托管建站 SaaS——用户购买订阅即获得主机、SSL、域名、AI 编辑器、电商工具、邮件营销，完全无需接触服务器。平台以**低门槛拖拽体验**著称（2,000+ 模板），是全球使用最广泛的 Website Builder，2026 年累计为超过 850 万个在线网站提供服务。Wix 正积极向 AI 方向转型，主力产品 Wix ADI（AI Design Intelligence）与 Wix AI Website Builder 已可对话生成网站，并收购 Base44 切入 AI 应用生成赛道。与 WordPress 的核心差异在于：Wix 完全托管，用户无法迁移/导出网站结构；WordPress 自托管可完全控制数据与代码。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 全托管 AI 建站 SaaS，拖拽零代码，月订阅制 |
| 开源 / 许可证 | 闭源 SaaS，无开源版本 |
| 主仓库 | 无（闭源）|
| 核心功能 | AI 拖拽编辑器（Wix Editor + Wix Studio）、2,000+ 模板、域名注册/管理、Wix 电商（支付/库存/税务）、Wix ADI（AI 建站）、Base44（AI 应用生成）、Wix Logo Maker、App Market（300+ 集成）、Wix Payments（线上+线下 POS） |
| 商业模式 / 定价 | 纯 SaaS；Free（带广告和 wixsite.com 子域）→ Light $17/月 → Core $29/月 → Business $36/月 → Business Elite $159/月（年付价）；域名额外收费 |
| 差异化 / 价值主张 | 最低学习门槛（1 小时建站）；AI 对话式建站引领行业；垂直解决方案丰富（餐厅、活动、健身、音乐）；Wix Studio 面向专业设计师/代理商 |
| 主要竞品（初判）| Squarespace、Webflow、WordPress.org、Ghost、Shopify（电商向）、GoDaddy Websites |
| Olares Market | WordPress ✅ 已上架、Ghost ✅ 已上架（Wix 本身为闭源 SaaS，不适合自托管） |

## 核心洞见

1. **品牌护城河**：Wix 品牌词"wix"月量 67.3 万、KD 99，完全不可正面刚。但**比较词护城河极弱**："wix vs wordpress"（1,600/月）、"wix vs squarespace"（4,400/月）、"wix alternatives"（1,000/月）竞争密度低（0.16-0.32），是可进攻的低 KD 机会带。

2. **可复制的打法**：
   - **程序化 VS 落地页**：Wix 有 `/lp-en/wix-vs-ss`（squarespace 对比）等专属 VS 落地页，并用付费广告购买 `squarespace`、`square space`、`weebly` 等竞品词直接导流，同时用自然搜索霸占"wix vs X"词群——这套"买对手词 + 程序化 VS 页"打法值得复制到 Olares 内容：可建"Wix vs WordPress（自托管）"、"Wix vs Ghost"、"Wix 替代方案"等对比页。
   - **博客内容飞轮**：Wix 对"kpi examples"（184 万/月）、"how to start a blog"（150 万/月）、"brand identity examples"（150 万/月）等与建站无直接关系的超大量信息词进行博客布局，形成巨量内容流量。Olares 可学习此打法，用"how to host your own website"、"wordpress self hosted tutorial"等信息词为自托管内容引流。

3. **对 Olares 最关键的词**：
   - `wix alternatives` / `wix alternative`（1,260/月合计）：最直接的替代意图词，WordPress + Ghost on Olares 是答案
