---

## 项目理解（前置）

Bolt.new 是 StackBlitz 于 2024 年推出的 AI 驱动全栈开发平台，核心技术是 **WebContainers**——在浏览器标签页内运行完整的 Node.js 环境，无需本地配置即可提示、运行、编辑并部署全栈 Web 应用。它以"开发者友好的 vibe coding"定位区分于 Lovable（锁定 React+Supabase）和 v0（只生成 UI 组件），支持 Next.js、Vue、Svelte 等多框架、多模型（Claude Opus 4.6、GPT-5.5、Gemini），并于 2026 年推出 Bolt V2。融资 $135M，估值约 $700M，在 vibe coding 品类与 Lovable 并驾齐驱。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 浏览器内 AI 全栈开发环境，"从提示到部署全在一个浏览器 Tab" |
| 开源 / 许可证 | 闭源 SaaS（对应开源社区 fork **bolt.diy**，MIT License） |
| 主仓库 | https://github.com/stackblitz/bolt.new（★ ~16K） |
| 核心功能 | WebContainers 浏览器运行时、多模型选择、一键 Netlify 部署、Figma 导入、团队协作工作区 |
| 商业模式 / 定价 | Token 计量：Free $0（1M token/月，300K/天上限）；Pro $25/月（10M+ token，支持 rollover）；Teams $30/人/月；Enterprise 定制 |
| 差异化 / 价值主张 | WebContainers 秒启零安装；框架不锁定；代码透明可导出；token rollover（竞品普遍不支持）|
| 主要竞品（初判）| Lovable（全栈 SaaS）、v0（Vercel UI 组件）、Replit（在线 IDE + Agent）、Cursor / Windsurf（本地 AI IDE）|
| Olares Market | 未上架（对应开源版 **bolt.diy** 亦未上架）|
| 来源 | [官网](https://bolt.new)、[GitHub](https://github.com/stackblitz/bolt.new)、[定价页](https://bolt.new/pricing)、[vibetools review 2026](https://vibetoolstack.com/tools/bolt)、[aiagentsquare review](https://aiagentsquare.com/agents/bolt) |

1. **品牌护城河比 Lovable 弱，但"bolt"词性歧义是双刃剑。** Bolt 的 SEMrush Rank 57,885 vs Lovable 9,647，SEO 成熟度差距悬殊。品牌词 `bolt`（135K 月量）极强但 KD=97，且与"车用螺栓/闪电"严重混淆——Bolt 自己都得花大钱买付费来守住品牌词。Olares 不需要抢品牌词，专攻 KD<30 的替代词和开源词。
