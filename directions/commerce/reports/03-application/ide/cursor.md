## 项目理解（前置）

Cursor 是 Anysphere 公司做的 AI 优先代码编辑器——基于 VS Code fork，把「AI Agent + Tab 补全 + Chat + Composer」深度内嵌进 IDE，让开发者用自然语言写码、重构、调试。2024–2026 年成为 AI coding 赛道增长最快的商业产品，估值/ARR 均达数十亿美元级别。2025 年起补上 **Cursor CLI**（`cursor.com/cli`，把同一 Agent 带进终端/CI），与本行合并处理。它是**闭源 SaaS**，核心卖点是模型编排体验、代码库理解与「云端强模型（Claude/GPT/Gemini）」的即用性，代价是**订阅计费 + 代码/上下文需上云**。

| 项目 | 内容 |
|------|------|
| 一句话定位 | AI 优先的代码编辑器/IDE（VS Code fork）+ 终端 Agent（Cursor CLI）|
| 开源 / 许可证 | **闭源**（编辑器 fork 自 VS Code；Cursor 本体不开源）|
| 主仓库 | 无公开主仓库（仅 `getcursor/cursor` issue 追踪库）|
| 核心功能 | Agent/Composer 多文件改写、Tab 智能补全、代码库 Chat、MCP 生态、Cursor CLI/终端 Agent、Bugbot 代码审查 |
| 商业模式 / 定价 | 订阅制：Free / Pro（~$20/mo）/ Pro+ / Ultra / Teams / Enterprise；按 usage/credits 计量强模型调用 |
| 差异化 / 价值主张 | 最好的「云端强模型 + IDE 一体化」体验、代码库理解、Agent 自动化、模型随选 |
| 主要竞品（初判）| GitHub Copilot、Windsurf、Claude Code、Zed、Trae、Cline、Continue.dev |
| Olares Market | 未上架 Cursor（闭源）；平替 **Continue.dev + Ollama** 走自托管路径 |
| 来源 | cursor.com、cursor.com/cli、cursor.com/pricing、docs.cursor.com（2026-07 核对）|

**Olares 平替叙事**：Cursor 是闭源订阅 + 代码上云；Olares 侧用 **Continue.dev（开源 VS Code/JetBrains 插件）接本地 Ollama + Qwen3-Coder**，做到"AI 编程助手全程本地、无订阅、代码不出机"。
