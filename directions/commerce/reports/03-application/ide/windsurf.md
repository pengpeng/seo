---

## 项目理解（前置）

Windsurf 是一款以 AI Agent 为核心的代码编辑器 / IDE（VS Code 分叉），主打 "Cascade" agent 与自动补全 "Tab"，面向专业开发者与团队。品牌前身为 Codeium（2025 更名并全站迁至 windsurf.com，codeium.com 已基本死域），2025 年被 Cognition（Devin 的母公司）收购并入。它与 Cursor 是当前 AI-native IDE 里最直接的一对竞品，两者都是**云端闭源订阅服务**——代码上下文与推理都依赖厂商云。

| 项目 | 内容 |
|------|------|
| 一句话定位 | AI-native 代码编辑器 / Agentic IDE，VS Code 分叉，主打 Cascade agent + Tab 补全 |
| 开源 / 许可证 | **闭源商业**（编辑器本体免费下载，能力后端闭源、按订阅计费）|
| 主仓库 | 无公开主仓；文档 docs.windsurf.com |
| 核心功能 | ① Cascade agentic 流程 ② Tab 自动补全 ③ 多模型接入（Windsurf models）④ MCP 支持 ⑤ JetBrains/VS Code 插件 |
| 商业模式 / 定价 | 免费档 + Pro 订阅（含 credits/额度）+ 企业版；数据/推理走厂商云 |
| 差异化 / 价值主张 | agent-first 工作流、深度 IDE 集成、由 Cognition/Devin 背书 |
| 主要竞品（初判）| Cursor（最直接）、GitHub Copilot、Continue.dev、Codeium(旧身) |
| Olares Market | Windsurf 本体不上架；Olares 平替 = **Continue.dev + Ollama**（本地 AI 代码助手）|
| 来源 | windsurf.com / docs.windsurf.com / SEMrush（2026-07-06）|
