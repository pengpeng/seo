## 项目理解（前置）

Replit 是一个全浏览器的云端 IDE + AI agent 平台，用户无需本地开发环境，从自然语言 prompt 到部署上线都在同一界面完成。2016 年创立，最初定位是在线编译器/教育工具，2024-2026 年通过"vibe coding"叙事（AI 驱动、非开发者也能造 app）完成跨越式增长：2026 年 3 月以 $9B 估值融资 $400M，85% 的 Fortune 500 企业有用户在其平台上构建应用。Replit Agent 4（2026 年 Q2 发布）支持多 agent 并行、Design Canvas、Plan Mode，将平台从"AI 辅助编码"升级为"自主造物"。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 浏览器 IDE + AI 编码 agent，提示词驱动从零到部署 |
| 开源 / 许可证 | 闭源商业 SaaS；早期部分组件开源，核心平台不开源 |
| 主仓库 | 无公开主仓库（闭源）|
| 核心功能 | Replit Agent 4（自主多 agent 并行）、Design Canvas（设计画布）、Plan Mode、内置数据库/托管/Auth、Connectors、Figma 导入 |
| 商业模式 / 定价 | Free（有限）/ Core $20/月（$20 credits）/ Pro $100/月（15 协作席位 + Turbo Mode）/ Teams ~$40/用户/月 / Enterprise 定制；effort-based credits 按任务复杂度计费 |
| 差异化 / 价值主张 | 零配置全托管：不需要装任何东西，打开浏览器即开发、部署、协作；AI Agent 端到端，从 prompt 到 URL 全自动 |
| 主要竞品（初判）| Lovable、Bolt.new、Cursor、Windsurf、GitHub Copilot、v0 by Vercel、Base44 |
| Olares Market | 未上架（闭源）；Olares 平替：Code Server ✅ 已上架 + OpenCode ✅ 已上架 |
| 来源 | [replit.com](https://replit.com)、[replit.com/blog/replit-raises-400-million-dollars](https://replit.com/blog/replit-raises-400-million-dollars)、[docs.replit.com/billing/ai-billing](https://docs.replit.com/billing/ai-billing) |
