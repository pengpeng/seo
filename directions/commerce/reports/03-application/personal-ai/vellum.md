## 项目理解（前置）

Vellum Personal Intelligence 是一个**托管式个人 AI 助手**："你给它起名字、设性格、给它一个要学习的世界，从此它完全按你的方式塑造"。它默认跑在 **Vellum Cloud**（加密、隔离、always-on，笔记本合上也在跑），也可**在你自己的 Mac 上自托管**。助手可拥有自己的邮箱/账号/登录，能读文件、管日历、订餐、写 App、通过 macOS 无障碍 API **直接控制你的 Mac**，敏感操作需你批准。记忆系统分两层：小体积 always-loaded 工作记忆 + 按对话选择性检索的知识图谱。官方把自己定位成"比 OpenClaw 更成熟、更好用"的那一档——首页对比表把 OpenClaw / Hermes 标为 "self-hosted only、DIY 安全、你自己搭记忆栈"，把 Claude Cowork 标为 "Anthropic 云、无沙盒"，而 Vellum 主打 "managed memory + built-in security + Cloud or self-hosted + managed OAuth"。

母公司 Vellum（YC W23，创始 CEO Akash Sharma 等）原是 LLM 应用开发 / LLMOps 平台（prompt 工程、workflow、评测、模型 leaderboard），这条业务贡献了域名当前几乎全部自然流量；Personal Intelligence 是其 2026 年围绕"个人智能"重定位后的旗舰新品。

## 核心洞见

1. **品牌护城河：薄。品牌词多义 + 新品牌 SEO 冷启动，正面无需硬刚，但也不必怕。** vellum.ai 的 SEMrush Rank 91,570、$153K/月流量估值看似成熟，但**这套 SEO 全建立在旧 LLMOps 业务的 leaderboard/benchmark/workflow 内容上，与 Personal Intelligence 产品几乎无关**。PI 产品的自然词近零，处于冷启动期。品牌词 `vellum`（22,200）又被"犊皮纸 / Vellum 电子书排版软件 / Blender Vellum"严重稀释（KD 69、仅排第 2），真正专属的 `vellum ai`（880）量很小。**结论：品类词与对标词层面，Olares 完全有资格与之同场竞争。**

2. **可复制的打法：程序化 leaderboard 页 + "X alternatives" 博客矩阵，且全靠自然流量（零买量）。** Vellum 最有效的引流资产是 `/llm-leaderboard`（`best llm`、`llm leaderboard april 2026`、`best llm for coding` 全排第 1）与 `/blog/best-zapier-alternatives`、`/blog/best-n8n-alternatives` 这类"替代品清单"页。它**完全不投 Google Ads**。Olares Market 可对每个上架应用做同构的 "X alternative / best self-hosted X / X on Olares" 落地页矩阵——这是被 Vellum、Lovable 反复验证的低成本打法。

3. **对 Olares 最关键的 3 个词**：
   - **`openclaw alternative`（320 / KD 10）**：KD 极低、意图明确、跨 poke/genspark-claw 三报告共有的最高优先级词；Hermes Agent on Olares 是直接替代载体。
   - **`hermes agent`（1,900 / KD 30 / CPC $8.58）**：量、KD、CPC 三者俱佳，Olares 已上架，可写独立"Hermes Agent on Olares"教程吃满流量。
   - **`what is claude cowork`（2,900 / KD 25）**：Vellum 首页自比 Claude Cowork——借这个大流量认知词写"Claude Cowork 是什么 + 数据留在自己设备的开源自托管平替 on Olares"，一箭双雕承接 Cowork 与 Vellum 两波受众。

4. **最大攻击面：Vellum 自己把"自托管"当卖点，却是残缺的自托管——这正是 Olares 的切入缝。** Vellum 首页对比表主打 "Cloud or self-hosted"，但它的自托管只能在**一台 Mac** 上、要笔记本开着、无 always-on、无分布式；托管版则要租 Vellum Cloud + 付费 credits + API 成本。Olares 的叙事直击："你要的是 always-on 的自托管——那就是一朵你自己的云。Olares + OpenClaw/Hermes = 笔记本合上也在跑、数据不出家门、本地独显推理砍掉 API credits。" 攻击词：`self hosted ai assistant`（KD 0）、`openclaw self hosted`（KD 0）、`is openclaw free`（KD 16）、`claude cowork pricing`（KD 0）。

5. **隐藏低 KD 金矿：Claude Cowork 词族 + 一批 KD≤15 的品类/替代词。** `claude cowork pricing`（KD **0**，$7.42）、`claude cowork alternative`（KD **0**，$6.45）、`claude cowork review`（KD **0**）、`claude cowork vs claude code`（KD **0**，$12.54）——Anthropic 的新品类词几乎全部 KD=0，Olares 可低成本抢占并植入自托管叙事；再叠加 `manus alternative`（KD 8）、`openclaw alternative`（KD 10）、`proactive ai assistant`（KD 12）、`ai chief of staff`（KD 14）、`is openclaw free`（KD 16）、`second brain ai`（KD 16）一批极低 KD 词。

6. **GEO 前瞻布局。** `open source personal ai`、`personal ai assistant self hosted`、`self hosted personal ai assistant`（均近零量）语义与 Olares 完美契合；`what is personal intelligence`（70 / KD 30）是 Vellum 自造的品类叙事词——建议提前发布权威内容占 AI Overview / Perplexity 引用位，把"personal intelligence 应该归你自己（自托管）"这层意义抢先绑定到 Olares。

7. **与既有分析/报告的关联。** 本报告与同分区 [poke.md](./poke.md)、[genspark-claw.md](./genspark-claw.md) 高度互补，三者共享 `openclaw alternative`（320 / KD 10）、`hermes agent`（1,900 / KD 30）、`is openclaw free`（480 / KD 16）、`ai employee`（1,000 / KD 23）等核心词——建议 seo-content 阶段将三报告的 OpenClaw/Hermes 平替叙事合并成"Best OpenClaw / Personal AI Assistant Alternatives (self-hosted on Olares)"主词大文，并把 Vellum、Poke、Genspark Claw、Claude Cowork 作为并列对比对象。**本报告新增的高价值补充是 Claude Cowork 词族（KD 普遍为 0）与 `manus alternative`（KD 8）、`ai chief of staff`（KD 14）**，olares-500-keywords 词表目前缺此类词。
