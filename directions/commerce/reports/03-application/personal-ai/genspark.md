## 项目理解（前置）

Genspark（母公司 Mainfunc Inc.）由前百度副总裁 Eric Jing（微软 Bing 创始团队）和前 Google 搜索排名工程师 Kay Zhu 于 2023 年创立，总部 Palo Alto，设新加坡和东京办公室。它不自研大模型，而是**编排 70+ 个前沿模型**（OpenAI/Anthropic/Google），根据子任务路由最优模型，最终拼合成文档、PPT、表格、代码等完整交付物。2025 年从 AI 搜索转型 Agentic AI，2026 年发布"首位 AI 员工" Genspark Claw（桌面常驻 Agent，专属云电脑执行多步任务）和 Workspace 4.0（AI 设计、AI 幻灯片、AI 表格、M365 集成）。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 多模型编排的全能 AI 工作台——Super Agent 自动完成研究/内容创作/代码/数据分析/电话等复杂任务 |
| 开源 / 许可证 | **闭源**，纯 SaaS |
| 主仓库 | 无公开仓库 |
| 核心功能 | Super Agent（30+ 模型 + 150+ 工具 + 700+ MCP 集成）、Genspark Claw（桌面 AI 员工）、AI Slides/Sheets/Docs、AI Design（Claude Opus 4.7 驱动）、AI Phone Calls |
| 商业模式 / 定价 | Free（100 credits/天）；Plus $24.99/月（10K credits）；Pro $249.99/月（125K credits）；Team $30/座；Enterprise 定制；2026 年底前付费套餐 AI chat/图生成零 credits |
| 差异化 / 价值主张 | 一个订阅替代 ChatGPT Plus + Claude Pro + 多模态工具；Genspark Claw = "托管的 OpenClaw"，用户无需自建运维 |
| 主要竞品（初判）| Manus AI、Perplexity、OpenClaw（开源平替）、Claude Cowork、ChatGPT |
| Olares Market | 未上架（闭源 SaaS，无法自部署） |
| 来源 | [官网](https://www.genspark.ai/)、[TechTimes 2026-06-29](https://www.techtimes.com/articles/319240/20260629/genspark-expands-its-ai-workspace-openai-anthropic-microsoft.htm)、[GetLatka 2026-06-23](https://getlatka.com/companies/genspark.ai)、[Tracxn](https://tracxn.com/d/companies/genspark/__DlTsNPTygT2CGPOJLBFzdx5Sw1bv2C7rP7f8bbYirtg) |
