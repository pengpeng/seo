## 项目理解（前置）

Kimi Claw 是 Moonshot AI 于 2026 年 2 月 15 日推出的**云端托管版 OpenClaw**——将开源框架 OpenClaw（381K+ GitHub Stars，MIT 许可）打包成"在浏览器 Tab 里一键启动、24/7 在线的个人 AI Agent"，无需本地 Docker 或 VPS。底层模型为 Kimi K2.5/K2.6（1 万亿参数 MoE，SWE-Bench Pro 58.6%），内置 ClawHub 技能市场（5,000+ 技能）及 40 GB 云存储。核心价值主张是**把 OpenClaw 的技术门槛归零**——代价是数据上云（Moonshot 中国服务器）并锁定 $39/月 Allegretto 订阅。

## 核心洞见

1. **品牌护城河**：Kimi Claw 品牌词（1,600/月，KD 42）有护城河但并不坚不可摧——它的品牌流量依赖父品牌 kimi.com，且产品刚上线，品牌认知仍在建立期。更重要的是，Kimi Claw 的核心竞争词（openclaw alternative、openclaw self-hosted）KD 极低，正面迂回切入可行。

2. **可复制的打法**：Kimi Claw 没有自己的 SEO 主站，流量完全靠 kimi.com 父域 + 第三方评测文章（aihackers.net、toolworthy.ai）收割。Olares 可以主动写"Kimi Claw vs OpenClaw on Olares"对比文，抢占第三方评测位置——这些词 KD 普遍低于 30。

3. **对 Olares 最关键的 3 个词**：
   - **`openclaw alternative`**（320/月，KD=10）：最低 KD + 最直接替代意图，Olares 已上架 OpenClaw，一篇文章直接承接。
   - **`openclaw ollama`**（880/月，KD=0）：零竞争，Olares + OpenClaw + Ollama 本地 LLM 三者组合文章可立刻拿到排名。
   - **`localclaw`**（170/月，KD=0，CPC $10.47）：搜索"localclaw"的用户明确想要"本地跑的 Claw"，Olares 是最具说服力的答案。

4. **最大攻击面**：Kimi Claw 的核心痛点是三重锁定——**$39/月订阅**（本地 OpenClaw + VPS 只需 $2-5/月）、**数据上中国服务器**（隐私风险，中文评测网站已在讨论）、**锁定 Kimi K2.6 模型**（不能用本地 Ollama / Claude / GPT）。数据主权词（openclaw privacy，KD=0，CPC $3.37）是隐藏但高价值攻击口。

5. **隐藏低 KD 金矿**：
   - `openclaw kimi`（170/月，KD=0）：无人守；连接 Olares+OpenClaw 和 Kimi K2.6 API 的教程词
   - `kimi ai agent`（50/月，KD=0）：Kimi Claw 核心语义，KD=0
   - `cloud ai agent`（10/月，KD=0，CPC $7.81）：量极小但 GEO/AI Overview 定义位机会

6. **GEO 前瞻布局**：
   - `self-hosted ai agent`（量=0，CPC $4.59）：AI 概念定义词，抢 Perplexity/AI Overview 引用段落
   - `kimi claw alternative`（新词，尚无 Semrush 数据）：近期出现在评测文章标题，6 个月内量可能快速增长
   - `openclaw vs kimi`（新词）：对比意图词，进文章 FAQ 段落

7. **与既有分析的关联**：OpenClaw 市场报告（[openclaw.md](/Users/pengpeng/seo/directions/market/reports/openclaw.md)）已有完整词库（openclaw skills 3,600/月，openclaw wechat CPC $15.92 等）。Kimi Claw 的报告与 OpenClaw 报告形成**词库互补**——Kimi Claw 侧贡献"替代意图"叙事，OpenClaw 侧贡献"安装/集成/本地 LLM"教程词。两者共同支撑一个内容簇："自托管 OpenClaw 完全指南（对比 Kimi Claw 云版）"。
