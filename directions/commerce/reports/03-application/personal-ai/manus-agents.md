## 项目理解（前置）

Manus 在个人 AI 维度的核心产品是 **Manus Desktop App（My Computer 功能）**：通过 CLI 而非截图式 GUI 控制本地电脑，实现文件整理、本地应用开发、GPU 任务调度等工作，并支持从任意设备远程分配任务给常驻的家用/办公电脑。与 Anthropic 的截图式 Computer Use 不同，Manus 走"云端 Agent + 本地 CLI 执行"的混合路径，每条命令须用户显式审批。Meta 宣布以约 20 亿美元收购后，因中国监管要求于 2026 年 6 月正式取消，Manus 独立运营并再融资中；ARR 约 $400–500M 证明其个人/专业用户付费基础扎实。

## 核心洞见

1. **品牌护城河（能否正面刚？）**：
   Manus 在 personal AI 维度的品牌词（manus desktop、manus browser operator）KD 在 34–67——"manus browser operator"（KD=34）尚可攻，但主站排名第 1 是硬壁。**在"计算机控制"大品类词（claude computer use / computer use agent）上，Manus 自身未建立内容护城河——这才是真正的机会窗口**。

2. **可复制的打法**：
   Manus 靠工具落地页（image generator / PPT generator）获取泛类流量（见伴报告），但在 personal AI 助手这个品类几乎无专项内容。可复制的打法：建"Manus My Computer vs OpenClaw/Hermes Agent"对比页，以及"hermes agent + Olares 部署教程"——两者 KD 均在 20–37 之间，产出即可直接排名。

3. **对 Olares 最关键的词**：
   - `hermes agent`（1,900，KD=30，CPC=$8.58）——最佳单篇文章机会：量合理、竞争低、CPC 极高（商业价值）；Hermes 在 Olares Market；一篇"如何在 Olares 上运行 Hermes Agent"即可拿下
   - `ai personal assistant`（3,600，KD=37）——最大品类词，写"最佳开源个人 AI 助手"可触达 ai personal assistant + personal ai assistant 合计 ~5,500 月搜
   - `hermes agent vs openclaw`（170，KD=20，CPC=$6.28）——Olares 是唯一能同时运行两者的平台；这个词天然导向"在 Olares 上两者都跑"的答案

4. **最大攻击面**：
   - **信用点消耗不透明**：个人用户 $40/月的 Pro 仅能完成约 8 次深度任务；Hermes Agent on Olares = 永久免费（只付 API token）
   - **Meta 收购流产**：服务连续性疑虑（正在寻求再融资）→ 个人用户对"付费却可能停服"的风险敏感；`manus alternative` 系列词搜索量暴增的根本原因
   - **"My Computer"功能仍须云连接**：无法完全离线；Hermes Agent 完全本地，`private ai assistant` 搜索用户天然偏向 Olares 方案

5. **隐藏低 KD 金矿**：
   - `hermes agent vs openclaw`（170，**KD=20**，CPC=$6.28）——两大开源产品都在 Olares；写对比文 Olares 是天然赢家
   - `browser use ai`（390，**KD=35**，CPC=$4.23）——OpenClaw 是 browser use 类最热开源项目，Olares 降低其部署门槛
   - `best personal ai`（40，**KD=38**，CPC=$2.09）——评测词，量虽小但并入主文边际成本=0

6. **GEO 前瞻布局**（近零量但语义精准，抢 AI Overview / Perplexity 引用位）：
   - `personal ai assistant open source`（10，KD=0）——完全匹配 Hermes on Olares；放入文章 FAQ 段
   - `hermes agent ai`（90，KD=0）——进 FAQ；答：Hermes Agent 是什么、如何自托管
   - `manus computer use`（20，KD=0，CPC=$4.55）——对比问题词；文章段落答："Manus My Computer vs 自托管 CLI agent 的区别"
   - `ai personal secretary`（20，KD=0，CPC=$5.20）——场景描述词；Hermes on Olares via Telegram = "你口袋里的 24/7 AI 秘书"

7. **与既有分析的关联**：
   - 伴报告 [manus.md](/Users/pengpeng/seo/directions/commerce/reports/03-application/general-agent/manus.md) 已覆盖通用 agent 维度；本报告专注"personal AI"与"computer use"词簇，两份报告不重叠
   - [hermes-agent.md](/Users/pengpeng/seo/directions/market/reports/hermes-agent.md) 是 Hermes Agent 的独立 SEO 报告；本报告在 Olares 叙事角度上将两份报告串联——`hermes agent vs openclaw`（KD=20）是跨报告文章的核心词
   - olares-500-keywords 词表未收录 `hermes agent`、`ai personal assistant`、`computer use agent` 系列词；本报告补充了 15+ 有效词，其中 `ai personal assistant`（3,600 月搜）价值最高，建议补录
