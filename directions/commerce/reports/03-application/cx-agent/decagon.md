## 项目理解（前置）

Decagon 是面向中大型数字优先品牌的企业级 AI 客服 Agent 平台，由 Ashwin Sreenivas 与 Jesse Zhang 于 2023 年创立，总部位于旧金山。产品核心是"Agent Operating Procedures（AOP）"——用自然语言描述 AI Agent 的行为规则，跨 Chat / Email / Voice / SMS 等全渠道统一运行。与 Sierra AI 不同，Decagon 不依赖 FDE 驻场管理服务，更强调客户团队可自主配置；定位"AI 礼宾（AI Concierge）"，追求端到端解决（End-to-End Resolution）而非单纯工单偏转。2025 年 1 月完成 $250M D 轮，估值在六个月内从 $1.5B 飙升至 $4.5B，成为 CX AI 赛道最快上升的独角兽之一。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 企业级 AI 礼宾 Agent 平台：AOP 配置行为 + 全渠道 + 结果计费 + 自主可观测 |
| 开源 / 许可证 | 完全闭源；无开源版 |
| 主仓库 | 无（闭源 SaaS） |
| 核心功能 | AOP（Agent Operating Procedures，自然语言配置 AI 行为）、Autopilot（自主 End-to-End 解决）、Agent Assist（人工辅助层）、Watchtower（生产 QA 监控）、Voice（ElevenLabs 合作，2025-02 发布）、多渠道统一（Chat/Email/Voice/SMS） |
| 商业模式 / 定价 | 完全 cloud-only，结果计费（per-resolution / per-conversation 两种模型）；无公开定价页；第三方估算年合同 $95K–$590K+，中位 ~$386–$400K；$50K 最低门槛 |
| 差异化 / 价值主张 | AOP 让非工程师团队自主配置 AI 规则；ProductBench（行业首个端到端 CX 评测基准）；可验证 AI 决策过程（Verifiable AI）；Chime 70% 解决率 / Duolingo 80% 偏转率等标杆案例 |
| 主要竞品（初判） | Sierra AI（最直接，同样 cloud-only 管理服务）、Intercom Fin（$0.99/resolution 透明定价）、Zendesk AI、Ada、Forethought、Salesforce Agentforce、Assembled |
| Olares Market | 未上架（闭源）；开源近似路线：Dify ✅ + Flowise ✅ 已上架，Chatwoot（开源客服 Inbox）尚未上架，Botpress / Rasa 自建方案 |
| 来源 | [decagon.ai](https://decagon.ai)、[businesswire Series D](https://www.businesswire.com/news/home/20260128580542/en)、[getmacha 深度评测](https://www.getmacha.com/blog/decagon-ai-complete-guide)、[eesel AI 定价分析](https://www.eesel.ai/blog/decagon-ai-pricing)、[myaskai 完整指南](https://myaskai.com/blog/decagon-ai-complete-guide-2026) |

## 核心洞见

1. **品牌护城河：Decagon 自身护城河在加速建设，但并非不可绕开。** "decagon ai"（6,600/mo）+ "decagon"（40,500/mo）等品牌词 90% 流量由自己吃掉，KD 54–56，短期无法正面争夺。真正的机会在两个侧翼：①Decagon 自己**没有排名**的品类大词（`ai customer support` / `conversational ai platform` 等，自然流量几乎为零，全靠付费广告支撑）；②Decagon 制造但尚未垄断的新兴概念词（`autonomous customer service` KD 10、`agent operating procedures` KD 24、`decagon aop` KD 13）。

2. **最可复制的打法：买竞品品牌词 + Glossary 程序化落地页双轨。** Decagon 在付费端买了大量 Sierra AI 关键词（"sierra ai"、"sierra ai pricing"、"sierra ai competitors"），这是最直接的截流手法。自然端则通过 `/glossary/` 路径建设行业词典（已覆盖 knowledge base、tokens、BPO、latency 等），以中低 KD 词切入大量信息流量——"what is a knowledge base" 90,500/mo 排第 15，一旦升入前三将是数倍级流量增长。此打法完全可以复制到 Olares 博客/文档。

3. **对 Olares 最关键的三个词：`decagon vs sierra`（260/mo，KD 7）、`autonomous customer service`（320/mo，KD 10）、`decagon alternatives`（40/mo，KD 2）。** `decagon vs sierra` 是 Decagon 自己付费购买、KD 极低的买家意图词——写"三方对比（Decagon vs Sierra vs 自托管 Olares）"既能抢 Decagon 自己投放的词，也能为 Olares 建立自托管参照系。`autonomous customer service` 是 Decagon 的核心营销叙事词，KD 仅 10，Olares 可在"自主客服 = 自托管 + 成本透明"角度切入。`decagon alternatives` KD=2 极低，CPC $27.10 反映强商业意图，现在写胜率极高。
