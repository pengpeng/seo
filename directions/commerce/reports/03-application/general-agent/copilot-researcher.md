## 项目理解（前置）

Microsoft Copilot Researcher 是 Microsoft 365 Copilot 内置的深度研究 Agent，于 2025 年 3 月发布、2025 年 6 月 GA，由微软第一方构建，专为企业用户设计。其核心差异化在于**同时检索公开网页与组织内部数据**（Outlook 邮件、Teams 会话、SharePoint 文件、会议纪要），并使用 **Critique/Council 双模型交叉验证**机制（GPT 起草 + Claude 审稿）产出带引用的结构化研究报告。2026 年 3 月推出的 Council 模式更可同步运行两套模型并展示分歧，是目前闭源深度研究产品中企业数据融合最深的产品。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 企业内外数据融合型深度研究 Agent，M365 Copilot 内置，一报告同时拥有公司内部知识与实时网页情报 |
| 开源 / 许可证 | **闭源，Proprietary**（微软第一方，不可修改或自托管） |
| 主仓库 | 无公开仓库 |
| 核心功能 | 多步骤深度研究、内部数据融合（Graph/Connectors）、Bing 网页检索、Critique 自动审稿、Council 多模型并行对比、结构化报告（带图表与引用）、iOS/Android 支持 |
| 商业模式 / 定价 | 包含在 Microsoft 365 Copilot 订阅内（$30/用户/月）；Critique 约贵 20%、Council 约贵 2.5×；25 次/用户/月联合查询上限（Researcher + Analyst 共享） |
| 差异化 / 价值主张 | 唯一能同时扒公司内部邮件/文件与外部网页的研究 Agent；企业合规内置（M365 权限体系）；Critique 机制可减少幻觉 |
| 主要竞品（初判） | ChatGPT Deep Research（OpenAI）、Gemini Deep Research（Google）、Perplexity Pro、Claude Deep Research；开源平替：DeerFlow 2.0（字节开源，本地/自托管） |
| Olares Market | **未上架**（闭源；开源平替 DeerFlow 2.0 可在 Olares 自托管） |
| 来源 | [learn.microsoft.com/researcher-agent](https://learn.microsoft.com/en-us/microsoft-365/copilot/researcher-agent)、[learn.microsoft.com/faq-researcher](https://learn.microsoft.com/en-us/microsoft-365/copilot/faq-researcher)、[support.microsoft.com: get-started-researcher](https://support.microsoft.com/en-gb/topic/get-started-with-researcher-in-microsoft-365-copilot-e63ab760-f3de-4c47-ae87-dad601b0e9c4)、[appliedai.tools: Multi-Model](https://appliedai.tools/microsoft/microsoft-researcher-multi-model-intelligence/) |

## 核心洞见

1. **品牌护城河：M365 生态锁定，正面硬刚无胜算**。"microsoft 365 copilot"（49,500, KD100）、"copilot 365"（9,900, KD95）等品牌词完全不可攻击。Copilot Researcher 本身（390/mo, KD32）是产品词而非品类词，且产品的核心价值（企业内部数据融合）是 Olares 短期内无法直接复制的——因此**不建议正面抢品牌词，转走品类词 + 开源替代词路线**。

2. **可复制的打法：深度研究品类正在爆发，"比较页 + 开源清单"是进场口**。gemini deep research（8,100）、chatgpt deep research（5,400）说明整个品类词盘子极大；但 KD 均在 44-69，Olares 内容力无法直接排名这些词。**可复制 Lovable 的"guides/alternatives"程序化落地页打法**：针对每个主流深研工具写对比页，把 DeerFlow on Olares 作为开源自托管代表嵌入。

3. **对 Olares 最关键的词：`deerflow ai`（KD=0，CPC=$9.24）、`ai research agent`（KD=15）、`deerflow`（KD=39）**。`deerflow ai` 是当前所有词里性价比最高的单词——零竞争、高 CPC 证明商业意图、量已达 110/mo（且实际需求远高于 Semrush 显示量）。上架 DeerFlow 2.0 到 Olares Market 并写一篇"在 Olares 上安装 DeerFlow"教程，可以近乎零成本吃下这个词。

4. **最大攻击面：25 次/月 query 上限 + M365 生态锁定**。Copilot Researcher 的最大痛点是：(a) 25 次/月联合上限（Researcher + Analyst 共享），Council 模式消耗更多；(b) 必须购买 $30/用户/月的 M365 Copilot 许可。这两点正是 Olares + DeerFlow 2.0 的差异化叙事核心：**"无 query 上限、不绑微软、数据不出本地、免 SaaS 订阅费"**。关注词：`what is copilot researcher`（KD0）、`copilot researcher vs analyst`（出现在 fullsearch）等均是切入的信息词。

5. **隐藏低 KD 金矿：`deerflow ai`（KD=0, CPC=$9.24）+ `ai research agent`（KD=15, CPC=$5.29）**。前者是有 CPC 但零竞争的品牌词，后者是有月量（210）且 KD 极低的品类词，这两个词组合已足够支撑一篇"自托管 AI Research Agent（DeerFlow on Olares）"文章获得排名。

6. **GEO 前瞻布局**：`self hosted deep research`（KD=0）、`open source research agent`（KD=0）、`what is copilot researcher`（KD=0）目前近零量，但随着企业对数据主权需求增长，这些词的搜索意图正在形成。建议提前写权威内容，抢 AI Overview / Perplexity 引用位。特别是"which ai is best for deep research"词族（CPC $3-11）正在快速形成，多道语义问题词均是 GEO 候选。

7. **与既有分析的关联**：DeerFlow 词族（590+110）与 `olares-500-keywords` 中的"AI Agent / 本地 LLM"词族高度互补。本报告的开源研究 Agent 叙事与 Manus 报告中的"DeerFlow 作为开源替代"初判完全一致——建议将 DeerFlow on Olares 作为跨多个 general-agent 竞品报告的统一"开源研究 Agent"叙事锚点，形成 `/general-agent/` 类目下的程序化内容矩阵。
