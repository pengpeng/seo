> 域名：chatgpt.com（参考 openai.com）| SEMrush US | 2026-07-06
> ChatGPT Agent 是 OpenAI 将 Operator（网页操控）+ Deep Research（深度研究）+ 代码执行三合一的旗舰 AI Agent，面向付费用户提供完整的自主任务执行能力。

## 项目理解（前置）

ChatGPT Agent 于 2025 年 7 月 17 日正式发布，由 OpenAI 将三大能力整合：Operator 的可视化浏览器操控（点击/填表/滚动）、Deep Research 的多轮深度研究报告生成、以及终端代码执行与外部 App Connector（Gmail / Google Drive / GitHub 等）。它取代了原先独立存在的 Operator 研究预览，Deep Research 功能则作为子集保留并可单独调用。产品定位为"无需技术技能，从对话直接驱动多步自主任务"，是 OpenAI 面向个人和企业的最高端旗舰功能。

| 项目 | 内容 |
|------|------|
| 一句话定位 | OpenAI 旗舰 AI Agent：将网页操控 + 深度研究 + 代码执行合一，付费订阅驱动 |
| 开源 / 许可证 | 闭源，商业 SaaS；模型权重不开放 |
| 主仓库 | 无开源仓库；官方 Agent SDK（openai-agents-python）开源于 GitHub，约 7k+ Stars |
| 核心功能 | ① 可视化浏览器 Agent（网页操控） ② Deep Research 多轮合成报告 ③ 终端代码执行 ④ 外部 App Connector（Google Drive、Gmail 等） ⑤ 多任务并行、中途人工介入 |
| 商业模式 / 定价 | Plus $20/月（10 次 Deep Research/月）；Pro $100/月（~5x Plus）；Pro $200/月（250 次 Deep Research/月 + 1M token 上下文）；Business/Enterprise 按席位定制；Agent Mode：Pro 400 条/月，其余付费用户 40 条/月 |
| 差异化 / 价值主张 | ① 品牌心智最强（chatgpt = AI 代名词）② 三合一打通研究→行动闭环 ③ 官方 Connector 生态（非开源集成）④ GPT-5.5 + o3 模型加持 |
| 主要竞品（初判）| Gemini Deep Research（Google）、Perplexity Deep Research、Manus（国内）、Claude Research（Anthropic）；开源替代：DeerFlow 2.0（ByteDance MIT） |
| Olares Market | 未上架（闭源 SaaS）；开源替代 DeerFlow 2.0（bytedance/deer-flow）可在 Olares 自托管 |

## 核心洞见

### 1. 品牌护城河

chatgpt.com 是 SR #26 的全球超级域名，品牌词 KD=100、月量 6800 万+，正面用品牌词单挑**毫无胜算**。但关键洞察是：**agent / deep research 子词 KD 仅 50-70**，距品牌核心词有明显落差，且 chatgpt.com 在这些子词上多处于位置 2-7——竞品内容仍有切入空间，尤其是"open source / alternative / local / cost"角度。

### 2. 可复制的打法

OpenAI 在 chatgpt.com 的内容结构非常清晰：
- `/features/agent/` 专页承接所有 agent mode 词，SEO 落地页功能单一；

### 4. 最大攻击面（痛点词）

ChatGPT Agent 的核心痛点集中在**付费壁垒**：
- Plus $20/月仅给 10 次 Deep Research，Pro $200/月才"解锁上限"；
- Agent Mode 每月 40 条（Pro 用户 400 条），超限需要额外积分；
- 所有查询数据归 OpenAI，隐私无法保证；
- 闭源黑盒，无法定制/扩展工具链。

攻击词：`chatgpt agent cost`（110/月，KD 47）、`is chatgpt agent free`（110/月）、`chatgpt deep research limit`（480/月，KD 46）、`chatgpt research credits`、`openai credits`（390/月，KD 52）。
