## 项目理解（前置）

Gemini CLI 是 Google 于 2025 年 4 月开源的终端 AI agent（Apache 2.0，TypeScript），将 Gemini 2.5 Pro 模型能力直接带入命令行，支持多文件代码编辑、shell 执行、MCP 工具扩展、Google Search grounding 和 GitHub Action 集成。免费个人用户可获得 1,000 次/天配额。**关键时间节点**：2026-06-18 Google 宣布对免费、AI Pro/Ultra 用户停止 Gemini CLI 服务，迁往 Antigravity CLI（agy）——一个闭源 Go 重写版本，这使得"Gemini CLI 已弃用"成为当前 SEO 窗口期的核心叙事。对 Olares/开源阵营而言，这是布局"Gemini CLI alternative"和开源平替内容的最佳时机。

## 核心洞见

1. **品牌护城河**：Gemini CLI 品牌词 KD 60-75 且被 github.com 占据 #1，正面刚品牌词无意义。但竞品对比词（"vs claude code" KD=14-27）和安装/使用词（大量 KD 40-60）是可攻击的。**最关键窗口**：2026-06-18 弃用公告后，"gemini cli alternative"/"deprecated"类词正在快速成长，KD 还处于 0-30 区间，是布局的最佳时机。

2. **可复制的打法**：geminicli.com 官网在大量 how-to 词上拿下 #1（KD 40-75），依靠的是官方权威性+深度文档。内容竞争方走不同路径：应攻替代/比较词（KD 低）和本地/开源信号词（近零竞争），而不是和官方抢安装教程。

3. **对 Olares 最关键的词**：
   - `gemini cli alternative`（KD=24，迁移期窗口词）
   - `qwen coder cli`（KD=28，Qwen Code on Olares 核心教程词）
