## 项目理解（前置）

Mendix 是由 Siemens 旗下的企业级低代码开发平台，面向大型企业 IT 团队与"Fusion Team"（业务+技术混合），允许用可视化模型驱动 IDE（Studio Pro）快速构建、部署、运维 Web/Mobile 应用组合。平台核心差异是企业级治理深度——内建 DevOps、多云部署（Mendix Cloud/私有云/本地）、可追溯审计与 AI 辅助（Maia），是 OutSystems 最直接的竞品。但平台完全闭源，月费高（标准版 $998+/应用，不含云基础设施费），形成明显的迁移壁垒与价格痛点。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 企业级低代码平台（Gartner LCAP Leader），用可视化模型驱动 IDE 构建企业应用组合 |
| 开源 / 许可证 | **完全闭源**；社区版免费但功能受限（共享数据库，无生产支持） |
| 主仓库 | 无（闭源）；官方 SDK/连接器有少量开源组件 |
| 核心功能 | 可视化域模型 + 拖拽页面编辑；Microflow/Nanoflow 逻辑建模；内建 Git 版本控制；Marketplace（数百个预置模块）；MCP 集成；AI 助手 Maia |
| 商业模式 / 定价 | 免费层（原型，共享 DB）；Standard $998/月/应用（云基础设施另计）；Premium 自定义报价；云 Token 按 RAM/CPU 计费 |
| 差异化 / 价值主张 | Gartner 连续 9 年 Leader；企业 ISV/OEM 支持；端到端 SDLC 治理；与 Siemens 工业系统深度集成 |
| 主要竞品（初判）| OutSystems、Appian、Microsoft Power Apps、Retool、Appsmith（开源） |
| Olares Market | **未上架**；开源平替 Appsmith、NocoBase、NocoDB 已在 Olares Market |
| 来源 | [mendix.com/platform](https://www.mendix.com/platform/)、[mendix.com/pricing](https://www.mendix.com/pricing/)、[Gartner LCAP 2024](https://www.gartner.com/)、[hostadvice.com review 2026](https://hostadvice.com/ai-app-builders/mendix-review/) |
