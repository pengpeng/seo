## 项目理解（前置）

Base44 是由以色列创始人 Maor Shlomo 在 2024 年底创立、2025 年 6 月被 Wix 以约 $92M（含对赌）收购的 AI 全栈 app builder。用户通过自然语言描述需求，平台自动生成包含数据库、认证、部署的完整 web 应用——无需任何代码。与 Lovable/Bolt 定位开发者不同，Base44 主打非技术业主，将后端配置完全自动化，号称"30 秒生成可用 app"。收购后 Wix 大规模押注推广（Q1 2026 广告支出超 $9,000 万含 Super Bowl），Base44 ARR 在 2026 年 5 月达 $150M。2026 年 6 月 Base44 推出自研模型 Base 1 以降低推理成本并强化差异化。([来源：TechCrunch](https://techcrunch.com/2025/06/18/6-month-old-solo-owned-vibe-coder-base44-sells-to-wix-for-80m-cash/) / [Wix IR](https://www.wix.com/press-room/home/post/wix-further-expands-into-vibe-coding-with-acquisition-of-base44-a-hyper-growth-startup-that-simplif) / [RuntimeWire](https://runtimewire.com/article/base44-base-1-model-maor-shlomo-wix-defensibility))

| 项目 | 内容 |
|------|------|
| 一句话定位 | 非技术用户版 AI 全栈 app builder，自然语言生成带数据库/认证/部署的 web app |
| 开源 / 许可证 | 闭源 SaaS（Wix 全资子公司）；2026-06 推出自研模型 Base 1 |
| 主仓库 | 无开源仓库；[docs.base44.com](https://docs.base44.com) 为文档站 |
| 核心功能 | 自然语言→全栈 web app；自动数据库/认证/部署；API 集成；内置分析；GitHub 导出（Builder+ 档） |
| 商业模式 / 定价 | Freemium（25 msg credits/月）；Starter $16/月；Builder $40/月（自定义域名+GitHub）；Pro $80/月；Elite $160/月；双 credits 计费（message + integration credits） |
| 差异化 / 价值主张 | 后端全自动（vs Lovable 需手动配置）；Wix 生态集成；Base 1 自研模型降成本；30 秒出 app |
| 主要竞品（初判）| Lovable（开发者）、Bolt.new（快速原型）、Replit（编码学习）、Bubble（重逻辑 no-code）、Glide（数据驱动 app） |
| Olares Market | 未上架（闭源）；平替 Appsmith ✅ 已上架、NocoBase ✅ 已上架 |
| 来源 | [base44.com](https://base44.com)、[docs.base44.com/Account-and-billing/Billing-and-plans](https://docs.base44.com/Account-and-billing/Billing-and-plans)、TechCrunch、Wix 2025 Annual Report |
