## 项目理解（前置）

OutSystems 是一家总部位于葡萄牙里斯本、现已全球化的企业级低代码平台公司，成立于 2001 年。其平台允许专业开发者与业务分析师用可视化 IDE（Service Studio / ODC）构建全栈应用，覆盖 Web、移动和 AI Agent，主打大型企业数字化转型和遗留系统现代化。2026 年平台已深度整合生成式 AI 能力，战略重心向"AI-native app building"转移，同时赞助大量与 vibe coding / Copilot AI 相关的付费关键词，与其企业定位形成明显反差——这反映出它在追赶 AI 原生开发平台的竞争叙事。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 面向大型企业的低代码全栈应用开发平台，侧重复杂、任务关键型应用 |
| 开源 / 许可证 | **闭源，商业许可**；无公开源代码 |
| 主仓库 | 无（闭源）；Forge 社区扩展可复用组件（outsystems.com/forge/）|
| 核心功能 | 可视化全栈开发（Web/Mobile）、AI Agent 内置、企业集成（200+ 连接器）、CI/CD + DevSecOps、ODC 云原生架构 |
| 商业模式 / 定价 | 纯订阅制，按"Application Objects（AO 复杂度）+ 用户数"计费；ODC 入门价 **$36,300/年**（3 环境 + 100 内部用户）；平均企业合同 **$220,000/年**，大客户可达 $340K+；无透明公开定价，必须联系销售；提供 10 天免费试用 |
| 差异化 / 价值主张 | 完整 SDLC 生命周期管理、高性能企业级应用（而非 citizen developer 工具）、Gartner 9 连领导者背书、ODC 云原生架构、AI 代理构建能力 |
| 主要竞品（初判）| Mendix、Microsoft Power Apps、Appian、ServiceNow App Engine、Salesforce Platform |
| Olares Market | **未上架**（闭源商业平台）；Olares 平替方向：Budibase（✅ 已调研）、NocoBase（✅ 已上架 Olares Market）、Appsmith（✅ 已上架 Olares Market）|
| 来源 | [官网](https://www.outsystems.com/)、[Superblocks 定价分析](https://www.superblocks.com/blog/outsystems-pricing)、[EuropeanStack 2026 Review](https://europeanstack.com/software/outsystems)、[HokAI](https://hokai.io/hub/tools/outsystems) |
