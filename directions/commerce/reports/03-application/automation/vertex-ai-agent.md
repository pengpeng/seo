## 项目理解（前置）

Google Vertex AI Agent Builder 是 Google Cloud 的企业级 AI Agent 全生命周期平台，覆盖构建（Agent Studio 低代码 + ADK 代码优先）、部署扩展（Agent Engine 托管运行时）、治理（Agent Identity / Gateway / Model Armor）和优化（Gen AI Evaluation + 可观测）四大支柱。2026 年 Google Cloud Next 大会上，该产品正式更名为 **Gemini Enterprise Agent Platform**，并将 Agentspace 并入，现有用户无需迁移。产品定位为已深度使用 Google Cloud（BigQuery、IAM、Workspace）的大型企业，是 Google Cloud 的 AI 旗舰，对比 AWS Bedrock Agents 和 Azure AI Agent Service。开源社区的替代方向为 Dify、n8n、Flowise、Rasa 等，其中 Dify 和 n8n 已上架 Olares Market。

| 项目 | 内容 |
|------|------|
| 一句话定位 | Google Cloud 企业级 AI Agent 全生命周期平台（Dialogflow CX + Vertex AI 整合） |
| 开源 / 许可证 | 闭源 SaaS；ADK（Agent Development Kit）为 Apache 2.0 开源框架，可独立使用 |
| 主仓库 | [google/adk-python](https://github.com/google/adk-python)（ADK 部分，★约 8k+；平台本体不开源） |
| 核心功能 | Agent Studio 低代码画布、ADK Python/Go/Java/TypeScript、Agent Engine 托管运行时、Model Garden（200+ 模型）、RAG Engine + Vector Search、Agent Gateway 治理 |
| 商业模式 / 定价 | 完全按量计费，四表计费：Agent Engine 运行时 $0.0864/vCPU·h、Session/Memory Bank 事件 $0.25/千次、Vertex AI Search $1.50–$6.00/千次查询、基础模型 token 单独计费；新客 $300/90 天免费额度 |
| 差异化 / 价值主张 | Google Cloud 原生集成（BigQuery、IAM、Workspace）；200+ 模型 Model Garden；企业级治理与合规；支持多框架部署（LangChain/LangGraph/CrewAI） |
| 主要竞品（初判）| Microsoft Copilot Studio、AWS Bedrock Agents、OpenAI Assistants API；开源：Dify、n8n、Flowise、Rasa |
| Olares Market | Dify ✅ 已上架；n8n ✅ 已上架（Vertex AI Agent Builder 本体为闭源云服务，不可自部署） |
| 来源 | [cloud.google.com/vertex-ai](https://cloud.google.com/vertex-ai)、[docs.cloud.google.com/gemini-enterprise-agent-platform](https://docs.cloud.google.com/gemini-enterprise-agent-platform/overview)、[betterclaw.io Vertex AI review 2026](https://www.betterclaw.io/blog/google-vertex-ai-agent-builder) |

## 核心洞见

1. **品牌护城河**：Google 品牌词（`vertex ai` 22,200/月，KD68；`google vertex ai` 5,400/月，KD69）正面硬刚无意义。但产品词 `vertex ai agent builder`（2,400/月，KD50）和下游对比/替代词仍可竞争。2026 更名"Gemini Enterprise Agent Platform"后品牌词尚未稳定，**Dialogflow 旧品牌词（2,900/月）将逐渐流失**，是短期内容机会窗口。

2. **可复制打法**：Vertex AI Agent Builder 本体无独立域名、无程序化落地页，内容主要靠 cloud.google.com 的权重压制。但竞争对手（n8n、Dify、Flowise 等开源工具的博客/文档站）正在大量制造"选型比较"内容（`n8n vs zapier` 1,900/月、`flowise ai` 1,600/月），**对比/替代文是可复制的内容打法**。Google ADK 系列词的极高 CPC（`google agent development kit` $106/次、`gemini agent builder` $110/次）证明企业买家在这个赛道有极强商业意图。

3. **对 Olares 最关键的词**：
   - `n8n alternative`（720/月，KD18）——Olares 直接机会词，KD 极低
   - `dify vs n8n`（140/月，KD13）——两个 Olares Market 应用同框比较
   - `enterprise ai chatbot`（480/月，KD25，CPC $94）——企业买家高意图词，Dify on Olares 精准替代 Dialogflow CX

4. **最大攻击面（定价痛点）**：四表计费复杂度是 Vertex AI Agent Builder 最核心的用户吐槽——真实生产成本因 Session 事件数、Search 查询量、token 消耗叠加，可达 $500–$2,000+/月，甚至有团队收到五位数账单。**"Vertex AI pricing" (480/月，KD39) 和 "vertex ai cost" (90/月，KD32，CPC $37) 是切入点**，用 Dify/n8n on Olares"一次部署、按硬件成本算"做对比。

5. **隐藏低 KD 金矿**：
