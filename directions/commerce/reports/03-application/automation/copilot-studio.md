## 项目理解（前置）

Microsoft Copilot Studio 是微软的低代码 AI Agent 构建平台，前身为 Power Virtual Agents（2023 年末更名并升级），集成于 Microsoft 365 / Power Platform 生态。它允许用户用自然语言或可视化界面构建对话式 AI Agent、接入 SharePoint/Teams/Dynamics 等企业知识源，并通过 600+ Power Automate 连接器接入外部系统。采用 **Copilot Credits 消耗计费**，与 Microsoft 365 Copilot ($30/user/month) 深度捆绑，**只运行在微软 Azure 云**，无法自托管。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 微软低代码 AI Agent / 对话机器人构建平台，深度嵌入 M365 生态 |
| 开源 / 许可证 | **闭源 SaaS**，云端运行 |
| 主仓库 | 无开源仓库（Microsoft Power Platform 相关 SDK 有部分开源组件） |
| 核心功能 | 自然语言构建 Agent、企业知识源（SharePoint / Dataverse / Web）接入、600+ 连接器、多渠道发布（Teams / 网站 / 电话）、多 Agent 协作 |
| 商业模式 / 定价 | 消耗制（Copilot Credits）：$200/月 = 25,000 credits；Pay-as-you-go $0.01/credit；M365 Copilot ($30/user) 含内部使用权；Enterprise E7 $99/user/month 套件含 |
| 差异化 / 价值主张 | 深度 M365 集成（Teams/SharePoint/Outlook）、低代码无需写代码、企业级治理/合规（GCC 可用）、与 Power Platform 无缝联动 |
| 主要竞品（初判）| Dify（开源）、n8n、Langflow、Flowise、Google Vertex AI Agent Builder、Botpress、Rasa |
| Olares Market | **Dify 已上架** ✅（[报告](/Users/pengpeng/seo/directions/market/reports/dify.md)）；Flowise、n8n 亦可在 Olares 部署；Copilot Studio 本身闭源不可上架 |
| 来源 | [microsoft.com/copilot-studio](https://www.microsoft.com/en-us/microsoft-365-copilot/microsoft-copilot-studio)、[learn.microsoft.com/copilot-studio](https://learn.microsoft.com/en-us/microsoft-copilot-studio/)、[pricing](https://www.microsoft.com/en-us/microsoft-365-copilot/pricing/copilot-studio)（2026-07-06） |
