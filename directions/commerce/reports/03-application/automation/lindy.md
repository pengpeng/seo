---

## 项目理解（前置）

Lindy 是面向专业人士的个人 AI 工作助手，核心定位是"全工作循环自动化"：统一处理邮件分拣与回复、会议准备与记录、日历调度、跟进提醒及 CRM 更新，并支持通过 iMessage/SMS 委托任务。与 Zapier/n8n 等偏开发者的工作流自动化工具不同，Lindy 以"自然语言委托结果"为核心差异化，主打非技术知识工作者。截至 2026-05 用户 >40 万，已融资 ~$50M，定价 $49.99–$199.99/月。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 个人 AI 执行助手，闭环管理收件箱/会议/日历/CRM |
| 开源 / 许可证 | **闭源 SaaS**；官方明确声明不适合自托管或开源工作流构建 |
| 主仓库 | 无公开 GitHub 仓库 |
| 核心功能 | 邮件分拣与起草、会议全程（准备→记录→跟进）、日历协调与调度、CRM 自动更新（HubSpot/Salesforce）、iMessage/SMS 手机委托 |
| 商业模式 / 定价 | SaaS 订阅；Plus $49.99/月，Pro $99.99/月，Max $199.99/月，Enterprise 报价；7 天免费试用 |
| 差异化 / 价值主张 | 跨 Gmail/Outlook/Slack/Notion/Zoom 的单一助手；phone-first 委托（iMessage/SMS）；HIPAA/SOC2/GDPR/PIPEDA 合规 |
| 主要竞品（初判） | Zapier（工作流自动化）、n8n（开源替代）、Gumloop（AI 自动化）、Fyxer AI（邮件/会议）、Superhuman（邮件效率） |
| Olares Market | **未上架**（Lindy 为闭源 SaaS，无法自托管；开源替代 n8n 已收录） |
| 来源 | [lindy.ai](https://www.lindy.ai)（2026-07-06）、[lindy.ai/pricing](https://www.lindy.ai/pricing) |
