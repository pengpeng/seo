## 项目理解（前置）

Make（前身 Integromat，2022 年更名）是一款面向中高级用户的**可视化、无代码/低代码工作流自动化平台**，提供 3,000+ 应用集成，通过"模块-画布"式流程图构建自动化场景（Scenarios）。与 Zapier 的线性"Zap"相比，Make 支持更复杂的数据路由、迭代器、聚合器、条件判断等操作，因此更受有一定技术背景的业务人员青睐。与 n8n 相比，Make 是纯云 SaaS（闭源、不可自托管），面向不想自建服务器的用户。Make 由捷克公司创立，2022 年被 Celonis（流程挖掘）收购，ARR ~$52.6M，处于中型 SaaS 成熟期。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 可视化画布工作流自动化，Zapier 的价格杀手，n8n 的商业云替代 |
| 开源 / 许可证 | **闭源 SaaS**，无公开代码仓库 |
| 主仓库 | 无（闭源） |
| 核心功能 | 可视化场景画布、3,000+ 应用集成、数据转换（迭代/聚合）、Webhooks、AI Agents（Beta）、MCP Server |
| 商业模式 / 定价 | 免费（1,000 credits/月）→ Core $9/月（10K credits）→ Pro $16/月 → Teams $29/月 → Enterprise 定制；credits = 每个模块操作消耗 1 credit |
| 差异化 / 价值主张 | 比 Zapier 便宜且功能更强（复杂数据处理）；比 n8n 易用（托管云）；视觉化画布更直观 |
| 主要竞品（初判）| Zapier（主要竞品）、n8n（开源自托管竞品）、Pabbly Connect（价格竞品）、Activepieces（新兴开源）|
| Olares Market | **未上架**（开源平替 n8n 已上架；[market 报告](/Users/pengpeng/seo/directions/market/reports/n8n.md)）|
| 来源 | [make.com/en/pricing](https://www.make.com/en/pricing)、[make.com/en/integrations](https://www.make.com/en/integrations)、公开融资/ARR 数据 |
