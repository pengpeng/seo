## 项目理解（前置）

n8n Cloud 是 n8n GmbH（柏林，创始人 Jan Oberhauser）基于开源 n8n 推出的托管 SaaS 产品，数据中心在法兰克福（EU 合规），面向技术团队和中小企业。与 Zapier/Make.com 的最大定价差异是：**按"执行次数（execution）"计费，而非"步骤数/操作数"**——一次工作流全量运行无论多少步都算一次，比 Zapier 的 task-based 计费更实惠（n8n 官方声称"一次 execution 可替代 Zapier 10,000 个 task"）。但 n8n 社区版（Community Edition）可免费自托管，造成 SaaS 版的最大竞争对手就是它自己。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 技术团队的开源工作流自动化平台，SaaS 云端版按执行次数收费 |
| 开源 / 许可证 | 是；Sustainable Use License + Apache-2.0（核心），[github.com/n8n-io/n8n](https://github.com/n8n-io/n8n)（56k+ ★） |
| 核心功能 | 400+ 集成节点、AI Agent 工作流、Webhook/定时触发、代码节点（JS/Python）、MCP 服务器 |
| 商业模式 / 定价 | Starter 20€/mo（2.5K executions）、Pro 50€/mo（10K）、Business 667€/mo（40K，含 SSO/Git）、Enterprise 询价；年付约八折 |
| 差异化 / 价值主张 | 执行按整条工作流计费（非步骤）；开源透明度+社区生态；技术用户可视化 + 代码混用 |
| 主要竞品（初判）| Zapier、Make.com、Activepieces、Gumloop、Latenode |
| Olares Market | ✅ 已上架（n8n 开源自托管版，直接一键部署） |
| 来源 | [n8n.io/pricing](https://n8n.io/pricing/)、[github.com/n8n-io/n8n](https://github.com/n8n-io/n8n) |

## 核心洞见

1. **品牌护城河**：n8n 对所有定价词（`n8n pricing`、`n8n cloud pricing` 等）均排名第 1，且是自己的最大竞争对手（社区版免费自托管）。正面刚品牌词意义不大；**Olares 的机会在"n8n Cloud 定价 vs 自托管成本对比"这条叙事线**——用户在搜索定价词时就是在做"付月费 vs 自建"的决策。

2. **可复制的打法**：n8n 在 Google Ads 上大量购买 **AI 模型名词**（veo 3、claude 3.7、ai picture generator）并导向工作流模板页——用 AI 热词流量喂养品牌认知，再转化为订阅。**Olares 可以学习这个"热门 AI 工具 + 工作流模板"路径**，用模板页 SEO 带动 n8n on Olares 的安装量。

3. **对 Olares 最关键的 3 个词**：
   - `n8n cloud pricing`（3,600/mo，KD=39）——定价决策词，直接对应"自托管省钱"叙事；
   - `n8n alternatives`（1,000/mo，KD=25）⭐——低竞争替代词，Olares Market 有多个 n8n 替代应用；
   - `zapier vs make vs n8n`（110/mo，KD=18，CPC=$9.94）⭐——KD 极低、CPC 极高的三方对比词，一篇文章命中多组关键词链路。

4. **最大攻击面**：n8n Cloud 的**定价结构跳跃**极为陡峭——Starter 20€（2.5K executions）→ Pro 50€（10K）→ Business 667€（40K + SSO/Git）。Business 比 Pro 贵 13 倍，而 SSO/SAML 是许多团队的必须项。这个价格悬崖是最大的用户焦虑来源（`n8n enterprise pricing` KD=24，`n8n pricing changes` KD=25，`n8n cloud cost` KD=19，CPC 普遍高），Olares 私有部署可以绕开这个价格墙。

5. **隐藏低 KD 金矿**：
   - `gumloop vs n8n`（170/mo，KD=17）⭐ + `n8n vs gumloop`（210/mo，KD=22）——Gumloop 是新兴 SaaS 竞争者，两者的对比词 KD 都极低，可以用一篇文章同时拿下；
   - `zapier vs make vs n8n`（110/mo，KD=18）——三方对比词 KD 意外低，CPC=$9.94 代表广告主认为这批用户高价值；
