## Gaps

### 未达门槛但值得跟踪的开源/小团队 CLI
- **Aider**（Paul Gauthier）：git-native 老牌 terminal agent，无 >$20M 融资
- **Goose**（Block→AAIF/Linux Foundation）：开源免费，2025-12 捐给 AAIF[27]
- **OpenCode**（社区/Olares Market）：开源替代
- **Continue.dev**：被 Cursor acqui-hire(~2026-06)，累计融资 ~$5M，Apache2.0 代码仍可用
- **Roo Code / Kilo Code**：Cline 生态延伸，无独立融资

### 结构变化
- Gemini CLI → Antigravity CLI（2026-06 对个人 Pro/Ultra 停服）
- Amazon Q Developer CLI → Kiro CLI（2027-04 EOL）
- Amp ← Sourcegraph 分拆（2025-12）
- Windsurf：Google acqui-hire → Cognition 吸收 → Devin 生态
- Cursor ← Continue acqui-hire（2026-06）

### 财务盲区
- GitHub Copilot / Codex / Junie / Jules / Kiro：无官方独立 ARR
- Amp standalone：分拆后无独立披露
- Factory Droids：有估值/融资，无公开 ARR
- Claude Code $8B+（2026-05）：仅第三方 [unverified]
