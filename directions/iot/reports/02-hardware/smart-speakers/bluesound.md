## 项目理解（前置）

Bluesound 是 Lenbrook Group 旗下（母公司同时拥有 NAD Electronics、PSB Speakers）的发烧 Hi-Fi 无线流媒体品牌，成立于 2012 年，总部位于加拿大安大略省。产品以 BluOS® 多房间操作系统为核心，覆盖无线音箱（PULSE 系列）、网络串流播放器（NODE 系列，旗舰 NODE ICON）、无线功放（POWERNODE）、CD 翻录中枢（VAULT）等品类，全系支持 Hi-Res Audio（24-bit/192kHz）并与 Roon Ready 认证深度绑定。与 Sonos 侧重多房间易用性不同，Bluesound 主打发烧友市场：支持 Roon、MQA、Tidal Connect、Qobuz Connect，是少数兼顾"价格合理"与"发烧级格式支持"的闭源硬件生态。

| 项目 | 内容 |
|------|------|
| 一句话定位 | 发烧 Hi-Fi 无线多房间流媒体，BluOS 操作系统 + Roon Ready 认证 |
| 开源 / 许可证 | 闭源商业产品（Lenbrook Group 私有）；BluOS 固件与 App 均不开源 |
| 主仓库 | 无开源仓库（bluesound.com / bluos.net） |
| 核心功能 | 24-bit/192kHz Hi-Res 本地/流媒体播放、BluOS 多房间同步、Roon Ready、AirPlay 2、Spotify/Tidal/Qobuz Connect、Alexa 语音集成 |
| 商业模式 / 定价 | 一次性硬件购买；NODE 约 $549、NODE ICON 约 $999、POWERNODE $799；App 免费；无月租 |
| 差异化 / 价值主张 | 发烧格式支持最全（MQA/Roon/Qobuz/Tidal Hi-Fi）、NAD/PSB 同源调音背书、比 Aurender/Lumin 亲民 |
| 主要竞品（初判）| WiiM（平价蚕食）、Sonos（多房间大众市场）、NAD Electronics（同母公司）、Cambridge Audio（英系竞品）、Naim |
| Olares Market | Music Assistant ✅ 已上架（Home Assistant 生态，可替代 BluOS 多房间控制层） |
| 来源 | [bluesound.com](https://www.bluesound.com)、[bluos.net](https://www.bluos.net)、[Lenbrook 官网](https://www.lenbrook.com)、[Roon Ready 认证列表](https://roon.app/en/roon-ready) |
