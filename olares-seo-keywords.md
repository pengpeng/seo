## 四、优先行动计划

### Phase 1（立即，0-3个月）：低KD快速得分

**目标：在60-90天内建立排名基础**

1. **Synology alternative**（KD=7）
   - 文章：「Synology NAS vs Olares One：2026年哪个更适合你？」
   - 落地页支持：olares.com/vs/synology

2. **Open WebUI**（KD=34，月搜 8,100）⭐最大机会
   - 文章：「How to Run Open WebUI on Olares in 5 Minutes」
   - 这是最高量+最低KD的组合之一

3. **Authelia + Headscale**（KD 29/33，月搜 1,900/2,900）
   - 这两个是 Olares 内置功能，做技术解析文章
   - 「Authelia SSO with Olares: Zero-Config Setup Guide」

4. **Cursor alternative / GitHub Copilot alternative**（KD 25/13）
   - 「Best Self-Hosted Coding AI Alternatives to Cursor in 2026」
   - 「Run Your Own GitHub Copilot with Olares」

5. **Home Assistant local LLM**（KD=49，月搜 110）
   - Olares 差异化词，做一篇深度指南
   - 「Connect Home Assistant to a Local LLM with Olares」

### Phase 2（3-6个月）：中KD量级词

1. **local LLM**（KD=39，月搜 2,900）- 系列内容
2. **Plex alternative / Jellyfin**（KD 33/76）- 媒体服务器专题
3. **CasaOS / Umbrel 对比系列**
4. **Vaultwarden / AdGuard Home 部署指南**（量大）

### Phase 3（6-12个月）：长期流量词

- Ollama、ComfyUI、Stable Diffusion、Immich 的深度指南
- Home Assistant、Nextcloud 对比内容
