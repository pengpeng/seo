# 第 3 页逐字稿 · Olares 与 Personal Cloud

首先，我想先介绍一下 Olares 是什么。

我们相信，未来三到五年，每个人都会有自己的 Personal Agent。你可以把它理解成一个更好用、更强大的 OpenClaw。它帮我们完成工作，处理日常事务，管理个人信息，打理数字生活。而且随时随地都能用——手机、眼镜、音箱，当然也包括电脑。

那 Personal Cloud 是什么？它就是用来跑这个 Agent 的家，一个始终在线的家。它让 Agent 一直运行，把完整的个人上下文放在一个地方。不管你用哪台设备，都能安全地连上它。

大部分 Personal Agent，可能会跑在公有云上——比如 OpenAI、Google、字节跳动托管的云。但也有一部分人和小企业，想自己来跑。为什么？因为隐私，因为数据所有权，因为可控性，因为更低的延迟，也因为长期的成本。

Olares 就是为这些人做的。

打个比方。Android 是智能手机的操作系统，Windows 是个人电脑的操作系统。而 Olares，是面向 Personal Cloud 的开源操作系统。我们是一家软件公司，总部在新加坡。

Personal Cloud 给一个人提供一个入口——单一、始终在线、随时随地能访问。同时，它也为这个人和他所有的 AI 设备，提供一个统一的上下文执行环境。

你想，今天我们的数据是散的。散在 Google、Meta、OpenAI、Apple 各种服务里。但一个真正好用的 Agent，需要一个地方能看懂我们全部的上下文，然后安全地替我们做事。它得在后台一直跑着。而且我们要能随时随地、安全地连上它。

这就是 Personal Cloud 和个人电脑不一样的地方。所以我们相信，下一代 PC 不只是一台更强的电脑，而是一朵个人 AI 云。

那么，我们能用它做什么呢？

简单说，几乎所有跑在公有云上的服务，都能跑在 Personal Cloud 上。ChatGPT，可以用 Ollama 跑开源模型来替代。Midjourney，有 ComfyUI。Slack，有 Mattermost。GitHub，有 GitLab。我们甚至能在上面玩游戏、跑 Windows。

今天，这些条件都齐了：有开源模型，有能投产的开源应用，也有云原生基础设施。Olares 把它们整合到一个安全的环境里，再配上一个自然语言的操作界面。

接下来，我们看一下市场。

因为 AI，每个人需要的算力可能会涨一百倍。按我们的测算，到 2035 年，公有云一年的收入可能到十万亿美元。Personal Cloud 哪怕只占其中的百分之十，那也是一个万亿美元级的市场。

用户为什么会选它？因为数据是自己的。因为 Agent 的延迟很低。因为系统始终在自己手里。还因为它的边际成本，不会随着每一次 AI 交互不停往上涨。

那问题来了：为什么今天还没看到这个市场？

原因是，对大多数人来说，本地 AI 还没跨过体验的门槛。但模型正在快速变小、变强。而且每年都是同一个规律：2022 年的 GPT-3.5，到 2023 年就有了消费级的 Llama 2 13B。2023 年的 GPT-4，到 2024 年有了 Gemma 2 27B。2024 年的 GPT-o1，到 2025 年有了 Qwen3-30B-A3B。2025 年的 Claude Opus 4.5，到 2026 年有了 Qwen3.6-27B。

所以我们判断，2026 年底会是技术拐点。到那时，2025 年最好的 AI 工具，将开始在消费级硬件上本地运行。再到 2027 年初，用户就能在家里的设备上，跑开源版的 Cursor、Manus 或者 Perplexity，体验接近 2025 年的云端产品。

那一刻，就是 Personal Cloud 市场开始起飞的时候。而 Olares，正在为它做操作系统。
