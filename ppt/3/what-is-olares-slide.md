## Slide copy（中文底稿）

### 标题

**Olares 是什么？**

一个你用**自然语言**操作的**开源个人云操作系统**，用来在**你自己拥有的硬件**上运行**本地 AI agent 与 LLM**。

### A Personal Agent for everyone（人人都有的个人智能体）

AI 正从「应用」转向「智能体」。在不远的将来，每个人都会拥有一个 Personal Agent：随时可用，处理工作与日常事务、管理个人信息、打理数字生活，从任何设备都能触达。

### A Personal Cloud to run it（承载它的个人云）

一个单一、始终在线、随时可访问的入口，以及面向个人及其所有 AI 设备的统一执行环境。

- Windows → 运行个人电脑（Personal Computer）
- Android → 运行智能手机（Smartphone）
- Olares → 运行个人云（Personal Cloud）

### Olares 面向谁？

- 想要自托管 AI agent 的高级用户、小企业与注重隐私的用户。
- 看重隐私、数据所有权、可控性与长期成本的用户。

### Olares 解决什么问题？

- 公有云 agent 迫使你把上下文、运行时和操作权交给第三方。
- Olares 把 agent 与 AI 留在你自己的硬件上：NAS、Box、PC、树莓派，或由多台设备组成的集群。
- 用自然语言操作，无需管理服务器、容器、网络、存储或 AI 基础设施。

## 工作底稿（备查）

- 主体是一个橙色描边的四象限卡片；平台三卡强调「不同 OS 承载不同形态」。
- Olares 不是 NAS 或 AI PC，而是个人云操作系统；NAS/Box/PC 只是可运行的硬件载体。
