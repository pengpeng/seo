Advantages

  1. Broad host coverage, near-native LLM speed OCuLink or Thunderbolt covers more NAS, mini-PC and CPU platforms; weights stay in VRAM, leaving steady-state only ~2–3% below internal x16 versus 25–40% loss in gaming.
  2. Flexible upgrades, longer hardware life Choose B60 / B70 by budget, move external compute across compatible hosts, then upgrade only the GPU module.
  3. A lighter product and supply chain Keep GPU power, cooling, BOM and inventory outside the base NAS — without compute-tier SKUs.
  4. Four high-value workloads, one module Personal Agent · Audio AI · Smart Home · Private Workspace.
  5. More Arc Pro sales, a larger software ecosystem Every AI NAS creates a B70 / B60 attach opportunity and expands Intel inference software and developer reach.
  6. A compelling price advantage ~$1,100–1,300 added to an existing NAS, or ~$1,600–2,000 new — well below complete-system alternatives.

What needs work

  1. Prefill software upside remains untapped Personal Agents process long context every turn. B70 is still ~615 pp t/s, but the hardware ceiling has not been reached; XMX attention and the inference stack need further optimization.
  2. The price advantage is cyclical If unified-memory prices return to 2025 levels, the gap narrows; reuse, local data and modular upgrades must carry the long-term value.
